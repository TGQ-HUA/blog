## 简介

TangNotes 是一款 Windows 桌面便签工具，支持**记忆笔记**和**任务管理**，深色/豆沙绿护眼双主题，系统托盘运行，数据纯本地存储。

---

## 四大标签

### 🧠 记忆便签
管理笔记、备忘、知识点。

- **标题**：横向排列，点击切换，双击可重命名
- **内容**：支持 Markdown 格式（加粗、代码、字号、颜色）
- **编辑**：鼠标移到右侧出现 ✏️ 按钮，或**双击卡片**直接编辑
- **序号**：卡片左侧数字圆点，双击可改序号重排
- **移动**：右键卡片 → 选择目标组 → 跨组移动
- **搜索**：标签上方搜索框，输入自动过滤所有标题组

### ✅ 任务便签
管理待办事项，完成后归档时间线。

> 操作方式与记忆便签相同——双击编辑、右键移动、序号排序、搜索过滤。

### 🗑 回收站
删除的记忆内容临时存放，支持恢复或彻底删除。

### 📊 时间线
已完成任务按**年月→任务组**两级分组，可折叠展开。

## 编辑弹窗

双击卡片或点击 ✏️ 按钮弹出编辑器。

| 按钮 | 语法 | 说明 |
|------|------|------|
| **B** | `**文字**` | 加粗 |
| *I* | `*文字*` | 倾斜 |
| U | `__文字__` | 下划线 |
| 🔆 | `****文字****` | 高亮 |
| 📋 | `- 列表` | 无序列表 |
| 🔢 | `1. 编号` | 有序列表 |
| `<>` | `` `代码` `` | 行内代码 |
| 字号 | `[16]文字[/]` | 调整大小 |
| 色点 | `[#E67E22]文字[/]` | 字体颜色 |

---

## 右键移动

右键卡片 → 弹出"请选择目标组"提示 → 标题按钮脉冲高亮 → 点击目标标题 → 移动成功。

---

## 窗口操作

| 操作 | 方式 |
|------|------|
| 拖拽 | 标签栏左侧 `⋮⋮` |
| 缩放 | 右下角折角 |
| 卷起 | 左下角 📜，点窗口展开 |
| 隐藏 | 右上角 ✕ |
| 主题 | 🌙↔☀️ 切换暗色/护眼 |
| 帮助 | ❓ 打开本页面 |

---

## 数据说明

所有笔记保存在 `%APPDATA%\TangNotes\data.json`，纯本地存储，**不会联网上传**。
