## 简介

TangNotes 是一款 Windows 桌面便签工具，支持**记忆笔记**和**任务管理**两大功能，深色/护眼双主题，系统托盘运行，数据纯本地存储。

---

## 四大功能标签

### 🧠 记忆便签
管理笔记、备忘、知识点。

> **标题**：上方横向排列，点击切换，点 `+` 新建  
> **内容**：支持 Markdown（加粗、代码、字号、颜色）  
> **编辑**：鼠标移到内容右侧 → ✏️ 按钮  
> **搜索**：标签上方搜索框，输入自动过滤所有标题组  
> **拖拽排序**：标题按钮可拖拽重新排序

### ✅ 任务便签
管理待办事项，完成后自动归档时间线。

> **添加**：在任务组中点击 `+ 添加待办`  
> **完成**：勾选圆圈 → 弹窗确认 → 移至时间线  
> **搜索**：同记忆便签的搜索方式

### 🗑 回收站
记忆便签删除的内容临时存放，支持恢复或彻底删除。

### 📊 时间线
已完成任务按**年月 → 任务组**两级分组，支持折叠展开和搜索。

## 编辑弹窗

添加或编辑内容时弹出，上方编辑、下方实时预览。

| 按钮 | 语法 | 说明 |
|------|------|------|
| **B** | `**文字**` | 加粗 |
| *I* | `*文字*` | 倾斜 |
| U | `__文字__` | 下划线 |
| 🔆 | `****文字****` | 高亮强调 |
| 📋 | `- 列表项` | 无序列表 |
| 🔢 | `1. 编号项` | 无序列表 |
| `<>` | `` `代码` `` | 行内代码 |
| 12/16/20 | `[16]文字[/]` | 字号 |
| 🟠🔴🔵 | `[#E67E22]文字[/]` | 字体颜色 |

## 窗口操作

| 操作 | 方式 |
|------|------|
| 拖拽移动 | 标签栏左侧 `⋮⋮` 手柄 |
| 缩放窗口 | 右下角折角手柄 |
| 隐藏到托盘 | 点击 ✕ |
| 退出程序 | 托盘图标右键 → 退出 |

---

## 主题和设置

- **明暗主题**：🌙/☀️ 按钮切换。暗色深色背景，亮色为豆沙绿护眼模式
- **开机自启**：🚀 按钮，🟢 表示已开启

---

## 数据说明

所有笔记保存在 `%APPDATA%\TangNotes\data.json`，纯本地存储，**不会联网上传**。
