# B.02.01 弯头制造规则
矩形风管弯头的类型、样式选择表、变径注意事项。
![](/img/knowledge/pipe-fitting-spec/弯头样式.webp)
---

## B.02.01.1 类型与方向
![](/img/knowledge/pipe-fitting-spec/弯头类型.webp)

| 类型 | 转向方向 |
|------|----------|
| **水平弯** | 左右水平转向 |
| **立弯** | 上下垂直转向 |

## B.02.01.2 弯头样式（统一 4 片下料）

弯头由 **扇形顶板 + 底板 + 内外弧板** 组成，外形选择规则：

| 条件 | 建议样式 | 原因 |
|------|----------|------|
| 弧板宽度 > 1400mm | **内菱角** | 太长不方便滚圆 |
| R < 150（不含直位） | **内菱角** | 内弧与直位均分 |
| R < 100（不含直位） | **内直角** | — |
| R直角 < 50 | **改做 50** | 弯头 < 90°时两端直位 < 50 也按 50 |

## B.02.01.3 变径弯头（不等高）注意事项

| 条件 | 处理 |
|------|------|
| 有高差（变径） | 偏差 ≤ 100 → **内弧形**；偏差 > 100 → **内菱角**（直位默认50） |
| 变径 | ❌ **不能做内直角或外直角** |
| 等高 + 内菱角 | 直位与内弧均分 |
| 不等高 + 内菱角 | 直位 50，内弧 = R - 50 |
| 角度 < 20° | ⚠️ 注意展开检查 |
| 非 90° 弯头 | ⚠️ 注意调整拼接缝选项 |

## B.02.01.4 变径弯头必须改加大小头的场景

以下情况**必须先改成等高再加大小头**：
- 内外弧板需做超尺寸拼接缝的
- 有直位的
- R要求做直角的
