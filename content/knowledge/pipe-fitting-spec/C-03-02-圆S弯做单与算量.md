## C.03.02.1 圆 S 弯

### C.03.02.1.1 分段规则

- 常规分 **3 节** 做
- **长度 ≥ 1200mm** 或 **偏差 ≥ 1000mm** → 做 **5 节**，调内径 R 保证首段最短 > 50mm
- 中间段长度 > 1500mm 时，拆分成 2 节弯头
- 长度过短导致**限流**时，需沟通加长

### C.03.02.1.2 接缝设定

同圆弯头。

### C.03.02.1.3 做单示例

**排版尺寸标注**：
```
直径——Φ400  长——800  偏移——200
左直位-50  右直位-50
编号：Y3  数量：1
材质/料厚：镀锌/0.75  接口：无  接缝：碰焊5  分段接口：立咬口
```

**客户常规表述**：

| 客户说 | 含义 |
|--------|------|
| S弯：Φ400-偏差200-长800—1节 | 偏差 S 弯 |

### C.03.02.1.4 面积计算

```
3.14 × 管径 × 中心线长
```
