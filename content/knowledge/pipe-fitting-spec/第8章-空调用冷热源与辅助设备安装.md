# 第8章 空调用冷热源与辅助设备安装

第 8 章规定了制冷机组、热泵、锅炉、冷却塔、水泵、换热器等冷热源及辅助设备的安装质量验收要求，包含**两条强制性条文**（8.2.4、8.2.5）。

---

## 8.1 冷水机组/热泵安装

| 设备类型 | 安装要点 |
|----------|----------|
| **离心式/螺杆式冷水机组** | 基础水平度 ≤ 1‰；减振装置压缩量均匀；冷冻水/冷却水接管正确 |
| **涡旋/活塞式冷水机组** | 机组水平度 ≤ 2mm/m；冷凝器留有检修空间 ≥ 0.8m |
| **风冷热泵机组** | 通风间距 ≥ 1.5m（进风侧）/ ≥ 2.0m（检修侧）；冬季防冻措施 |
| **地源/水源热泵** | 地下换热器冲洗达标；防冻液浓度符合设计 |

### 8.1.1 安装允许偏差

| 项目 | 允许偏差 |
|------|----------|
| 水平度 | ≤ 1‰ |
| 纵向水平度 | ≤ 0.5‰ |
| 皮带传动轮中心线 | 偏移 ≤ 1mm |

---

## 8.2 冷却塔安装

| 检验项目 | 要求 |
|----------|------|
| 基础 | 水平度 ≤ 2‰，承重满足满载运行要求 |
| 安装位置 | 通风良好，远离热源和排风口，进风口间距 ≥ 2m |
| 管道连接 | 多台并联冷却塔的进出水管路须设平衡管，确保水量均匀分配 |
| 补水与排水 | 浮球阀灵活可靠；溢水管和排污管独立排至排水系统 |
| 减振 | 冷却塔须设减振垫，与基础弹性连接 |

---

## 8.3 水泵安装

| 要点 | 要求 |
|------|------|
| 水平度 | ≤ 0.1mm/m（整体安装）/ ≤ 0.05mm/m（解体安装） |
| 同轴度 | 电机与泵轴的径向偏差 ≤ 0.05mm，端面偏差 ≤ 0.03mm |
| 减振 | 设减振台座或减振器 |
| 进出口连接 | 水泵进出口须设**柔性接头**（橡胶软接头或金属软管） |
| 管道支撑 | 水泵进出口管道须设独立支架，**不得使泵体承受管道重量** |

---

## 8.4 锅炉安装

| 锅炉类型 | 安装核心要求 |
|----------|-------------|
| **燃气/燃油锅炉** | 燃烧器与锅炉匹配；烟道严密、保温完整；燃气管道须通过气密性试验 |
| **电热锅炉** | 电气安全保护装置齐全；金属外壳可靠接地 |
| **常压锅炉** | 系统须设置**通大气管**（不可装阀门），确保常压运行安全 |

---

## 8.5 主控项目（强制性条文）

### 🔴 8.2.4 — 燃气系统管道安装

> [!danger] 强制性条文 8.2.4
> **燃气系统管道与机组的连接不得使用非金属软管。燃气管道的吹扫和压力试验应为合格。**
>
> - 燃气管道必须使用**金属管**（无缝钢管或铜管），严禁使用橡胶软管等非金属软管
> - 管道安装后须进行**吹扫**（清除焊渣杂物）和**强度/严密性试验**
> - 试验压力及合格标准按 **GB 50028《城镇燃气设计规范》** 执行
> - 与机组连接处的前端须设**手动快速切断阀**和**电磁阀**

### 🔴 8.2.5 — 制冷剂管道安装

> [!danger] 强制性条文 8.2.5
> **空调制冷剂管道系统安装的焊接、试验必须符合设计文件的规定。当设计无规定时，应符合现行国家标准《制冷设备、空气分离设备安装工程施工及验收规范》GB 50274 的有关规定。**
>
> - 焊接人员须持有**焊工合格证**，并在许可项目范围内施焊
> - 管道焊接后须进行**外观检查**（无气孔、裂纹、未熔合等缺陷）
> - 系统安装完毕后须进行**压力试验**（氮气或干燥空气，一般试验压力为设计压力的 1.5 倍）
> - 压力试验合格后进行**真空试验**和**制冷剂检漏**

## 8.6 换热器与蓄冷/蓄热设备

| 设备 | 安装要求 |
|------|----------|
| **板式换热器** | 按设计流程方向安装；换热片组压紧尺寸符合厂家要求 |
| **壳管式换热器** | 封头端留有抽管检修空间 ≥ 管束长度 |
| **蓄冰槽** | 保温层完整；液位计清晰可读；乙二醇浓度符合设计 |
| **蓄热水罐** | 温度分层装置安装正确；保温层完整、防水 |

---

## 8.7 与 CAMduct 的关联

冷热源设备安装与 CAMduct 直接关联较弱，但需注意：

- **冷却塔与水泵**的进出口管道尺寸须与 CAMduct 输出的空调水管一致
- 设备布置影响了风机和空调机组风管的布局，详见 风管建模与设计
- 锅炉房/制冷机房的**通风换气**要求（排除余热/泄漏制冷剂）决定了配套通风风管的设计标准
