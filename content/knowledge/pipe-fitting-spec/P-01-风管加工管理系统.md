# P.01 风管加工管理系统

> 联泰定制开发的 CAMduct 配套管理系统，打通从排版下料到车间生产、人员绩效、文件归档的全流程闭环管理。
> 版本号：2025.10.25 | 仅限联泰内部使用

---

## P.01.1 核心定位与行业背景

### P.01.1.1 行业痛点

- CAMduct/PM2000 等排版软件仅聚焦下料优化，缺乏人员、进度、文件的一体化管理能力
- 车间生产数据分散，绩效统计依赖人工，误差率高、效率低下
- 加工文件（.MAJ 格式）流转混乱，易丢失、版本追溯困难

### P.01.1.2 软件定位

CAMduct 配套管理系统，核心目标是打通从排版下料到车间生产、人员绩效、文件归档的全流程闭环管理。

---

## P.01.2 功能模块

### P.01.2.1 顶部核心工具栏

| 模块名称 | 核心功能 | 业务价值 |
|----------|----------|----------|
| 风管排版 | 对接 CAMduct，实现风管展开、套料排版 | 优化板材利用率，降低原材料浪费 |
| 绩效看板 | 员工加工量、平方数、作业类型多维度统计 | 量化员工产出，支撑计件绩效核算 |
| 人员信息 | 员工账号、权限、工位管理 | 明确生产责任，实现责任到人 |
| 资料查询 | 历史加工记录、文件追溯、项目检索 | 支持项目复盘与质量问题溯源 |
| 辅助工具 / 出错情况 / 图纸储存 | 加工错误记录、图纸版本管理、异常预警 | 降低返工率，规范文件版本管理 |
| 系统清量 / 设备记录 / 角铁排版 | 板材库存管理、设备维护记录、角铁加工排版 | 实现生产全要素数字化管控 |

### P.01.2.2 核心业务模块：下单排版

**路径配置**：

| 路径类型 | 配置路径 | 业务用途 |
|----------|----------|----------|
| 本地排料路径 | D:\CAM\pb | 对接 CAMduct 本地排版文件目录，读取 .MAJ 加工文件 |
| 下单留档路径 | X:\排版文件 | 加工文件归档目录，实现历史文件分类存储 |
| 下单上传路径 | E:\企业微信存储 | 对接企业微信，实现文件云端备份与跨端流转 |

**一键上传功能**：自动读取本地 .MAJ 文件，同步归档至留档目录与云端，同时统计员工、文件数、加工平方数。

**上传记录字段**：序号、姓名、日期/时间、文件名、作业类型（下单/检查）、加工类型（咬合/焊接/设备）、平方数。

---

## P.01.3 核心业务流程

```
CAMduct排版生成.MAJ文件
    → 系统读取本地排料目录文件
    → 员工执行检查/下单操作
    → 文件自动归档 + 数据统计
    → 上传企业微信云端备份
    → 绩效看板自动汇总数据
```

### P.01.3.1 关键数据维度

| 维度 | 说明 |
|------|------|
| 人员维度 | 按员工统计加工量、作业类型、工艺类型，直接支撑计件工资核算 |
| 项目维度 | 基于文件名提取项目信息，快速统计单个项目的加工总量与进度 |
| 工艺维度 | 区分咬合、焊接、设备等加工类型，分析不同工艺的产能与效率差异 |
| 时间维度 | 按日期/时间记录操作，实现生产进度的实时追踪与交付管控 |

---

## P.01.4 软件特点

1. **高度行业定制化** — 完全适配风管加工行业术语（咬合/焊接/角铁排版）与 CAMduct 文件格式（.MAJ），无需大规模改造现有工作模式
2. **轻量化闭环管理** — 无需复杂 ERP 系统，即可实现生产、数据、文件的全流程管控
3. **安全与权限管控** — 绑定内部账号与 IP 地址，限制外部访问
4. **跨端协同支持** — 对接企业微信实现云端存储，解决文件传输、跨部门协同痛点

---

## P.01.5 潜在优化方向

- 开发 .MAJ 文件自动解析功能，一键提取项目、工艺、工厂信息
- 对接企业微信小程序，实现车间扫码查看加工图纸
- 新增产能趋势、设备利用率、返工率等可视化报表
- 对接 ERP/进销存系统，实现板材库存、采购、加工、出库全链路打通
