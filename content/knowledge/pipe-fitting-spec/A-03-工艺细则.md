# A.03 工艺细则

工艺参数硬限制——圆管口径最小尺寸、翻边细则、焊接料厚、直管标准、激光/梭边/角铁限制。

---

## A.03.1 圆管口径最小尺寸

| 材质 | 条件 | 最小口径 |
|------|------|----------|
| 304# / 镀锌 | 料厚 0.75~1.0 | **100mm** |
| 304# / 镀锌 | 料厚 ≥ 1.0 | **150mm** |
| 201# | 1.0 厚 | **200mm** |
| 所有 | 直径 ≥ 300mm，料厚 ≤ 3.0 | 任何材质可做 |

> 直径 ≥ 800mm 宜用法兰连接，不宜内接。

---

## A.03.2 圆管翻边细则

- 默认 **10mm**，可选 8mm/13mm
- 口径 > 2000mm 或料厚 > 2.0mm → **不可翻边**

## A.03.3 焊接细则

| 类型 | 最薄要求 |
|------|----------|
| 常规焊接 | ≥ 0.7mm |
| 满焊-不锈钢（方/圆异形件） | ≥ 0.75mm |
| 满焊-镀锌板（方/圆异形件） | ≥ 1.0mm |
| 点焊 | ≥ 0.7mm |
| 自动焊氩弧焊-不锈钢 | ≥ 0.7mm |
| 自动焊氩弧焊-镀锌 | ≥ 1.0mm |
| 自动焊气保焊-镀锌 | ≥ 1.2mm |
| 联泰最厚能力 | **3.0mm** |

## A.03.4 对接圆管长度

| 参数 | 值 |
|------|-----|
| 长度 | 2500mm 或 3750mm |
| 不锈钢管接口 | 翻边 10mm 对接 |
| 镀锌管接口 | 直边对接 |
| 圆直管咀长滚圆 | 最长 **1500mm** |

## A.03.5 直管标准

| 规则 | 值 |
|------|-----|
| 开板总长 | ≤ 4000mm |
| L 型：开板总长 | ≥ 500mm，短边 ≥ 120mm |
| 口字型：口径最小 | 160×120mm，单边 ≤ 1000mm，总长 ≤ 3000mm |

## A.03.6 激光标准

| 分厂  | 常规板长                           | 特殊板长   |
| --- | ------------------------------ | ------ |
| 长沙北 | 4100mm                         | 6000mm |
| 板宽  | 镀锌 1250mm，不锈钢 1240mm，特殊 1220mm |        |

## A.03.7 梭边限制

- 咬口最厚：镀锌 **1.5mm**，不锈钢 **1.2mm**
- 变径三通弯头（咬口半成品）：+10元/㎡（不锈钢 ≤ 1.0，镀锌 ≤ 1.2）
- 变径三通弯头（咬口成品）：+20~30元/㎡（同上限制）

## A.03.8 角铁方法兰

自动焊机范围：3500×1350 ~ 300×250mm，超出需手工焊。

## A.03.9 咬口成型受限条件

201# 料厚 ≥ 1.0 → 接缝不建议咬口，建议改直焊边或下调料厚。
