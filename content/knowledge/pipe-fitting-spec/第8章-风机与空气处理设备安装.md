# 第8章 风机与空气处理设备安装

> [!important] 章节定位
> 第8章规定离心风机、轴流风机、空调机组（AHU/PAU）、风机盘管、VAV 箱等通风与空气处理设备的安装工艺。设备安装是系统性能的基础，安装偏差直接影响运行效率、噪声和使用寿命。

---

## 一、风机安装（离心风机 & 轴流风机）

### 1.1 安装前检查

| 检查项目 | 技术要求 |
|----------|----------|
| **基础验收** | 混凝土基础强度 ≥ 设计强度 70%；坐标偏差 ≤ 20mm；标高偏差 ≤ ±10mm |
| **地脚螺栓** | 预留孔洞尺寸正确，孔内清洁无杂物 |
| **设备开箱** | 核对型号、规格、旋转方向、叶轮完整性 |
| **随机文件** | 产品合格证、安装说明书、特性曲线 |

### 1.2 离心风机安装

| 工序 | 技术要求 |
|------|----------|
| **就位** | 按基础中心线和标高就位，垫铁组放置平稳 |
| **减振器安装** | 各减振器压缩量均匀，高度差 ≤ **2mm** |
| **水平度** | 风机纵/横向水平度偏差 ≤ **0.2/1000** |
| **皮带传动** | 皮带轮端面在同一平面内，偏差 ≤ 1mm；皮带松紧适度（拇指按压约 15mm 下沉量） |
| **联轴器传动** | 🔑 径向位移 ≤ **0.05mm**；轴向倾斜 ≤ **0.2/1000**；联轴器间隙 2～3mm |
| **叶轮间隙** | 叶轮与机壳间隙均匀，轴向/径向间隙符合设备技术文件 |
| **进风口** | 进风口与叶轮之间的间隙均匀，确保气流均匀进入 |
| **防护罩** | 🔴 外露传动部位必须设置**防护罩**（强制性要求） |

### 1.3 轴流风机安装

| 工序 | 技术要求 |
|------|----------|
| **安装方式** | 墙式安装（支架固定）或管道式安装（法兰连接） |
| **叶片间隙** | 叶轮与机壳间隙均匀，径向间隙 2～4mm（或按厂家要求） |
| **水平度/垂直度** | 轴流风机水平度/垂直度偏差 ≤ 1/1000 |
| **气流方向** | 叶轮旋转方向与气流方向必须匹配（箭头标识） |
| **软接头** | 进出风端均设柔性短管（长度 150～300mm） |

### 1.4 风机减振措施

| 减振方式 | 适用场景 | 要点 |
|----------|----------|------|
| **弹簧减振器** | 转速 > 1500rpm 或大型风机 | 各减振器压缩量均匀，高度差 ≤ 2mm |
| **橡胶减振垫** | 转速 ≤ 1500rpm 中小型风机 | 满垫铺设，不得局部受力 |
| **惰性块（惯性基座）** | 屋顶风机、低频振动大的风机 | 混凝土基座质量 ≥ 风机质量 × 2 |
| **柔性短管** | 所有风机进出风口 | 长度 150～300mm，隔离振动传递 |

### 1.5 风机安装偏差速查

| 项目 | 允许偏差 |
|------|:--------:|
| 中心线平面位移 | ≤ 10mm |
| 标高 | ≤ ±10mm |
| 水平度（纵/横） | ≤ 0.2/1000 |
| 皮带轮端面偏差 | ≤ 1mm |
| 联轴器径向位移 | ≤ 0.05mm |
| 联轴器轴向倾斜 | ≤ 0.2/1000 |
| 减振器高度差 | ≤ 2mm |

## 二、空气处理机组（AHU/PAU）安装

### 2.1 AHU 类型与安装要点

| AHU 类型 | 结构形式 | 安装要点 |
|----------|----------|----------|
| **组合式空调机组** | 多个功能段现场组装 | 各段对正、密封垫严密、功能顺序正确 |
| **整体式空调机组** | 整机出厂 | 整体就位、水平度、软接头连接 |
| **吊顶式空调机组** | 吊顶安装 | 减振吊架、冷凝水排水坡度 |

### 2.2 组合式 AHU 现场组装工艺

| 工序 | 技术要求 |
|------|----------|
| **功能段排序** | 按气流方向：混合段→过滤段→表冷/加热段→加湿段→风机段→消声段→送风段 |
| **段间连接** | 专用螺栓连接，中间垫 5mm 密封胶条，逐段紧固 |
| **密封检查** | 段间接缝用硅胶密封，不得漏风 |
| **基础水平度** | 整机安装后水平度偏差 ≤ 1/1000 |
| **表冷器接口** | 进水管在下方、出水管在上方（下进上出），保证排气 |
| **冷凝水排水** | 水封高度 ≥ 50mm（或按机组负压计算），排水坡度 ≥ 0.005 |
| **检修空间** | 过滤段、风机段、表冷段侧方预留 ≥ 600mm 检修通道 |

### 2.3 AHU 辅助连接

| 连接部位 | 要求 |
|----------|------|
| **风管连接** | 均通过柔性短管（150～300mm）连接，隔离机组振动 |
| **水管连接** | 软接头（金属软管或橡胶接头），隔离管道应力 |
| **电源接线** | 按电气图纸接线，电机绝缘电阻 ≥ 0.5MΩ |
| **自控接线** | 温度传感器、压差开关、电动阀接线正确 |

## 三、风机盘管（FCU）安装

| 项目 | 技术要求 |
|------|----------|
| **安装方式** | 卧式暗装（吊顶内）、立式明装/暗装、卡式（天花嵌入式） |
| **水平度** | 卧式风机盘管水平度偏差 ≤ 1mm/m |
| **减振** | 吊装风机盘管须设减振吊架（弹簧或橡胶） |
| **冷凝水管** | 坡度 ≥ 0.01（**比 AHU 更严格**），坡向排水点 |
| **风管连接** | 回风箱 + 送风短管，连接严密 |
| **电气接线** | 三速开关与控制线正确接入 |
| **排气** | 高处设手动排气阀 |

---

## 四、VAV 箱 / CAV 箱安装

| 项目 | 技术要求 |
|------|----------|
| **安装位置** | 支管上游，送风口之前 |
| **进出口直管段** | 入口直管段 ≥ 3D（D 为风管直径），出口直管段 ≥ 1.5D |
| **风速传感器** | 传感器探头指向正确，不得装反 |
| **执行器** | 与阀轴同心，0～10V 信号对应 0～100% 开度 |
| **保温和密封** | VAV 箱体外保温，检修门密封 |
| **接线** | DDC 控制器接线端子牢固，屏蔽线可靠接地 |
