# B.11.01 消声器规则

矩形消声器的接缝与长度规则，及消声弯头规范。

---

## B.11.01.1 矩形消声器（本厂制造）
![](/img/knowledge/pipe-fitting-spec/该图.webp)

| 规则 | 详细 |
|------|------|
| 料厚 ≤ 0.8mm | 接缝做 **按扣式 30+10** |
| 料厚 > 0.8mm | 接缝做 **大小边** |
| 消音层 50mm 厚 | 口径长边 > 1100mm = 大口径 → 长度默认 1000mm，最长 1050mm |
| 消音层 100mm 厚 | 口径长边 > 1000mm = 大口径 → 长度默认 900mm，最长 950mm |

### B.11.01.1.1 尺寸与配用

| 项目      | 规格                   |
| ------- | -------------------- |
| **尺寸**  | 宽 × 高 × 长（含咀长）       |
| **消音层** | 一般为 50 或 100mm 厚     |
| **长度**  | 一般包含咀长，两端接口位置相同且大小相同 |

### B.11.01.1.2 消声器 / 静压箱内部结构

| 类型 | 夹层 | 内层 |
|------|------|------|
| **常规**（空调/通风） | 夹层填入消音棉 | 内层采用穿孔板 |
| **除尘系统** | 夹层为空腔 | 内层采用微孔板 |

---

## B.11.01.2 消声弯头
![](/img/knowledge/pipe-fitting-spec/gt3.webp)

| 项目 | 规格 |
|------|------|
| **尺寸** | 宽 × 高 × R |
| **消音层** | 一般为 50 或 100mm 厚 |
| **形状** | 常规形状为**内斜外直** |

---

## B.11.01.3 消声器类型速查（行业通用）

| 类型 | 适用管径 | 结构 | 特点 |
|------|----------|------|------|
| **管式消声器** | 小管径圆管 | 圆形外壳，内填吸音棉 | 适合小风量支管 |
| **片式消声器** | 大风量矩形管 | 矩形箱体，多片吸音片平行排列 | 适合大风量主干管 |
| **阻抗复合消声器** | 机房/风机出口 | 抗性段 + 阻性段组合 | 宽频降噪，机房出口常用 |
