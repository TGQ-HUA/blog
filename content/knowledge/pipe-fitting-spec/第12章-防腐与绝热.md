# 第12章 防腐与绝热

> [!important] 章节定位
> 第12章是风管与水管系统安装完成后的**表面处理章**，涵盖除锈、涂装（防腐）和绝热层、防潮层、保护层施工。防腐与绝热是保证系统使用寿命和节能效果的关键工序，必须在系统试压合格后进行。

---

## 一、防腐施工（涂装）

### 1.1 施工程序

```
基面清理（除锈） → 涂底漆 → 干燥 → 涂中间漆（需要时） → 干燥 → 涂面漆 → 干燥 → 检验
```

### 1.2 除锈要求

| 除锈等级 | 方法 | 适用条件 |
|:--------:|------|----------|
| **Sa 2½** | 喷砂/抛丸（近白级） | 高温高湿环境、室外管道、重要设备 |
| **St 2** | 手工/动力工具（彻底） | 一般室内风管支吊架 |
| **St 3** | 手工/动力工具（非常彻底） | 要求较高的室内金属表面 |
| **FI** | 火焰除锈 | 厚钢板、旧涂层清除 |

| 材质 | 除锈要求 |
|------|----------|
| **镀锌钢板风管** | 锌层完好时**不需要除锈**（仅锌层破损处补涂富锌漆） |
| **普通碳钢板风管** | 除锈至 St 2 级以上 |
| **支吊架（碳钢）** | 除锈至 St 2 级 + 热浸镀锌或涂防锈漆两道 |
| **不锈钢板** | 一般不需涂装（环境要求时除外） |

### 1.3 涂装工艺

| 工序 | 技术要求 |
|------|----------|
| **环境条件** | 相对湿度 ≤ 85%，钢板温度高于露点温度 ≥ 3°C |
| **底漆** | 醇酸铁红底漆 / 环氧富锌底漆 / 无机富锌底漆 |
| **底漆厚度** | 干膜 ≥ 30μm（一般环境）；≥ 50μm（高湿/室外） |
| **中间漆** | 环氧云铁中间漆（需要时，室外/高腐蚀环境） |
| **面漆** | 醇酸面漆 / 聚氨酯面漆 / 丙烯酸面漆 |
| **总厚度** | 干膜总厚度 ≥ 100μm（室内一般）；≥ 150μm（室外/腐蚀环境） |
| **涂层数** | 至少两遍（底漆+面漆），每遍涂刷方向交错（十字交叉法） |

### 1.4 干燥时间

| 涂料类型 | 表干（20°C） | 实干（20°C） | 重涂间隔（最短） |
|----------|:-----------:|:-----------:|:----------------:|
| 醇酸漆 | 2～4h | 16～24h | 4h |
| 环氧漆 | 1～2h | 24～48h | 4h（超 7 天需拉毛） |
| 聚氨酯漆 | 0.5～1h | 12～24h | 2h |
| 无机富锌 | 5～10min | 24～48h | 24h |

> [!warning] 涂装禁忌
> - ❌ 雨、雪、雾、大风天气户外涂装
> - ❌ 相对湿度 > 85% 时涂装（易起泡、附着力差）
> - ❌ 底漆未实干就涂面漆 → 咬底、起皱
> - ❌ 涂层漏涂、厚度不足 → 早期锈蚀
> - ❌ 镀锌板满涂防锈漆 → 浪费 + 附着力差（锌层本身防锈）

### 1.5 涂层质量检查

| 检查项目 | 合格标准 |
|----------|----------|
| **外观** | 颜色均匀、无流挂、无起泡、无漏涂 |
| **厚度** | 用涂层测厚仪抽检，≥ 设计值 90% |
| **附着力** | 划格法 1mm×1mm 格，脱落面积 ≤ 5% |
| **针孔** | 电火花检测（有防腐层要求的管道），无针孔 |

## 二、绝热层施工

### 2.1 绝热材料选型

| 材料类型 | 导热系数 W/(m·K) | 适用温度 | 常见场景 |
|----------|:----------------:|:--------:|----------|
| **离心玻璃棉** | ≤ 0.037 | -50～250°C | 风管保温（最常用） |
| **橡塑海绵 (NBR/PVC)** | ≤ 0.036 | -50～105°C | 水管保温、小风管（湿环境） |
| **岩棉** | ≤ 0.044 | ≤ 650°C | 高温排烟风管 |
| **聚氨酯** | ≤ 0.024 | -50～120°C | 冷风管、冷媒管 |
| **酚醛泡沫** | ≤ 0.025 | -50～120°C | 洁净空调 |

### 2.2 绝热层厚度参考

| 管道/风管类型 | 介质温度 | 玻璃棉厚度 (mm) | 橡塑厚度 (mm) |
|--------------|:--------:|:---------------:|:-------------:|
| 空调送风管 | 14～18°C | 30 | 20～25 |
| 空调回风管 | 24～28°C | 25 | 19～20 |
| 冷冻水管 | 7/12°C | 40～50 | 32～40 |
| 冷却水管 | 32/37°C | 30 | 25 |
| 冷凝水管 | ~15°C | 20 | 13～19 |
| 排烟风管 | >280°C | —（岩棉 50mm） | 不适用 |

### 2.3 风管绝热层施工（玻璃棉板/毡）

| 工序 | 技术要求 |
|------|----------|
| **板材切割** | 尺寸准确，切口平整，按风管尺寸裁剪 |
| **胶粘剂涂刷** | 风管表面与保温板材背面**满涂**胶粘剂，不得有空鼓 |
| **板材粘贴** | 纵横向接缝错开（错缝 ≥ 100mm），不得十字通缝 |
| **接缝处理** | 板缝间隙 ≤ 2mm，用密封胶或专用胶带密封 |
| **保温钉** | 风管底面及侧面用保温钉固定，间距 ≤ 300mm，每块板 ≥ 4 个钉 |
| **金属件隔离** | 保温钉、法兰、支吊架处加隔热垫块，防冷桥 |

### 2.4 水管绝热层施工（橡塑管壳）

| 工序 | 技术要求 |
|------|----------|
| **管壳切割** | 纵向切口，切口平直 |
| **涂胶** | 管道表面 + 管壳内表面 + 接缝面，三面涂胶 |
| **安装** | 管壳套紧，纵缝朝上或侧面（**不得设在底部**） |
| **接缝粘合** | 纵缝+横缝粘合后用专用胶带密封 |
| **弯头/三通处** | 用同材质板材剪裁拼接，不得裸露 |
| **阀门/法兰处** | 可拆卸式保温盒，便于检修 |

## 三、防潮层施工

### 3.1 防潮层设置条件

| 条件 | 是否需要防潮层 |
|------|:------------:|
| **冷介质管道/风管**（介质温度低于环境露点温度） | ✅ 必须 |
| **室外绝热管道/风管** | ✅ 必须 |
| **地沟/潮湿环境** | ✅ 必须 |
| **热介质（常年高于环境温度）室内** | ❌ 可不设 |

### 3.2 防潮层施工要求

| 项目 | 技术要求 |
|------|----------|
| **材料** | 阻燃型沥青玛蹄脂 + 玻璃布 / 铝箔复合防潮层 |
| **搭接宽度** | 30～50mm |
| **搭接方向** | 🔑 **顺向搭接**（上压下周，不得逆向，防水流入） |
| **密封** | 搭接处用胶带或玛蹄脂密封 |
| **完整性** | 无破损、无漏包、无气泡 |

---

## 四、保护层施工

### 4.1 保护层设置条件

| 条件 | 是否需保护层 | 推荐保护材料 |
|------|:----------:|------------|
| **室外风管/水管** | ✅ 必须 | 铝板（≥ 0.5mm）/ 彩钢板 / 玻璃钢 |
| **室内（无机械损伤风险）** | 可不设 | 铝箔即可 |
| **可踩踏/通行区域** | ✅ 必须 | 金属板保护 |
| **洁净室** | ✅ 必须 | 不锈钢板 / 彩钢板 |

### 4.2 金属保护层施工

| 项目 | 技术要求 |
|------|----------|
| **板材厚度** | 铝板 ≥ 0.5mm；镀锌钢板/彩钢板 ≥ 0.4mm |
| **接缝方向** | 🔑 **顺水搭接**（上压下，防雨水进入） |
| **搭接宽度** | ≥ 30mm |
| **固定** | 自攻螺钉（不锈钢或镀锌），间距 ≤ 200mm |
| **弯头/异形** | 虾壳弯分节加工，每节咬口 + 螺钉固定 |
| **法兰/阀门处** | 可拆卸式金属罩壳 |

## 五、施工质量通病总结

| 通病 | 原因 | 后果 |
|------|------|------|
| 绝热层空鼓 | 未满涂胶粘剂 | 结露、保温失效 |
| 接缝十字通缝 | 未错缝排版 | 冷桥、结露 |
| 冷桥 | 金属件未隔热处理 | 局部结露、滴水 |
| 防潮层逆向搭接 | 施工方向错误 | 雨水渗入、保温失效 |
| 保温钉穿透防潮层 | 先做保温钉后做防潮层 | 防潮层破损 |
| 室外无保护层 | 设计/施工遗漏 | 绝热层老化、进水失效 |
