# I.06.03 系统设置

> Opus 工作环境配置——常规选项与网格选项。

---

## 3.1 常规选项（General Options）

路径：`View → General Settings`，或点击 Opus 窗口底部 **Settings** 按钮。

| 选项 | 说明 |
|------|------|
| **Default Work Width** | Opus 工作区默认宽度（mm） |
| **Default Work Height** | Opus 工作区默认高度（mm） |
| **Location by Mouse** | 鼠标当前坐标显示开关。也可点击底部坐标按钮切换 |
| **Curve Quality** | 圆/圆弧的线段数。值越大越平滑，但数据量也更大 |
| **Cursor Select Threshold** | 鼠标选择偏移容差。不须精确对准元素即可选中（像素） |
| **Use Hardware Cursor** | 设为 **No** 可自定义鼠标光标 |
| **Cursor Size** | 自定义光标大小（像素）。仅关闭硬件光标时可用 |
| **Cursor Color** | 自定义光标颜色。仅关闭硬件光标时可用 |
| **Show Directions** | 在已绘制零件上显示默认切割方向 |
| **Show Hints** | 设为 **Yes** 时工具提示显示更详细信息 |
| **Snaps Override Polar** | 设为 **Yes** 时其他捕捉模式可覆盖极轴捕捉 |
| **Tool-Tip Delay (ms)** | 鼠标悬停时工具提示延迟时间（毫秒）。不影响 UI 控件提示 |

---

## 3.2 网格选项（Grid Options）

路径：`View → Grid Settings`，或点击窗口底部 Settings / Grid Options 按钮。

| 选项 | 说明 |
|------|------|
| **Points** | 使用网格点代替网格线（或组合）。点大小由 Point Size 决定 |
| **Lines** | 开关网格线显示 |
| **Pattern** | 背景棋盘格图案 |
| **Transparency** | 网格线/图案透明度百分比。越低越浅 |
| **Grid Spacing** | 网格点间距（mm） |
| **Point Size** | 网格点尺寸 |
| **Line Color** | 网格线颜色 |
| **Point Color** | 网格点颜色 |
| **Pattern Color** | 背景图案颜色 |
| **Adaptive Zoom-In** | **Yes** = 放大视图时自动缩小网格间距 |
| **Adaptive Zoom-Out** | **Yes** = 缩小视图时自动扩大网格间距 |
