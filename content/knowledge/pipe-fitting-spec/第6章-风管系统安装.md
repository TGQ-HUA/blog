# 第6章 风管系统安装

第 6 章规定了风管系统安装（包括支吊架、风管连接、穿越结构、阀门安装等）的质量验收要求，包含**两条强制性条文**（6.2.2、6.2.3）。

---

## 6.1 支吊架安装

### 6.1.1 支吊架间距

GB 50243-2016 第 6.1.1 条规定了支吊架最大间距：

| 风管类型 | 风管长边 ≤ 400mm | 风管长边 > 400mm |
|----------|-----------------|------------------|
| **水平金属风管** | ≤ 4.0m | ≤ 3.0m |
| **水平非金属风管** | ≤ 3.0m | ≤ 2.5m |
| **垂直金属风管** | ≤ 4.0m（且不少于 2 个固定点） | |
| **垂直非金属风管** | ≤ 3.0m（且不少于 2 个固定点） | |

### 6.1.2 支吊架安装要点

| 要点 | 具体要求 |
|------|----------|
| 吊杆直径 | 长边 > 1250mm 时，吊杆直径 ≥ 10mm |
| 横担规格 | 长边 ≤ 630mm 用角钢 L25×3；长边 > 2000mm 用槽钢 |
| 防腐处理 | 支吊架须做防腐处理（镀锌或涂防锈漆） |
| 防晃支架 | 风管末端、转弯处、分支处及垂直风管每隔 2 层须设**防晃支架** |

> 支吊架参数直接影响 风管建模与设计 中的 Hanger/Support 配置。

## 6.2 风管连接

### 6.2.1 连接方式选择

| 连接方式 | 适用场景 | 密封要求 | CAMduct 对应 |
|----------|----------|----------|-------------|
| **角钢法兰** | 大尺寸、中高压风管 | 法兰垫片密封 | Flange Connector |
| **共板法兰 (TDC/TDF)** | 中低压、中小尺寸矩形风管 | TDF 垫片密封 | TDC / TDF Connector |
| **插接连接** | 圆形风管 | 密封胶带 + 插接密封 | Slip Connector |
| **咬口连接** | 直管段接长 | 咬口成型 + 密封胶 | Pittsburgh / Snap Lock |

### 6.2.2 法兰垫片材料

| 系统类型 | 垫片材料 | 厚度 |
|----------|----------|------|
| 一般通风空调 | 闭孔海绵橡胶、橡胶板 | 3~5mm |
| 防排烟系统 | **不燃材料**（石棉橡胶板、陶瓷纤维垫片） | 3~5mm |
| 洁净空调 | 闭孔海绵橡胶或硅橡胶 | 5~8mm |
| 输送含尘气体 | 橡胶板（耐磨型） | 3~5mm |

---

## 6.3 风管穿越结构

| 穿越类型 | 要求 |
|----------|------|
| **穿越防火分区** | 🔴 必须设防火阀，距防火墙表面 ≤ 200mm；穿越处用**不燃材料**严密填塞 |
| **穿越沉降缝/变形缝** | 风管两侧设**软连接**补偿位移 |
| **穿越楼板/墙体** | 穿越处设套管，风管与套管间填充柔性不燃材料 |
| **穿越易燃易爆区域** | 风管须**可靠接地**，法兰间设**跨接线** |

---

## 6.4 风管安装允许偏差

| 检验项目 | 允许偏差 | 检验方法 |
|----------|----------|----------|
| 水平度 | 每米 ≤ 3mm，总偏差 ≤ 20mm | 水准仪、拉线 |
| 垂直度 | 每米 ≤ 2mm，总偏差 ≤ 20mm | 吊线、钢卷尺 |
| 管口平面度 | ≤ 5mm | 塞尺 |
| 法兰接口错位 | ≤ 2mm | 游标卡尺 |

---

## 6.5 主控项目（强制性条文）

### 🔴 6.2.2 — 防火风管必须为不燃材料

> [!danger] 强制性条文 6.2.2
> **防火风管的本体、框架与固定材料、密封垫料必须为不燃材料，其耐火等级应符合设计规定。**
>
> 此条与 第4章 风管与配件|第4章 4.2.5 相互呼应——4.2.5 管"制作"材料，6.2.2 管"安装"材料，双重把关防火安全。

### 🔴 6.2.3 — 风管穿越防火区必须设防火阀

> [!danger] 强制性条文 6.2.3
> **风管穿越防火分区的隔墙、楼板以及防火墙两侧 2m 范围内必须设置防火阀，且防火阀距防火墙表面不应大于 200mm。穿越处缝隙必须用不燃材料严密填塞。**
>
> - **防火阀间距**：≤ 200mm（用钢卷尺现场测量）
> - **填塞要求**：穿越处 360° 全断面填塞
> - **联动测试**：安装完成后须进行防火阀的**熔断关闭及信号反馈**测试

### 6.2.4 — 易燃易爆环境风管接地

| 要求 | 说明 |
|------|------|
| **风管接地** | 输送易燃易爆气体的风管，其金属法兰间必须设**铜编织带跨接线**，并与建筑物的**接地干线**可靠连接 |
| **接地电阻** | ≤ 4Ω（用接地电阻测试仪检测） |

## 6.6 一般项目

| 条目 | 内容 | 要求 |
|------|------|------|
| 6.3.1 | 支吊架位置 | 不妨碍风口、阀门的操作和检修 |
| 6.3.2 | 风管连接 | 接口严密，法兰垫片不凸入管内 |
| 6.3.3 | 风阀安装 | 操作手柄置于便于操作位置 |
| 6.3.4 | 柔性短管 | 长度 150~300mm，无扭曲和过度拉伸 |
| 6.3.5 | 风管清洁度 | 管内无杂物、油污、积水 |

## 6.7 与 CAMduct 的关联

| CAMduct 环节 | 第6章影响 |
|-------------|----------|
| **Service 模板** | 防排烟系统 Service 需配置防火风管 + 不燃垫片 |
| **连接方式选择** | 根据压力等级和系统类型选择 TDC/TDF 还是角钢法兰（风管连接方式） |
| **管段分割** | 考虑支吊架间距（3~4m）优化管段标准长度 |
| **穿越标注** | 在管段图纸上标注防火分区穿越位置和防火阀安装点 |
| **NC 清单** | 输出包含防火阀定位信息的安装清单 |
