### C.07.02.1.2 做单示例

**排版尺寸标注**：
```
左主管-Φ500  右主管-Φ500  上支管-Φ400  下支管-Φ400
主管长-700  上支长-100  下支长-100
编号：Y6  数量：1
材质/料厚：镀锌/0.75  接口：无  主管接缝：拼接15  支咀接缝：碰焊5
```

**客户常规表述**：

| 客户说 | 含义 |

| 四通：Φ500\*Φ500\*Φ400\*Φ400—长700—1节 | 四通 |

### C.07.02.1.3 面积计算

```
3.14 × 主管径 × 中心线A  +  3.14 × 支直径 × 中心线B
```
