# I.06.08 钢结构导入（Structural Steel Import）

> CAMduct 可从钢结构设计软件（如 XSteel、StruCAD、AutoCAD 插件）导入零件数据进行自动处理。

---

## 支持格式

- **DSTV**（最常用，XSteel 默认）
- **DXF**
- **ASCII 文本文件**

特殊过滤器可自动处理三种格式的连接板。

---

## 连接板（Fastening Plates）

- 用途：连接钢结构梁柱的钢板
- 厚度范围：**2–25 mm**
- 薄矩形板 → 剪切打孔；厚板 → 轮廓切割（激光/氧燃/等离子）

---

## CAMduct 钢结构处理功能

| 功能 | 说明 |
|------|------|
| **DSTV 过滤器创建** | 零件几何、孔尺寸/位置、零件编号、材质、厚度、数量 |
| **剪切 vs 轮廓** | 可将特定材质/厚度设为剪切（不经过轮廓切割） |
| **Align Parts** | 比对长宽 → 设置纹理角，使长边朝向指定方向（水平/垂直）。禁用时零件可自由旋转 |
| **Auto-Increment 必须禁用** | 否则导入零件编号会被覆盖 |

---

## 导入流程（DSTV 示例）

1. 创建 **Installed Filter**（安装过滤器）
2. File Extension 设为 `NC1`（XSteel 默认扩展名）
3. 可选配置**剪切材料列表**（哪些材质/厚度不经过轮廓切割）
4. 导入文件（可用 `Ctrl+A` 全选）
5. 对话框可**逐件更改材质**
6. 进入工单内容后可正常编辑和套料

> 💡 实测：**800+ 零件、约 3 吨的工单 → 处理时间 < 5 分钟**（含报表打印和切割）。

---

## 关键设置速查

| 设置 | 值 |
|------|-----|
| 板厚范围 | 2–25 mm |
| 典型文件格式 | DSTV（XSteel → 扩展名 `.NC1`） |
| 800+ 零件（3 吨）处理时间 | < 5 分钟 |
| Auto-Increment Item Numbers | **必须禁用** |
| Align Parts | 启用 = 控制纹理方向；禁用 = 自由旋转 |
| 材质分配 | 导入时可逐件更改 |
