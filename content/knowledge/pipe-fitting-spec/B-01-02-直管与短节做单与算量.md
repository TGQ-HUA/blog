## B.01.02.1 直管
![](/img/knowledge/pipe-fitting-spec/矩形直管样图.webp)
### B.01.02.1.1 直管类型

| 类型 | 下料方式 | 说明 |
|------|----------|------|
| **标准直管** | 五线开料 | 实惠/节约/省事，长度由接口类型决定 |
| **短直管** | 激光开料 | 非标准直管的全部算短直管 |

### B.01.02.1.2 标准节长度与短节就料长度

| 接口类型 | 标准节长度 | 短节就料长度 |
|----------|:----------:|:------------:|
| **共板法兰** | 1160 | 520 |
| 共板-堵 | 1175 | 535 |
| 共板-直边 | 1205 | 565 |
| **角铁法兰** | 1235 | 600 |
| 角铁-堵 | 1215 | 575 |
| 角铁-直边 | 1240 | 605 |
| **插条边** | 1220 | 580 |
| 插条-堵 | 1205 | 565 |
| 插条-直边 | 1235 | 595 |
| **直边** | 1250 | 610 |

> **静压箱就料高度、箱子就料长度**：550

**短直管长度特殊情况**：

| 情况 | 处理 |
|------|------|
| 长度与标准节差 ≈ 100mm | 建议做成标准直管 |
| 长度 > 2000mm | 拆成「标准直管 + 短直管」 |
| 客户无要求 | 按短直管就料长度 |

### B.01.02.1.3 成型方式

| 成型方式 | 条件 |
|----------|------|
| **口字形** | 口径周长 < 1000mm（最小 160×120，单边 ≤ 1000，总长 ≤ 3000） |
| **L 形** | 1000 ≤ 口径周长 ≤ 7600mm（短边 ≥ 120，板长 ≤ 7600） |
| **四片式** | 口径周长 > 7600mm（或长度 > 标准节 且 周长 > 2400） |

> 简化版：口字形 < 1000 ≤ L形 ≤ 4000 < 四片式

更详细的成型决策树见 [B.01.01 直管与短节成型决策规则](/knowledge/pipe-fitting-spec/B-01-01-直管与短节成型决策规则/)。

### B.01.02.1.4 做单示例

**排版尺寸标注**：
```
宽——1000  高——500  长——400
编号：Z1  成型方式：L形  数量：2
接口：共板  材质/料厚：镀锌/0.75  接缝：大小边
```

**客户常规表述**（无长度说明 = 按标准节长度）：

| 客户说 | 含义 |
|--------|------|
| 直管：1000\*500——5节 | 标准直管，共板标准节长 1160 |
| 直管堵：1000\*500——1节 | 直管带堵头 |
| 短直管：1000\*500\*400——3节 | 明确给出长 400 的短管 |

**报价单示例**：

| NO  | 材质  | 名称  | 规格(1)    | 长度  | 数量  | 厚度   | 接口  | 状态  | 备注  |
| --- | --- | --- | -------- | --- | --- | ---- | --- | --- | --- |
| Z1  | 镀锌  | 短直管 | 1000x500 | 400 | 2   | 0.75 | 共板  | 半   | L型  |

### B.01.02.1.5 面积计算

```
(宽 + 高 + 大小边) × 2 × (长 + 接口 × 2)
```
