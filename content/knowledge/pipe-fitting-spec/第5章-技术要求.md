# 第5章 技术要求

> [!important] 章节定位
> 第5章是 GB 15930-2007 的**产品性能核心章节**，规定了防火阀、排烟防火阀、排烟阀三类产品在设计制造中必须达到的全部技术指标。涵盖耐火性能、动作温度、漏风量、启闭可靠性、控制方式、耐腐蚀性、绝缘电阻等关键参数，是产品 CCC 认证和进场验收的技术判定依据。

---

## 一、耐火性能

### 1.1 耐火极限

> [!warning] 强制性要求
> 防火阀、排烟防火阀、排烟阀的耐火时间均**不得低于 1.50 h**。

| 指标 | 要求 | 试验标准 |
|------|------|----------|
| **耐火时间** | ≥ **1.50 h**（三类阀门统一要求） | 按 GB/T 9978（ISO 834）标准升温曲线 |
| **试验状态** | 阀门安装在耐火试验炉中，模拟实际工况 | 详见 第6章 试验方法 |
| **判定准则** | 试验期间保持规定的开启/关闭状态，无影响功能的变形或损坏 | 完整性 + 隔热性 |

> [!tip] 为什么统一要求 ≥1.5h？
> 1.5h 耐火极限与 GB 50016 中**一级防火墙的耐火极限（3.0h）** 和**防火隔墙（2.0h）** 相衔接，确保阀门在最不利火灾场景下能保持有效隔断。通风管道穿越防火分区时，防火阀是管道路径上唯一的机械隔断，其耐火极限必须与建筑防火分隔匹配。

### 1.2 耐火试验中的关键要求

| 项目 | 试验前状态 | 试验中要求 | 试验后要求 |
|------|:----------:|------------|:----------:|
| **防火阀** | 常开 | 达到 70°C 时自动关闭 | 保持关闭，背火面无火焰穿透 |
| **排烟防火阀（常开型）** | 常开 | 达到 280°C 时自动关闭 | 保持关闭，背火面无火焰穿透 |
| **排烟防火阀（常闭型）** | 常闭 | 电动开启→280°C 自动关闭 | 保持关闭 |
| **排烟阀** | 常闭 | 电动开启 | 保持开启 |

## 二、动作温度与温感器

### 2.1 公称动作温度

| 阀门类型 | 公称动作温度 | 允差 | 温度元件 |
|----------|:-----------:|:----:|----------|
| **防火阀** | **70°C** | **± 2°C** | 70°C 易熔合金温感器 |
| **排烟防火阀** | **280°C** | **± 3°C** | 280°C 易熔合金温感器 |
| **排烟阀** | 不适用 | — | 无温感器 |

### 2.2 温感器技术要求

| 项目 | 要求 |
|------|------|
| **动作温度精度** | 实际动作温度与公称温度的偏差必须在允差范围内（70±2°C / 280±3°C） |
| **响应时间** | 温感器暴露在规定温度环境中，应在合理时间内动作 |
| **可靠性** | 经 50 次启闭循环后，温感器动作温度仍应在允差范围内 |
| **不可复位性** | 温感器一旦动作（熔断），不可自行恢复，必须更换 |

> [!danger] 温感器不可替代性原则
> 防火阀（70°C）和排烟防火阀（280°C）的温感器**绝对不能互换**。排烟系统若误装 70°C 温感器，在正常排烟工况（烟气温度可达 150~200°C）下将导致阀门误关闭，排烟系统完全失效。

## 三、漏风量限值

### 3.1 漏风量指标

在环境温度下，阀门两侧保持 **300 Pa** 静压差时的漏风量要求：

| 阀门类型 | 状态 | 漏风量限值 | 单位 |
|----------|:----:|:----------:|------|
| **防火阀** | 常开状态 | ≤ **500** | m³/(h·m²) |
| **防火阀** | 关闭状态 | ≤ **200** | m³/(h·m²) |
| **排烟防火阀** | 常开状态 | ≤ 500 | m³/(h·m²) |
| **排烟防火阀** | 关闭状态 | ≤ 200 | m³/(h·m²) |
| **排烟阀** | 常闭状态 | ≤ **200** | m³/(h·m²) |

### 3.2 漏风量的工程意义

| 漏风量过高影响 | 后果 |
|----------------|------|
| **通风空调系统** | 风量分配失衡，末端风口风量不足 |
| **排烟系统** | 漏风稀释烟气浓度，降低排烟效率 |
| **防火分隔** | 关闭状态下漏风意味着烟气可通过阀门缝隙扩散，削弱防火隔断效果 |

## 四、启闭可靠性与循环寿命

### 4.1 启闭循环要求

> [!warning] 强制性要求
> 防火阀门经 **50 次** 全行程启闭循环后，各项性能指标仍应符合本标准全部要求。

| 项目 | 要求 |
|------|------|
| **循环次数** | ≥ **50 次**全行程启闭 |
| **循环后性能** | 动作温度、漏风量、耐火性能、控制功能均不得降低 |
| **循环后外观** | 叶片、连杆、执行机构无松动、磨损或永久变形 |
| **循环后复位** | 手动复位机构操作灵活，复位到位 |

### 4.2 50 次循环的工程意义

50 次循环对应于产品的全生命周期使用需求：
- 安装调试：2~3 次启闭
- 消防联动测试（每年 1 次 × 建筑使用寿命 50 年）：50 次
- 日常维护检测：按需操作

## 五、控制方式

### 5.1 控制方式分类

| 控制方式 | 代号 | 说明 | 适用阀门 |
|----------|:----:|------|:--------:|
| **温度熔断器动作** | W | 温感器达到公称温度后自动熔断，释放关闭机构 | 防火阀、排烟防火阀 |
| **手动控制** | S | 就地手动操作手柄或按钮，控制阀门启闭 | 全部三类阀门 |
| **电磁铁控制** | Dc | DC 24V 电磁铁通电/断电控制阀门动作 | 排烟防火阀（常闭型）、排烟阀 |
| **电动执行机构** | Dj | 电动执行器（AC 220V 或 DC 24V）驱动阀门 | 大型阀门、远程控制 |
| **远距离控制** | K | 缆索式或控制箱远程手动操作 | 安装位置不便操作的阀门 |

### 5.2 各类阀门的最低控制功能要求

| 阀门类型 | 必配控制方式 | 可选控制方式 |
|----------|-------------|-------------|
| **防火阀** | W（温控自动）+ S（手动） | K（远距离控制） |
| **排烟防火阀** | W（温控自动）+ S（手动） | Dc/Dj + K |
| **排烟阀** | S（手动）+ Dc（DC 24V 电控） | Dj + K |

### 5.3 电气控制可靠性

| 项目 | 要求 |
|------|------|
| **电压范围** | 额定电压的 **85% ~ 110%** 范围内应能正常工作 |
| **额定电压** | DC 24V 或 AC 220V |
| **绝缘电阻** | 带电部件与壳体之间 ≥ **20 MΩ** |
| **介电强度** | 应能承受规定的耐压试验，无击穿或闪络 |

## 六、其他技术要求

### 6.1 材料要求

| 项目 | 要求 |
|------|------|
| **阀体/叶片材料** | 应采用**不燃材料**（镀锌钢板、不锈钢等），厚度满足强度和耐火要求 |
| **密封材料** | 应采用不燃或难燃材料，高温下不脱落 |
| **轴承/轴套** | 应采用自润滑或耐高温材料（如铜合金、石墨铜套） |
| **耐腐蚀性** | 金属部件应经防腐处理，耐盐雾试验 ≥ **96 h** |

### 6.2 复位功能

| 阀门类型 | 复位方式要求 |
|----------|-------------|
| **防火阀** | 必须**手动复位**（严禁自动复位） |
| **排烟防火阀** | 必须**手动复位**（严禁自动复位） |
| **排烟阀** | 手动或电动复位均可 |

> [!danger] 为何必须手动复位？
> 防火阀和排烟防火阀关闭后，意味着火灾已蔓延至管道内。此时若阀门能自动复位，可能导致消防人员误判火势已受控。手动复位要求操作人员**主动确认**火情已消除后，才能人工恢复阀门至开启状态。

### 6.3 标示要求

| 项目 | 要求 |
|------|------|
| **气流方向** | 阀门上应清晰标示气流方向箭头 |
| **叶片状态** | 应能直观判断叶片的开/关位置 |
| **操作标识** | 手动操作部位应有明显的操作标识 |

## 📊 技术指标速查总表

| 技术指标 | 防火阀 (FHF) | 排烟防火阀 (PFHF) | 排烟阀 (PYF) |
|----------|:-----------:|:-----------------:|:-----------:|
| 耐火时间 | ≥ 1.5 h | ≥ 1.5 h | ≥ 1.5 h |
| 动作温度 | 70°C ± 2°C | 280°C ± 3°C | 不适用 |
| 常开漏风量 (300Pa) | ≤ 500 m³/(h·m²) | ≤ 500 m³/(h·m²) | — |
| 关闭漏风量 (300Pa) | ≤ 200 m³/(h·m²) | ≤ 200 m³/(h·m²) | ≤ 200 m³/(h·m²) |
| 启闭循环 | ≥ 50 次 | ≥ 50 次 | ≥ 50 次 |
| 盐雾试验 | ≥ 96 h | ≥ 96 h | ≥ 96 h |
| 绝缘电阻 | ≥ 20 MΩ | ≥ 20 MΩ | ≥ 20 MΩ |
| 控制电压 | — | DC 24V / AC 220V（带电控时） | DC 24V / AC 220V |
| 复位方式 | 手动 | 手动 | 手动/电动 |
