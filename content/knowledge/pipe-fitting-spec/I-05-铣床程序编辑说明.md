# I.05 铣床程序编辑说明

CAMduct → 北京精雕 的完整工作流——3 种刀路参数、文件格式转换、文本修改步骤。

---

## I.05.1 3 种刀路

### I.05.1.1 轮廓（切断）

| 参数 | 值 |
|------|-----|
| 雕深 | 8.7 |
| 吃刀 | 8.7 |
| G | 3.7 |
| 走刀 | 逆铣 |

### I.05.1.2 C 槽（小刀）

| 材质 | 雕深 | 吃刀 |
|------|------|------|
| 单彩 | 4 | 4 |
| 双彩 | 5 | 5 |

**C 槽用途**：
- 内外弧板铣 C 槽：内弧每道间距 **40**，外弧每道间距 **60**
- 外折铣 C 槽（如棱形弯头内侧板外折）

### I.05.1.3 V 槽（V 刀）

| 材质 | 雕深 | 吃刀 |
|------|------|------|
| 单彩 | 5 | 5 |
| 双彩 | 6 | 6 |

**V 槽用途**：内折铣 V 槽（如棱形弯头外侧板内折）

---

## I.05.2 格式转换

| 项目 | 参数 |
|------|------|
| 机械参数 | 缓降 **1000**、切削 **10000** |
| 服务参数 | 安全 **20**、缓降 **10** |
| 文件头 | `T200` → `M03` |
| 文件尾 | `M05` → `M30` |

---

## I.05.3 文本修改

| 操作 | 说明 |
|------|------|
| `T0(XXX)` 行 | **全部删除**（每个文件） |
| `T200` → `T100` | **仅 C 槽文件** |

---

## I.05.4 完整编辑流程

```
1. CAMduct 画好切槽路线 → 排板
2. 写出 NC 码 → 导出 DXF → 打印排版图
3. 排版图上标注 C 槽和 V 槽
4. 北京精雕 → 文件→输入→所有格式 → 选 DXF
5. 导入后确认 → 「放于初始位置」
6. 单块板依次写码：
   · C槽码 → 文件名 "XXXc"
   · V槽码 → 文件名 "XXXv"
   · 轮廓码(切断) → 文件名 "XXX"
       右边小料宽度<200 → 分两批：
        小料 "XXX-1"，大料 "XXX-2"
7. 转换格式 → 修改文本
```
