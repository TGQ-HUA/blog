# H.01 共板打孔规则

共板法兰打孔的旧规则（固定首孔）和新规则（整边均分）对比与参数速查。

---

## H.01.1 新规则：整边均分（2026年1月6日起默认）

### H.01.1.1 核心规则
!26.png

| 规则 | 说明 |
|------|------|
| **均分原则** | 边长等分打孔，相邻孔中心距 ≤ 客户要求孔距 |
| **超短边不打** | 边长 < 150mm → 不打孔（两端 75mm 后无空间） |
| **小孔距例外** | 孔距 < 100mm → 采用固定首孔规则，首孔边距 75mm |

### H.01.1.2 计算示例

边长 800mm，要求孔距 ≤ 140mm：
```
段数 = Ceil(800 / 140) = 6段 → 5个孔
实际孔距 = 800 / 6 = 133.33mm ✓
```

---

## H.01.2 旧规则：固定首孔（2026年1月6日前默认）

### H.01.2.1 核心规则
!25.png

| 规则 | 参数 |
|------|------|
| **首孔固定边距** | 110mm（两端各） |
| **均分区间** | 边长 - 220mm |
| **小边仅 1 孔** | 中间区域 < 孔距/2 |
| **超短边不打** | 边长 < 220mm |

### H.01.2.2 计算示例

同 800mm，孔距 ≤ 140mm：
```
中间区域 = 800 - 220 = 580mm
段数 = Ceil(580/140) = 5段 → 6个孔（含两端首孔）
实际中间孔距 = 580/5 = 116mm ✓
```

---

## H.01.3 新旧对比

| 维度 | 固定首孔(旧) | 整边均分(新) |
|------|------------|------------|
| 首孔边距 | 固定 110mm | 自适应 |
| 孔数计算 | 中间区域均分 | 整边均分 |
| 小边不打阈值 | < 220mm | < 150mm |
| 美观度 | 两端与中间孔距不同 | 全部均匀 ✓ |
| 使用建议 | 历史未完结项目 | 新项目默认 |
