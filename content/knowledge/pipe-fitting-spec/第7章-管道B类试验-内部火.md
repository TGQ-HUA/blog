# 第7章 管道B类试验（内部火）

> [!important] 章节定位
> 第7章规定了**管道B类耐火试验**——模拟管道内部着火、管道外部为常温环境的工况。该试验是**排烟管道**和**加压送风管道**耐火极限的核心验证方法，其结果直接决定管道耐火极限分级（0.5~3.0h）。GB 51251-2017 中排烟管道的耐火极限要求即依据本标准 B 类试验判定。

---

## 一、试验原理

### 1.1 模拟场景

```
      ┌─────────────────────────────┐
      │    常温环境（试验大厅）       │
      │                              │
      │   ╔══════════════════════╗   │
      │   ║  🔥 管 道 内 部 着 火  🔥 ║  │ ← 燃烧器在管道内部点火
      │   ║     ISO 834 升温      ║   │
      │   ╚══════════════════════╝   │
      │                              │
      │     外部环境温度：常温        │
      └─────────────────────────────┘
```

**关键特征**：
- **火在管道内** → 管道内表面为受火面
- **管道外为常温环境** → 无外部热源，仅内部高温传导至外壁
- 验证排烟管道在**输送火灾高温烟气**时的耐火完整性

### 1.2 适用对象

| 适用管道类型 | 典型工程场景 |
|--------------|--------------|
| 排烟风管（机械排烟系统） | 火灾时排出室内高温烟气，管道内部承受高温 |
| 正压送风管道 | 防烟楼梯间/前室加压送风，管道需在火灾时保持完好 |
| 排油烟管道 | 厨房排油烟系统，内部可能起火 |
| 工业高温排风管道 | 工业炉窑排风、高温工艺排风 |

---

## 二、试件准备

### 2.1 试件要求

| 项目 | 要求 |
|------|------|
| **试件长度** | ≥ 3m（整段管道，提供完整试验数据） |
| **试件截面** | 按工程实际，通常 ≥ 125mm × 125mm（矩形）或 φ125mm（圆形） |
| **管壁构造** | 与实际工程一致的单层/多层复合结构 |
| **连接方式** | 如工程含中段法兰/接头，试件应包含（验证接头的耐火性能） |
| **安装方式** | 水平安装，按设计要求的吊架/支架间距固定 |

### 2.2 燃烧器布置

| 项目 | 要求 |
|------|------|
| **安装位置** | 管道一端，燃烧器喷嘴伸入管道内部 |
| **火焰方向** | 顺着管道轴向，向另一端喷射 |
| **燃烧器数量** | 根据管道截面确定，保证内部温度均匀 |
| **燃料** | 天然气或液化石油气 |

> [!tip] B类试验的燃烧控制
> 不同于 A 类试验的炉膛燃烧，B 类试验需要在管道内部产生符合 ISO 834 的温度场。燃烧器通常采用**管道内燃烧器**（内径燃烧器），燃料和空气在管道内混合燃烧，通过调节燃气量和过量空气系数控制管内温度。

---

## 三、试验方法

### 3.1 试验步骤

| 步骤 | 操作 | 关键点 |
|:----:|------|--------|
| **1** | 管道试件水平安装在试验台架上 | 按工程支承方式固定 |
| **2** | 燃烧器安装在管道一端 | 火焰朝向管道远程 |
| **3** | 管道另一端开放（自然对流）或接排气管道 | 不强制通风 |
| **4** | 点燃燃烧器，管内温度按 ISO 834 曲线控制 | 同步启动数据采集 |
| **5** | 持续试验至试件丧失完整性或隔热性，或达到目标耐火极限 | 观察火焰穿出、棉垫试验、外壁温升 |
| **6** | 记录耐火极限，分级评定 | — |

### 3.2 管内温度控制

| 控制指标 | 要求 |
|----------|------|
| **测温位置** | 管道内部距燃烧器出口 500mm 处 + 管道中部 + 管道末端 |
| **目标温度** | ISO 834 标准升温曲线 |
| **允许偏差** | 同第5章规定（0~5min ±15%，5min后 ±10% 或 ±100°C） |
| **管内压力** | 不强制控制压力（自然通风条件），但宜记录 |

### 3.3 数据记录

| 记录项目 | 频率 | 用途 |
|----------|:----:|------|
| **管内温度** | ≥ 1次/min | 验证升温符合性 |
| **管道外表面温度**（4点） | ≥ 1次/min | 判定隔热性 |
| **位移**（管道变形） | ≥ 1次/min | 判定完整性 |
| **观察记录** | 事件触发 | 火焰穿出、开缝、坍塌、掉渣 |

---

## 四、判定准则

### 4.1 完整性丧失（同A类）

| 丧失条件 | 判定方法 |
|----------|----------|
| **火焰穿透** | 棉垫试验——管道外壁或接缝处，棉垫在 30s 内点燃/发光燃烧 |
| **φ6mm 探棒穿透** | 探棒可从外部穿入管道内部，且可沿缝隙移动 ≥ 150mm |
| **φ25mm 探棒穿透** | φ25mm 探棒可穿入 |
| **坍塌/掉落** | 管道丧失承载能力，发生坍塌 |

### 4.2 隔热性丧失（与A类相同标准）

| 丧失条件 | 判定阈值 |
|----------|:--------:|
| **平均温升** | 外表面 4 点平均温升 ≥ 140°C |
| **单点最高温升** | 外表面任意单点温升 ≥ 180°C |

> [!important] 判定逻辑
> **耐火极限 = min(完整性丧失时间, 隔热性丧失时间)**

---

## 五、耐火极限分级

B 类试验的结果按耐火极限时间划分为 **5 个等级**：

| 等级 | 耐火极限 t (h) | 对应工程要求（GB 51251） | 工程案例 |
|:----:|:-------------:|--------------------------|----------|
| **0.5** | 0.5 ≤ t < 1.0 | 部分排烟管道、≤100m 住宅加压送风 | 多层住宅排烟 |
| **1.0** | 1.0 ≤ t < 1.5 | ⭐ **最常见等级**：高层建筑排烟管道 | 办公楼排烟系统 |
| **1.5** | 1.5 ≤ t < 2.0 | 超高层核心筒外排烟、重要公建 | 综合医院、大型商场 |
| **2.0** | 2.0 ≤ t < 3.0 | 超高层建筑避难层/防烟楼梯间 | 200m+ 超高层办公楼 |
| **3.0** | t ≥ 3.0 | 特殊建筑（核电站、重要隧道、地下空间） | 特长隧道、核电厂房 |

### 5.1 分级选择建议

```
    建筑高度 → 耐火等级要求
    ─────────────────────────
    ≤ 100m        1.0 h
    > 100m        1.5 h
    超高层核心区    2.0 h
    特殊建筑       3.0 h
```

> [!warning] 工程中常见误区
> - 金属铁皮风管本身无耐火极限 → 必须加防火包裹或采用耐火成品风管
> - 仅有防火包裹还不够 → 必须提供按 GB/T 17428 B类试验的第三方检测报告
> - 防火包裹厚度不是越厚越好 → 须经有资质的实验室测试验证

## 六、B类试验与A类试验对比

| 对比项 | A类（外部火） | B类（内部火） |
|--------|:------------:|:------------:|
| **火灾位置** | 管道外部 | 管道内部 |
| **管内条件** | 冷空气流通 3m/s | 自然对流 |
| **受火面** | 外表面 | 内表面 |
| **测量关键面** | 外表面（隔热性判定面） | 外表面（隔热性判定面） |
| **典型场景** | 穿越防火分区的空调通风管 | 排烟管、加压送风管 |
| **分级** | 按判定时间直接给值 | 0.5/1.0/1.5/2.0/3.0 h 五级 |
| **GB 51251 依据** | 加压送风管道穿越非火灾区域 | 排烟管道耐火极限 |
