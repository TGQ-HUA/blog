# A.04 风管材质与规格速查

> 风管厂涉及的全部材质、系统分类、常用规格及板材厚度选用标准。

---

## A.04.1 风管材质

| 材质 | 分类 | 说明 | 常规厚度 (mm) |
|------|------|------|---------------|
| **镀锌板**（白铁皮） | 有花板 | 镀锌层 40G，车间常规有材料。 | 0.5 / 0.6 / 0.7 / 0.75 / 0.8 / 0.9 / 1.0 / 1.1 / 1.2 / 1.4 / 1.5 |
|  | 无花板 | 镀锌层 40G / 80G / 120G，接单需先确认车间库存 | 2.0 / 3.0（偶有项目，需问车间库存） |
|  | 武钢 | 武钢 80G / 武钢 120G，只有无花 | |
| **不锈钢板** | 201# | 含锰较高，表面亮带暗黑，易生锈。Cr 15%、Ni 5% | 0.6 / 0.75 / 1.0 / 1.2 / 2.0 |
| | 304# | 含铬较多，哑光不生锈。Cr 18%、Ni 9% | |
| **碳钢板**（黑板） | 冷板 | ≤2.0mm，用于冷折弯加工 | 1.5 / 2.0 / 3.0 |
| | 热板 | 较厚板，加热后锻造成型 | |
| **防火板** | 镁质复合材料，玻璃纤维布增强 | 见下表 | 6 / 8 / 14 / 20 |

> **材质关系**：镀锌板 = 冷轧钢板（热轧钢板）+ 表面镀锌

| 维度 | 选项 | 备注 |
|------|------|------|

| **彩钢面** | 单面彩钢 / 双面彩钢 / 白板 | 白板 = 无彩钢面，一般做防火包裹用 |
| **彩钢板** | 不压花（常规）/ 压花 | 彩钢厚 0.2mm，颜色白灰。压花需特别备注 |
| **板材规格** | 2.44m 板（2440×1200）/ 3.0m 板（3000×1200） | 下料排版依就料规格用 |

| 3.0H | 内 1.0 镀锌板 + 100 防火棉 + 外 0.75 镀锌板 | |

## A.04.2 风管系统类别

| 风压类别 | 常规用途 | 管内正压 P (Pa) | 管内负压 P (Pa) |
|----------|----------|-----------------|------------------|
| 微压 | — | P ≤ 125 | — |
| 低压 | 通风、空调 | 125 < P ≤ 500 | -500 ≤ SP < -125 |
| 中压 | 通风、空调 | 500 < P ≤ 1500 | -1000 ≤ SP < -500 |
| 高压 | 排烟 | 1500 < P ≤ 2500 | -2000 ≤ SP < -1000 |

> 用途涵盖：排烟、通风、空调、净化除尘等。

## A.04.4 板材厚度规定

### A.04.4.1 矩形风管 — 常规板材厚度 (mm)

| 长边尺寸 b | 微压/低压 | 中压 | 高压 | 除尘 |
|-------------|-----------|------|------|------|
| P ≤ 320 | 0.5 | 0.5 | 0.75 | 2.0 |
| 320 < P ≤ 450 | 0.5 | 0.6 | 0.75 | 2.0 |
| 450 < P ≤ 630 | 0.6 | 0.75 | 1.0 | 3.0 |
| 630 < P ≤ 1000 | 0.75 | 0.75 | 1.0 | 4.0 |
| 1000 < P ≤ 1500 | 1.0 | 1.0 | 1.2 | 5.0 |
| 1500 < P ≤ 2000 | 1.0 | 1.2 | 1.5 | 按设计 |
| 2000 < P ≤ 4000 | 1.2 | 1.2 | 按设计 | 按设计 |

### A.04.4.2 圆形（螺旋）风管 — 常规板材厚度 (mm)

| 长边尺寸 b | 微压/低压 | 中压正压 | 高压正压 | 中压负压 | 高压负压 |
|-------------|-----------|----------|----------|----------|----------|
| P ≤ 320 | 0.5 | 0.5 | 0.5 | 0.5 | 0.5 |
| 320 < P ≤ 450 | 0.5 | 0.5 | 0.6 | 0.6 | 0.75 |
| 450 < P ≤ 630 | 0.6 | 0.6 | 0.75 | 0.75 | 1.0 |
| 630 < P ≤ 1000 | 0.75 | 0.75 | 1.0 | 0.75 | 1.0 |
| 1000 < P ≤ 1500 | 1.0 | 1.0 | 1.2 | 1.0 | 1.2 |
| 1500 < P ≤ 2000 | 1.0 | 1.2 | 1.2 | 1.2 | 1.5 |
| 2000 < P ≤ 4000 | 1.2 | 1.5 | 1.5 | 1.5 | 1.5 |

### A.04.4.3 不锈钢风管 — 常规板材厚度 (mm)

| 风管长边 b 或直径 D | 正压风管 | 高压/负压风管 |
|----------------------|----------|----------------|
| 100 < P ≤ 500 | ≥ 0.5 | ≥ 0.75 |
| 320 < P ≤ 1120 | ≥ 0.75 | ≥ 1.0 |
| 1120 < P ≤ 2000 | ≥ 1.0 | ≥ 1.0 |
| 2000 < P ≤ 4000 | ≥ 1.2 | ≥ 1.2 |
