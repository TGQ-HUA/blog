# 第6章 风机安装

> [!important] 章节定位
> 第6章规定了离心风机、轴流风机、罗茨风机等通风机械的安装及验收要求。暖通工程中，**离心风机**（含风机箱、空调机组风机段）和**轴流风机**（含排烟风机）是最常见的两类。本章的间隙控制、减振器选型直接影响风机运行效率、噪声和使用寿命。

---

## 一、通用安装要求

### 1.1 安装前检查

| 检查项目 | 要求 |
|----------|------|
| **核对型号规格** | 风机铭牌参数（风量、风压、转速、功率）与设计图纸一致 |
| **外观检查** | 机壳无变形、锈蚀、损伤；叶轮转动灵活、无卡滞 |
| **主轴承检查** | 轴承座内清洁，滚动轴承加注润滑脂（填充量 2/3） |
| **电机检查** | 电机绝缘电阻 ≥ 0.5MΩ，转动方向标识清晰 |

### 1.2 安装通用要求

| 项目 | 技术要求 |
|------|----------|
| **基础处理** | 参见 第2章 施工准备 — 基础验收；预留垫铁空间 |
| **减振措施** | 风机须配备减振器/减振垫（见本章第四节） |
| **进出风管连接** | 风管自重不得由风机机壳承受，须独立设支吊架 |
| **柔性连接** | 风机进出口须设**柔性短管**（帆布/硅胶布），长度 150~300mm |
| **防护罩** | ⚠️ 外露传动部件（皮带轮、联轴器）必须设防护罩（强制性） |

## 二、离心风机安装

### 2.1 离心风机结构示意

```
          出风口
           ↑
    ┌──────────────┐
    │   蜗    壳    │
    │   ┌───┐      │
    │   │叶轮│     │ ← 叶轮在蜗壳内偏心布置
    │   └───┘      │
    │     主轴     │──── 联轴器/皮带 ──── 电机
    └──────────────┘
          蜗舌
    进风口（轴向）→
```

### 2.2 离心风机安装核心技术：叶轮与机壳间隙

离心风机的**叶轮与蜗壳间隙**是影响风机效率和气流稳定性的关键参数。

| 间隙位置 | 符号 | 功能 | 允许偏差 |
|----------|:----:|------|:--------:|
| **叶轮进口圈与集流器径向间隙** | δ₁ | 防止进口气流回流，影响容积效率 | 沿圆周均匀，偏差 ≤ 1.5mm |
| **叶轮外缘与蜗壳径向间隙** | δ₂ | 保证叶轮旋转不碰壳 | 各测点偏差 ≤ 对应标准值 |
| **叶轮后盘与蜗壳轴向间隙** | δ₃ | 防止轴向摩擦 | 按制造厂文件要求 |

#### 叶轮与机壳间隙检查方法

| 步骤 | 操作 |
|:----:|------|
| **1** | 在蜗壳上选取 4~6 个测量点（均布在圆周上） |
| **2** | 用塞尺逐点测量叶轮外缘距蜗壳内壁距离 |
| **3** | 记录各点值，计算最大值与最小值之差 |
| **4** | 偏差超标时，调整轴承座位置或蜗壳安装方位 |

### 2.3 离心风机安装精度

| 项目 | 允许偏差 | 检测方法 |
|------|:--------:|----------|
| **主轴水平度**（纵向） | ≤ 0.2/1000 | 框式水平仪置主轴 |
| **主轴水平度**（横向） | ≤ 0.2/1000 | 框式水平仪置轴承座剖分面 |
| **电机与风机联轴器对中** | 参见 第3章 设备就位与找正调平#三、找正方法（三表法）\|三表法 偏差表 | 三表法 |
| **皮带轮平行度** | 皮带轮端面在同一平面，偏移 ≤ 1mm | 拉线法 + 钢板尺 |
| **三角带张紧度** | 单根手指按皮带中部下沉量：15~25mm（按中心距调整） | 手压法 + 直尺 |

### 2.4 离心风机试运转检查

| 检查项目 | 判定标准 |
|----------|:--------:|
| 叶轮旋转方向 | 与机壳箭头标识一致 |
| 运转平稳性 | 无异常振动和撞击声 |
| 轴承温升 | ≤ 40°C（滚动轴承）/ ≤ 30°C（滑动轴承） |
| 轴承最高温度 | ≤ 80°C（滚动轴承）/ ≤ 70°C（滑动轴承） |
| 试运转时间 | ≥ 2小时连续运转 |

## 三、轴流风机安装

### 3.1 轴流风机结构特点

```
        ┌─────────────────────────┐
  气流 →│   ┌──── 叶轮 ────┐     │→ 气流
  进风 →│   │ ╱ 叶片 ╲    │     │→ 出风
        │   └──────────────┘     │
        └─────────────────────────┘
             风筒（机壳）
```

轴流风机气流方向**平行于主轴轴线**，结构紧凑，风压较低但风量大。暖通中常用于排烟、排风、冷却塔。

### 3.2 轴流风机核心技术：叶片与风筒间隙

| 间隙类型 | 要求 | 意义 |
|----------|------|------|
| **叶片顶端与风筒内壁径向间隙 (δ)** | 沿圆周均匀，偏差 ≤ 叶片直径的 **0.5%**，且最小值 ≥ **2mm** | 间隙过小→擦壳；过大→空气泄漏回流量大，效率剧降 |
| **各叶片间隙偏差** | 同一风机的所有叶片，间隙差 ≤ **1.5mm** | 保证动平衡 |

> [!tip] 轴流风机间隙检查实测方法
> 1. 在风筒圆周上取 4 等分点
> 2. 逐点用塞尺测该处叶片顶与风筒内壁的径向距离
> 3. 盘动叶轮，每叶片在每个测点记录
> 4. 如偏差超标，调整电机/轴承座底板垫铁

### 3.3 轴流风机特殊安装要求

| 项目 | 要求 |
|------|------|
| **风机水平度** | ≤ 0.2/1000（与离心风机一致） |
| **风筒与风管连接** | 风筒法兰与连接风管法兰平行，法兰面平面度 ≤ 2mm |
| **叶片角度调节** | 可调叶片风机，所有叶片角度必须一致（偏差 ≤ ±0.5°） |
| **直管段** | 风机进口前应有 ≥ 2D（D=风筒直径）的直管段；出口后 ≥ 1.5D |
| **电动机接线** | 排烟风机电机接线须用耐高温电缆，确保火灾初期可正常运行 |

> [!warning] 轴流排烟风机特别注意
> - 排烟轴流风机电机如置于气流通道内，必须使用 **耐高温电机**（280°C/30min 以上）
> - 风机与排烟管道间须设 GB15930-2007 建筑通风和排烟系统用防火阀门|防火阀 联动控制

## 四、减振器选型与安装

### 4.1 减振器的作用

风机运行时产生振动，通过结构传导可引发以下问题：
- 建筑结构振动和二次噪声（固体传声）
- 风机轴承过早磨损
- 连接风管疲劳开裂
- 影响周边精密设备/舒适性空间

### 4.2 减振器类型

| 类型 | 结构 | 固有频率 | 适用场景 |
|------|------|:--------:|----------|
| **橡胶减振垫** | 橡胶板材/波形垫 | 8~15 Hz | 小型风机、风机盘管、转速 < 1000 rpm |
| **弹簧减振器** | 钢制螺旋弹簧+阻尼器 | 2~5 Hz | 中大型风机、转速 ≥ 1000 rpm、隔振要求高 |
| **橡胶剪切减振器** | 橡胶剪切块 | 5~10 Hz | 中型风机，中等隔振需求 |
| **空气弹簧** | 气囊+压缩空气 | 0.5~2 Hz | 特殊精密隔振（实验室、精密厂房） |

### 4.3 减振器选型计算

| 参数 | 公式/取值 | 说明 |
|------|-----------|------|
| **总重量 (W)** | W = 风机自重 + 电机自重 + 机座重量 | 含底座槽钢 |
| **减振器数量 (n)** | 通常 4 或 6 个（对称布置） | 重心与减振器形心重合 |
| **单减振器载荷 (P)** | P = W ÷ n × K | K = 不均匀系数（1.1~1.2） |
| **减振效率 (η)** | η = (τ-1)/τ × 100% | 频率比 τ = f/f₀（f=风机转速频率，f₀=减振器固有频率），τ ≥ 3 时效佳 |
| **静态压缩量 (δ)** | 弹簧: δ = P/Ks（Ks=弹簧刚度）| 橡胶: 查制造商压缩曲线 |

### 4.4 减振器安装要求

| 项目 | 技术要求 |
|------|----------|
| **安装位置** | 减振器置于风机底座（型钢框架）与基础/结构楼板之间 |
| **对正** | 减振器应对称布置，各组受力均匀，静压缩量基本一致 |
| **水平度** | 安装后减振器顶面齐平，各减振器高度差 ≤ 2mm |
| **限位** | 当风机启动/停机通过共振区时，须设**限位装置**防止过大振幅 |
| **与结构连接** | 减振器底板与基础/楼板用膨胀螺栓固定，不得点焊（防止热影响） |
| **柔性连接** | 风机进出口必须设柔性短管，风管设弹性支吊架配合减振体系 |

### 4.5 减振体系检查

| 检查项目 | 判定标准 |
|----------|:--------:|
| 各减振器压缩量 | 最大与最小之差 ≤ 2mm |
| 减振器水平度 | ≤ 1/1000 |
| 试运转振动 | 无异常共振，基础振动速度 ≤ 4.5mm/s |
| 柔性连接 | 帆布软接无拉裂、无过度拉伸 |

> [!tip] 暖通工程减振体系设计
> 一个完整的减振体系应包括三级隔离：
> 1. **源头减振** → 风机底座减振器
> 2. **传递路径隔断** → 进出风口柔性短管 + 风管弹性吊架
> 3. **结构声阻断** → 风机房做浮筑地板/隔声围护结构（高要求场合）
