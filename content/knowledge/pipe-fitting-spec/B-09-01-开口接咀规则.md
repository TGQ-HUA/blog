# B.09.01 开口接咀规则
矩形风管开口接咀——圆孔/方孔/静压箱接咀的尺寸规则。

---

## B.09.01.1 圆孔接圆咀

**圆孔大小** = 圆咀口径 + 圆孔调整量

| 接缝类型 | 料厚条件 | 圆孔调整量 | 说明 |
|----------|:--------:|:----------:|------|
| **咬口** | 厚度 > 0.75mm | **+1mm** | 厚板咬口需要放大 1mm 间隙，保证圆咀顺畅插入 |
| **咬口** | 厚度 ≤ 0.75mm | **0** | 薄板咬口正尺寸即可 |
| **焊接** | 任意 | **−料厚** | 焊接时圆孔缩小一个料厚，抵消焊丝填充量 |

**圆咀长度**：

| 接缝类型 | 公式 |
|----------|------|
| 咬口 | 要求长度 + 10mm（余量）+ 法兰口用料量 |
| 焊接 | 要求长度 + 法兰口用料量 |

## B.09.01.2 方孔接方咀

```
· 方孔做锯齿
· 方咀对应做反大边
· 锯齿翻边与反大边咬口
```

## B.09.01.3 静压箱接咀

| 咀的法兰口类型     | 箱体开口尺寸 | 接咀尺寸 |
| ----------- | ------ | ---- |
| **角铁**      | +5mm   | +2mm |
| **其他**（共板等） | 正尺寸    | -3mm |
