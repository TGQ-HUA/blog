# C.09.01 侧支咀规则

圆形风管侧支咀的制造规范。
![](/img/knowledge/pipe-fitting-spec/圆侧支咀1.webp)
---

## C.09.01.1 45°+45° 侧支咀

| 项目       | 规则                    |
| -------- | --------------------- |
| **分段**   | 常规 **2 段**            |
| **长度公式** | `长度 = 口径 / 2.4 + 170` |
| **默认长度** | `100mm`（非公式计算时）       |
| **接主管处** | 默认需**翻边 10mm**        |
| **主管与咀子口径相同、口径 > 600** | 默认配弧板（弧板尺寸需**手动调整**，受模型限制） |

| **默认长度** | `100mm`（非公式计算时） |
