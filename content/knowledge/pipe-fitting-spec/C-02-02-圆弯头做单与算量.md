# C.02.02 圆弯头做单与算量

> 圆弯头的客户表述规范、内部报价单格式、面积计算公式。
!圆弯头样式2.png
---

## C.02.02.1 圆弯头

### C.02.02.1.1 弯头类型

| 类型       | 说明              |     |
| -------- | --------------- | --- |
| **冲压弯头** | 一体成型，表面光滑       |     |
| **波纹弯头** | 一体成型，表面波纹状凸筋    |     |
| **分段弯头** | 分段成型，内外菱形（本厂常规） |     |

> 分段弯头内 R 最小建议 **125**（勉强可做 100，否则需采购）。

### C.02.02.1.2 分段数量规定

| 角度 | 分段数 |
|------|:------:|
| < 45° | 2 段 |
| < 90° | 3 段 |
| < 135° | 4 段 |

> 详细分段规则见 C.02.01 圆弯头规则。

### C.02.02.1.3 接缝设定

| 口径 | 接缝 |
|:----:|------|
| ≤ 500 | 碰焊 5 |
| > 500 | 咬口（拼接 15） |

> 分段接口需加**立咬口边**。

**材质**： Φ250 及以下统一用无花板，Φ250 以上以生产单材质规定为准。

### C.02.02.1.4 做单示例

**排版尺寸标注**：
```
直径——Φ500  内径——R200
底直位-50  侧直位-50
编号：Y2  数量：2
材质/料厚：镀锌/0.75  接口：无  接缝：碰焊5  分段接口：立咬口
```

**客户常规表述**（内径一般无要求，清单默认尺寸做）：

| 客户说 | 含义 |
|--------|------|
| 90°弯：Φ500—2节 | 90°弯头 |
| 45°弯：Φ500—2节 | 45°弯头 |

**报价单示例**：

| NO | 材质 | 名称 | 规格(1) | 规格(2) | 长度 | 数量 | 厚度 | 接口 | 状态 |
|----|------|------|---------|----------|------|------|------|------|------|
| Y2 | 镀锌 | 圆弯头 | Φ500 | 【\*2】 | 250 | 2 | 0.75 | 咬口 | 成 |

### C.02.02.1.5 面积计算

```
3.14 × 管径 × 中心线长
```
