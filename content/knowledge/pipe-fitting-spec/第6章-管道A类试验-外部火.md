# 第6章 管道A类试验（外部火）

> [!important] 章节定位
> 第6章规定了**管道A类耐火试验**——模拟管道外部发生火灾、管道内部通冷空气流动的工况。该试验验证的是**防火包裹**或**防火板风管**在外部火灾侵袭下能否保持完整性和隔热性，典型场景为穿越防火分区的空调通风管道。

---

## 一、试验原理

### 1.1 模拟场景

```
        ┌─────────────────────────────┐
        │    试验炉（火灾环境）         │
        │   🔥🔥🔥🔥🔥🔥🔥🔥🔥🔥🔥     │
        │    ┌───────────────────┐    │
 冷空气 →│  → │   通风管道（试件）  │→  │→ 冷空气
  (3m/s) │    └───────────────────┘    │   (3m/s)
        │   🔥🔥🔥🔥🔥🔥🔥🔥🔥🔥🔥     │
        │        外部火灾              │
        └─────────────────────────────┘
```

**关键特征**：
- **火在管道外** → 管道外表面为受火面
- **管道内通冷空气**（3m/s）→ 内部介质带走部分热量，但也可能助燃
- 验证管道在**穿越防火分区**时的耐火性能

### 1.2 适用对象

| 适用管道类型 | 典型工程场景 |
|--------------|--------------|
| 镀锌钢板风管 + 外覆防火包裹（岩棉/硅酸钙板/防火板） | 通风空调管道穿越防火墙/楼板 |
| 预制成品的耐火风管（如纤维增强硅酸钙板风管） | 防排烟管道通过非火灾区域 |
| 铁皮风管外覆防火涂料 | 旧楼改造管道防火加强 |

---

## 二、试件准备

### 2.1 试件要求

| 项目 | 要求 |
|------|------|
| **试件长度** | ≥ 3m（其中受火段长度 ≥ 2m，两端各 0.5m 伸出炉外） |
| **试件截面** | 与工程实际一致，通常矩形或圆形 |
| **试件数量** | 同一规格至少 1 个（建议 2 个取较差值） |
| **支撑方式** | 按设计要求吊装/支撑，吊架间距同工程实际 |
| **端部连接** | 两端用法兰/接头与试验空气管道系统连接 |

### 2.2 试件养护

| 条件 | 要求 |
|------|------|
| **养护温度** | 20 ± 5°C |
| **养护湿度** | 50 ± 20% RH |
| **养护时间** | 防火包裹/防火板含水泥基材料：≥ 28 天 |

---

## 三、试验方法

### 3.1 试验步骤

| 步骤 | 操作 | 关键点 |
|:----:|------|--------|
| **1** | 试件安装在炉内，两端连接空气管道系统 | 管道水平安装，吊架/支架按设计 |
| **2** | 启动送风机，调整管内风速至 3 ± 0.5 m/s | 风速稳定后开始试验 |
| **3** | 点燃燃烧器，炉温按 ISO 834 曲线控制 | 同步启动数据采集 |
| **4** | 维持炉压 20 ± 3 Pa | — |
| **5** | 持续试验至试件丧失完整性或隔热性，或达到设计耐火极限 | 观察火焰穿透、棉垫试验、温升 |
| **6** | 停止燃烧，继续通风冷却 | 记录冷却过程中试件状态 |

### 3.2 试验过程中的数据记录

| 记录项目 | 频率 | 用途 |
|----------|:----:|------|
| **炉内温度**（4点） | ≥ 1次/min | 验证升温曲线符合性 |
| **管道外表面温度**（4点） | ≥ 1次/min | 判定隔热性 |
| **炉内压力** | ≥ 1次/min | 验证压力条件 |
| **管内风速** | ≥ 1次/min | 验证通风条件 |
| **位移**（管道变形） | ≥ 1次/min | 判定完整性 |
| **观察记录** | 事件触发 | 火焰穿透、掉渣、坍塌等 |

---

## 四、判定准则

### 4.1 完整性丧失判定

**完整性** = 试件保持其分隔火焰和高温气体的能力。以下任一条件发生时，判定完整性丧失：

| 丧失条件 | 判定方法 | 说明 |
|----------|----------|------|
| **火焰穿透** | 用棉垫（100mm×100mm×20mm）置试件背面疑似穿透处 30s 或距缝隙 20~25mm 处 30s，棉垫点燃（发光燃烧） | ⚠️ 仅发黑/冒烟不燃烧不算穿透 |
| **缝隙** | 用 φ6mm 探棒可以穿过试件或接缝处进入炉内，且可沿缝隙移动 ≥ 150mm | 探棒检查 |
| **φ25mm 探棒穿透** | φ25mm 探棒可穿入试件任何开口 | 较大开口 |
| **坍塌/掉落** | 试件失去支承能力，发生坍塌 | 整体性破坏 |

> [!tip] 棉垫试验要点
> - 棉垫为**脱脂棉**，质地蓬松
> - 试验时将棉垫固定在不燃框架上
> - 棉垫发光燃烧（火焰）→ 不合格；仅受热变黑/冒烟 → 不影响完整性

### 4.2 隔热性丧失判定

**隔热性** = 试件阻止热量从受火侧传导至非受火侧的能力。以下任一条件发生时，判定隔热性丧失：

| 丧失条件 | 判定阈值 | 测量部位 |
|----------|:--------:|----------|
| **平均温升** | 外表面 4 点平均温升 ≥ 140°C（高于初始温度） | 管道外表面—非受火面 |
| **单点最高温升** | 外表面任意单点温升 ≥ 180°C（高于初始温度） | 管道外表面—非受火面 |

> **公式表达**：
> - 平均温升 ≤ 140°C → 隔热性合格
> - 任一测点温升 ≤ 180°C → 隔热性合格
> - 两个条件同时满足才算隔热性合格

### 4.3 耐火极限判定

```
              耐火极限
                 │
         ┌───────┴───────┐
         ↓               ↓
    完整性丧失时间    隔热性丧失时间
         │               │
         └───────┬───────┘
                 ↓
          取两者较早者
         ＝ 耐火极限
```

> [!warning] 判定逻辑
> 耐火极限 = min(完整性丧失时间, 隔热性丧失时间)。例如：管道在 60 分钟时棉垫点燃（完整性丧失），但隔热性仍合格 → 耐火极限 = 60 分钟。

## 五、A类试验报告要点

| 报告要素 | 内容 |
|----------|------|
| **试件描述** | 管道截面尺寸、长度、壁厚、材质、防火包裹类型和厚度 |
| **支承方式** | 吊架/支架型号、间距、连接方式 |
| **试验条件** | ISO 834 曲线实测数据、炉压记录、管内风速记录 |
| **温度曲线** | 炉温 + 试件外表面温度时间曲线（4点） |
| **失效模式** | 完整性还是隔热性先丧失？具体现象描述 |
| **耐火极限** | 分钟 |
| **试验照片** | 试验前/中/后试件状态照片 |
