## B.04.01.1 类型

| 对齐方式   | 说明                  |
| ------ | ------------------- |
| **对中** | 左右对中（宽）、上下对中（高）、全对中 |
| **平边** | 左平/右平（边平）、顶平/底平（面平） |
| **偏差** | 左右偏移、上下高差           |

## B.04.01.2 成型决策规则

```
第1条 · 先看能不能走 U+I形
  必须同时满足：
  ┌───────────────────────────────────┐
  │ 条件A（尺寸门槛）                  │
  │   半周长 < 2000mm 且 长度 < 1000mm │
  │                                   │
  │ 条件B（一边平）                    │
  │   底平/顶平 → 宽不变（G = I）      │
  │   边平/侧平 → 高不变（H = J）      │
  └───────────────────────────────────┘
  两个条件都满足 → U+I形

第2条 · 其他一律 4片式
  包括：
  · 尺寸超过门槛（太大/太长）
  · 不是一边平的大小头（两边都变径：G≠I 且 H≠J）
```

## B.04.01.3 偏差规范

> ⚠ **大小头偏差过大**——变径管两端偏移量超出合理范围，导致中段截面收缩过猛。

| 问题 | 现象 | 危害 |
|------|------|------|
| **大小头偏差过大** | 变径两端偏移超出合理范围 | 中段管件变小 → 风阻增大 → 风量减小 → 影响风机效率 |
[图片缺失: 大小头偏差过大.png]

### 改善措施

1. **加长管件**：增加变径段长度，减缓截面变化梯度（如长500mm → 1000mm）
   [图片缺失: 菱形S弯.png]

2. **检查偏移量**：对中变径两侧偏移量应一致，平边变径一侧为0、另一侧在设计范围内

3. **高度不同时按S弯排版**：当两端高度不一致（上下偏移），不硬做变径，改用S弯（双向偏移）过渡
   [图片缺失: 大小头高差S弯.png]
