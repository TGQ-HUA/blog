# C.04.01 圆大小头规则

圆形变径管（大小头）的套法兰/不套法兰处理规则。
![](/img/knowledge/pipe-fitting-spec/圆大小头样式1.webp)
---

## C.04.01.1 圆大小头（变径）

### C.04.01.1.1 两边套法兰时

> 圆法兰套入主体圆口，主体圆口需根据**偏斜度加大**，相关负责人制定尺寸。

### C.04.01.1.2 两边不套法兰时

| 部位   | 尺寸调整             | 说明                |
| ---- | ---------------- | ----------------- |
| 主体大口 | **+4mm** 左右      | —                 |
| 主体小口 | **+2mm** 左右      | —                 |
| 主体长度 | 剪掉两边咀长 + 加长 20mm | 咀长：60（特殊）或 80（常规） |
| 两边   | 接咀子              | 大口/小口需根据偏斜度加大     |
