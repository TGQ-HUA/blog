# I.06.01 CAMduct 简介

> Autodesk Fabrication CAMduct 是一款强大的 HVAC 制造与风管加工软件，同时也是生产管理工具，用于高效生产 HVAC 系统。CAMduct 使用全面的三维参数化管件图库来满足制造需求。

---

## 核心功能一览

| 功能 | 说明 |
|------|------|
| **扩充的制造商图库与模板** | 数千种英制内容项与展开图模板（圆形 ductboard、地漏、排气交叉管、堵头等） |
| **按材质差异化套料参数** | 可为保温、ductboard、金属板材等不同材质设置独立套料参数，避免重复套料 |
| **条码与 QR 码** | 支持在标签和工单上输出数据丰富的条码和 QR 码 |
| **配置库导入/导出** | 单命令完成配置库的导入导出，含保温规格选项 |
| **云端套料** | 利用云端算力，使用 10 种算法同时对单一工单进行套料 |
| **共享图库与配置库** | 所有 Autodesk Fabrication 产品（ESTmep/CADmep/CAMduct）共享图库和配置库，实现估算→深化→制造→安装无缝衔接 |
| **风管管件图库** | 丰富的三维参数化矩形、圆形、椭圆形风管管件及展开图 |
| **简洁图形界面** | 新手与熟练用户均易于操作 |
| **板材与线材套料** | 先进套料算法提高材料利用率 |
| **NC 码输出与机床支持** | 内置后处理器支持多种机床类型，控制任意规模生产线 |
| **高级工单成本工具** | 基于多因素估算工单成本 |
| **附加软件工具** | CAMduct Components、Tracker、RemoteEntry 等扩展工具 |
