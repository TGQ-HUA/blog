# I.06.06 板材管理（Sheet Management）

> 板材管理系统用于追踪库存板材，并记录板材使用后的剩余部分（余料/Remnant），以最大化板材利用率。

---

## 核心价值

- 不使用板材管理时，**无法知晓已用板材的剩余情况**。
- 对于**厚板、不锈钢等贵重材料**尤其重要。
- 板材管理中的材质必须已在主配置库（**Materials** 选项卡）中设置。

---

## 主要流程

1. **设置材质及板材尺寸** → 在配置库中定义板材规格
2. **套料时使用库存板材** → 自动/手动套料可优先使用库存余料
3. **切割后自动记录余料** → 系统自动更新余料库
4. **余料可用于后续工单** → 新工单套料时优先消耗余料

---

## 配置关联

- 套料配置中的 **New Sheet → Use Stock Sheets from Sheet Management** 选项
- 余料管理中的 **Only Save Sheet if Rip-Cut / Minimum Length / Discard Used Portion** 选项

> 详见 I.06.07 套料Nesting#71-套料基本配置 和 I.06.07 套料Nesting#712-余料管理
