# I.01 排版流程与检查清单

从接单到下单的完整排版流程——事前检查 → 排入项目 → 自动排版 → 下单前核对。

---

## I.01.1 排版前检查

| 步骤 | 内容 |
|------|------|
| 打印 | 风管加工清单 + 参图或原单 |
| 检查 | 特殊要求：表头表尾/材质/接口/备注 |
| 备注参图 | 原单中图编号备注 |
| 接口标记 | 两头接口不一样的重点标出 |

## I.01.2 排入项目

| 步骤 | 内容 |
|------|------|
| 找图库 | 共板方管/角铁方管/插条边方管/圆管/焊接方管/焊接圆管 |
| 找图型 | 按名称/数据/参图 |
| 输入 | 尺寸/编号/数量/接口/接缝一一对应 |
| 检查 | 项目少排？节数错？拼接缝错？节数 > 5 优先查 |

## I.01.3 自动排版

- 点击自动排版 → 查看就料 → 优化不就料
- 无法优化 → 反馈找原因

## I.01.4 下单流程

| 步骤 | 内容 |
|------|------|
| 接收 | 下单群接收下料清单 |
| 找资料 | 排版文件/清单/原单参图 |
| 清单核对 | 特殊要求与排版时一致，增修单特别注意 |
| 排版检查 | 尺寸/偏差高差/接口接缝/参图 |
| 方变圆/圆大小头 | 分厂师傅给出调整数值（催促） |
| 上传 | 上传到分厂下料微盘，文件标注后缀 |
| 登记 | 分厂下料管理登记（最后排查） |

> 材料调整步骤已取消（1250 转 1500 宽）。
