# [第4章 金属风管](/knowledge/pipe-fitting-spec/第4章-金属风管/)制作

> [!important] 章节定位
> 第4章是 GB 50738 **最核心的工艺章节**，详细规定镀锌钢板、不锈钢板、铝板等金属风管的制作全过程——从板材下料、咬口连接到法兰装配、风管加固，是 CAMduct 数据库配置的直接参考依据。

---

## 一、板材下料

### 1.1 剪切精度要求

| 项目 | 技术要求 | 检测方法 |
|------|----------|----------|
| **直线度偏差** | ≤ 1mm/m | 钢板尺 + 塞尺 |
| **对角线偏差** | 矩形风管板材展开后对角线之差 ≤ 2mm | 钢卷尺测两对角线 |
| **切割毛刺** | 必须去除，毛刺高度 ≤ 0.2mm | 游标卡尺 |
| **切口平整度** | 切口无裂纹、无分层、无夹渣 | 目测 + 10倍放大镜 |

### 1.2 咬口余量预留

| 咬口类型 | 咬口余量 | 适用场景 |
|----------|:--------:|----------|
| **单咬口** | 7～12mm | δ ≤ 0.6mm 低压风管 |
| **联合角咬口** | 10～15mm | δ = 0.6～1.0mm 中低压矩形风管 |
| **Pittsburgh 锁扣** | 20～25mm | δ = 0.6～1.5mm 中高压矩形风管 |
| **按扣式咬口 (Snap Lock)** | 6～8mm | δ ≤ 0.8mm 低压圆形风管 |

> [!warning] 板材下料要点
> - 板材剪切前必须核对材质证明、板厚、镀锌层重量（≥ 275g/m² 双面）
> - 下料时应考虑咬口余量、法兰翻边余量（角钢法兰翻边 ≥ 7mm）
> - 不锈钢板不得在地面直接拖拽，必须用木质或橡胶垫隔离
> - 铝板剪切时剪刃间隙应调至 0.02～0.04mm，避免产生毛刺

## 二、咬口连接工艺（⭐ 核心工艺）

### 2.1 咬口类型与选用

| 咬口类型 | 结构示意 | 适用板厚 | 压力等级 | 风管截面 |
|----------|----------|:--------:|:--------:|:--------:|
| **单咬口** | 板材边缘单次折弯咬合 | δ ≤ 0.6mm | 低压 ≤500Pa | 圆形/矩形 |
| **联合角咬口** | 两板边缘联合扣压成型 | δ = 0.6～1.0mm | 低中压 ≤1500Pa | 矩形 |
| **Pittsburgh 锁扣** | 公扣插入母扣槽锁紧 🔑 | δ = 0.6～1.5mm | 中高压 >500Pa | 矩形 |
| **按扣式咬口 (Snap Lock)** | 预成型扣件按压锁合 | δ ≤ 0.8mm | 低压 ≤500Pa | 圆形 |

### 2.2 各咬口类型施工工艺

#### （1）单咬口 — 工艺要点

| 工序 | 操作要求 |
|------|----------|
| **折弯** | 咬口折边宽度均匀，全程一致 |
| **扣合** | 手工或机械扣合，逐段压实 |
| **压平** | 锤击或滚压使咬口平整密合 |
| **质量检查** | 咬口严密、平整，不得有半咬口、张裂现象 |

#### （2）联合角咬口 — 工艺要点

| 工序 | 操作要求 |
|------|----------|
| **四角处理** | 四角咬口均匀，转角处无裂纹 |
| **扣压顺序** | 先扣合两长边，再扣合两短边 |
| **锤击修整** | 转角处用小锤轻敲修整，确保90°紧密贴合 |
| **密封** | 中压风管联合角咬口缝须涂密封胶 |

#### （3）Pittsburgh 锁扣 — 工艺要点（CAMduct 核心咬口）

| 工序 | 操作要求 |
|------|----------|
| **轧制成型** | 公扣高度 = 母扣槽深 + 0.5mm（确保锁合紧密） |
| **插入锁合** | 公扣全段均匀插入母扣槽 |
| **锤击锁紧** | 用木锤或橡胶锤沿锁扣全长敲击锁紧 |
| **检查** | 锁扣处无松动、无脱扣，锤击后不得有反弹 |

> [!tip] Pittsburgh 锁扣与 CAMduct
> 在 CAMduct 中，Pittsburgh Lock 的 Seam Allowance 参数应设为 **20～25mm**（母扣侧），公扣高度按板厚自动计算。这是中国项目中高压矩形风管最常用的纵向缝连接方式。

#### （4）按扣式咬口 (Snap Lock) — 工艺要点

| 工序 | 操作要求 |
|------|----------|
| **预成型** | 板材边缘轧制成按扣形状 |
| **按压锁合** | 两板对正后按压到位，听到「咔嗒」锁合声 |
| **检查** | 锁合紧密，不得有松动或未完全锁入 |

### 2.3 咬口连接的质量要求

| 检查项目       | 合格标准                       |
| ---------- | -------------------------- |
| **咬口缝外观**  | 平整、紧密、宽度均匀                 |
| **半咬口/张裂** | 不允许                        |
| **转角裂纹**   | 不允许                        |
| **咬口漏风**   | 中压系统咬口缝须涂密封胶               |
| **纵向缝位置**  | 风管纵向咬口缝必须错开布置，不得在一条直线上连续贯通 |

## 三、焊接工艺

### 3.1 焊接方式选用

| 焊接方式 | 适用材质 | 适用板厚 | 适用场景 |
|----------|----------|:--------:|----------|
| **氩弧焊（TIG）** | 不锈钢板 | δ ≥ 1.0mm | 不锈钢风管拼接、法兰焊接 |
| **气焊（氧乙炔）** | 铝板 | δ ≥ 2.0mm | 铝板风管拼接 |
| **电阻点焊** | 镀锌钢板 | δ ≤ 1.5mm | 风管加固筋与管壁点焊 |
| **CO₂保护焊** | 碳钢板 | δ ≥ 2.0mm | 厚钢板风管焊接 |

### 3.2 焊接工艺要求

| 项目 | 技术要求 |
|------|----------|
| **焊前清理** | 焊接区域20mm范围内清除油污、铁锈、氧化皮 |
| **坡口** | 板厚 ≥3mm 时开 V 形坡口，角度 60°±5° |
| **焊缝外观** | 表面无裂纹、气孔、夹渣、未熔合 |
| **咬边深度** | ≤ 0.5mm，连续长度 ≤100mm，总长 ≤ 焊缝总长10% |
| **焊后处理** | 不锈钢焊缝酸洗钝化；镀锌板焊缝清理焊渣后涂富锌漆 |

## 四、法兰制作

### 4.1 法兰类型与选用

| 法兰类型 | 适用场景 | 连接方式 |
|----------|----------|----------|
| **角钢法兰** | 各类金属风管通用，高压/大尺寸必备 | 角钢下料→焊接组框→钻孔→铆接 |
| **TDC/TDF 共板法兰** | 中低压矩形风管（工厂化预制） | 专用轧机轧制成型→套在风管端部→压合/铆接 |
| **扁钢法兰** | 小尺寸圆形风管（DN ≤ 450mm） | 扁钢弯圆对焊 |
| **C 型插条法兰** | 低压矩形风管 | S 型插条插入风管端部法兰槽 |

### 4.2 角钢法兰制作工艺

| 工序 | 操作要求 |
|------|----------|
| **角钢选型** | 按风管长边尺寸选角钢规格（L25×3～L50×5） |
| **下料** | 角钢切 45° 斜口，长度=风管外边长+翻边余量×2 |
| **焊接组框** | 四角对角焊接，焊缝饱满、无虚焊 |
| **钻孔** | 螺栓孔距 ≤150mm（高压）/≤100mm，四角必须有孔 |
| **铆接** | 法兰与风管用铆钉铆接，铆钉间距 ≤150mm |
| **精度控制** | 法兰面平面度 ≤2mm，对角螺栓孔偏差 ≤1mm |

### 4.3 共板法兰 (TDC/TDF) 制作工艺

| 工序 | 操作要求 |
|------|----------|
| **轧制成型** | 用 TDC/TDF 专用轧机将风管端部板边轧制成法兰形状 |
| **法兰角连接** | 四角用法兰角码连接，螺栓紧固 |
| **密封垫片** | 嵌入法兰槽内的橡胶密封条 |
| **适用范围** | 长边 ≤2000mm 的中低压矩形风管 |

> [!tip] 法兰垫片选型（关联[第7章 风管系统安装](/knowledge/pipe-fitting-spec/第7章-风管系统安装/)）
> | 输送介质 | 垫片材质 |
> |----------|----------|
> | 一般空气（≤70°C） | 闭孔海绵橡胶、橡胶板 |
> | 高温烟气（>70°C） | 石棉橡胶板、耐高温硅橡胶 |
> | 含油空气 | 耐油橡胶板 |
> | 洁净空调 | 不得产生粉尘的闭孔材料，垫片应内移不外露 |

## 五、风管加固

### 5.1 加固条件

金属风管在下列情况下必须进行加固：

| 条件 | 加固要求 |
|------|----------|
| 矩形风管长边 > 630mm | 必须采取加固措施 |
| 圆形风管直径 > 800mm | 必须采取加固措施 |
| 高压风管（>1500Pa） | 无论尺寸大小均须加固 |
| 风管管段长度 > 1250mm | 中间必须增设加固点 |

### 5.2 加固方式与间距

| 加固方式 | 适用场景 | 最大间距 |
|----------|----------|:--------:|
| **角钢加固框** | 中高压矩形风管 | ≤ 1250mm |
| **管内支撑加固（拉杆/撑杆）** | 大尺寸矩形风管（长边 > 1500mm） | ≤ 1250mm |
| **压筋加固（楞线加固）** | 低压矩形风管（δ ≤ 1.2mm） | ≤ 500mm |
| **点焊加固筋** | 镀锌钢板，δ ≤ 1.2mm 低压风管 | ≤ 400mm |
| **扁钢加固圈** | 圆形风管 | ≤ 1500mm |

### 5.3 加固工艺要求

| 项目 | 技术要求 |
|------|----------|
| **加固框与风管铆接** | 铆钉间距 ≤ 200mm，加固框与风管壁紧密贴合 |
| **拉杆/撑杆** | 拉杆直径 ≥ φ8mm，撑杆直径 ≥ φ10mm，两端加垫圈锁紧螺母 |
| **管内支撑点** | 支撑杆与风管接触处须设垫圈，防局部变形 |
| **加固后检查** | 加固处风管不得有凹陷、扭曲或漏风 |

## 六、允许偏差表

### 6.1 风管制作允许偏差

| 项目 | 允许偏差 | 检测方法 |
|------|:--------:|----------|
| 矩形风管长边尺寸 | ≤300mm：±1mm；>300mm：±2mm | 钢卷尺 |
| 矩形风管对角线之差 | ≤ 2mm（长边≤300mm）；≤ 3mm（长边>300mm） | 钢卷尺 |
| 风管平面度（任意1m²） | ≤ 5mm | 1m 钢板尺 + 塞尺 |
| 风管管口平面度 | ≤ 2mm | 钢板尺 + 塞尺 |
| 圆形风管直径 | ≤300mm：±1mm；>300mm：±2mm | 钢卷尺 |
| 圆形风管圆度 | ≤ 0.5% D（D为直径） | 钢卷尺测多方向直径 |
| 法兰面平面度 | ≤ 2mm | 钢板尺 + 塞尺 |
| 法兰对角螺栓孔偏差 | ≤ 1mm | 钢卷尺 |

### 6.2 咬口与焊接允许偏差

| 项目 | 允许偏差 |
|------|:--------:|
| 咬口宽度偏差 | ±1mm |
| 咬口平直度 | 偏差 ≤ 2mm/m |
| 焊缝余高 | ≤ 2mm |
| 焊缝宽度 | 坡口每侧增宽 ≤ 2mm |
| 角焊缝焊脚尺寸 | 不小于较薄板厚的 0.7 倍 |
