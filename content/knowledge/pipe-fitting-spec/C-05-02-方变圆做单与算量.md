# C.05.02 方变圆做单与算量

## C.05.02.1 方变圆

### C.05.02.1.1 结构拆分

| 部件      | 说明                                |
| ------- | --------------------------------- |
| **方接口** | 下对应接口的法兰框，角铁则直角在主体方口翻小边           |
| **主体**  | 方口长边 ≥ 250 → 咬口（拼接15），< 250 → 碰焊5 |
| **圆接口** | 活法兰或直边时下圆咀，固定法兰时直接焊接固定            |
|         |                                   |
![](/img/knowledge/pipe-fitting-spec/方变圆样式1.webp)
### C.05.02.1.2 变径形态（同矩形大小头）

| 形态 | 说明 |
|------|------|
| 对中 | 左右对中(宽) / 上下对中(高) / 全对中 |
| 平边 | 左平/右平(边平) / 顶平/底平(面平) |
| 偏差 | 左右偏移 / 上下高差 |
![](/img/knowledge/pipe-fitting-spec/方变圆样式2.webp)
> 复合交错需顶板俯视图 + 文字注释。详细排版参数见 [C.05.01 方变圆规则](/knowledge/pipe-fitting-spec/C-05-01-方变圆规则/)。

### C.05.02.1.3 做单示例

**方变圆【共板+咀】排版尺寸**：
```
共板框(方接口)：宽——800  高——400  长——60（50+10）
圆咀(圆接口)：  圆直径-Φ500  长——80（70+10）
主体：          宽——800  高——400  圆直径-Φ500  长——420
                方直位-50  宽偏移-对中  高偏移-对中
编号：Y7  成型方式：2片式  数量：1
材质/料厚：镀锌/0.75  接口：共板/咀  接缝：咬口
```

**客户常规表述**（无偏差 = 默认对中，带偏差需参图）：

| 客户说 | 含义 |
|--------|------|
| 方变圆(中变)：宽800\*高400-变-Φ500-长500—1节 | 对中方变圆 |

**报价单示例**：

| NO  | 材质  | 名称  | 规格(1)   | 规格(2)   | 长度  | 数量  | 厚度   | 接口  | 状态  |
| --- | --- | --- | ------- | ------- | --- | --- | ---- | --- | --- |
| Y7  | 镀锌  | 方变圆 | 800x400 | DN Φ500 | 500 | 1   | 0.75 | 咬口  | 半   |

### C.05.02.1.4 面积计算

```
(宽×2 + 高×2 + 3.14×圆直径 + 接缝×4) / 2 × (长 + 方接口)
```
