# 第5章 风管部件

第 5 章规定了风管系统中各类功能部件（风口、风阀、消声器、静压箱、检查门、柔性短管等）的质量验收要求。

---

## 5.1 风管部件分类

| 部件类别 | 典型产品 | 功能 | CAMduct 关联 |
|----------|----------|------|-------------|
| **风口类** | 百叶风口、散流器、喷口、格栅风口 | 送风/回风气流组织 | 末端标注 |
| **风阀类** | 调节阀、止回阀、防火阀、排烟阀、多叶调节阀 | 风量调节与安全控制 | 管件制造(弯头三通等) |
| **消声类** | 阻性消声器、抗性消声器、复合消声器 | 降低管路噪声 | 设备连接件 |
| **静压箱** | 送风/回风静压箱 | 均压稳流 | 特殊构件建模 |
| **功能性部件** | 检查门、导流叶片、测定孔 | 维护检修与测量 | 风管配件 |

---

## 5.2 主控项目

### 5.2.1 风口质量（5.2.1~5.2.3）

| 检验项目 | 要求 | 检验方法 |
|----------|------|----------|
| 风口外观 | 装饰面无明显划伤、压痕，叶片转动灵活 | 目测 + 手动检查 |
| 风口尺寸偏差 | 矩形风口对角线差 ≤ 3mm，圆形风口直径偏差 ±2mm | 钢卷尺 |
| 风口性能 | 风量、射程、噪声等应符合设计要求 | 查验检测报告 |

### 5.2.2 风阀质量（5.2.4~5.2.6）

| 风阀类型 | 核心要求 |
|----------|----------|
| **手动调节阀** | 启闭灵活，关闭严密，叶片间隙 ≤ 2mm |
| **止回阀** | 阀板启闭灵活，关闭时严密，水平安装时平衡锤杆须灵活可靠 |
| **电动/气动调节阀** | 执行机构动作应与阀体启闭同步；开度指示与阀板实际角度一致 |

### 🔴 5.2.7 — 防火阀与排烟阀（强制性条文）

> [!danger] 强制性条文 5.2.7
> **防排烟系统的柔性短管必须采用不燃材料。防火阀和排烟阀（排烟口）必须符合有关消防产品标准的规定，并具有相应的产品合格证明文件。**
>
> - **防火阀**：须符合 GB15930-2007 建筑通风和排烟系统用防火阀门 标准，70°C 自动熔断关闭
> - **排烟阀**：须符合消防产品标准，280°C 自动关闭，具有手动和自动开启功能
> - **CCC 认证**：防火阀、排烟阀属于强制性认证消防产品，必须有 **CCC 认证标志**（或有效的消防产品认证证书）
> - **进场检验**：每个阀门均须查验认证证书和出厂检验报告，**不合格产品严禁使用**

## 5.3 消声器（5.2.8~5.2.9）

| 检验项目 | 要求 |
|----------|------|
| 外观质量 | 外壳无锈蚀、凹陷，法兰平整 |
| 内部结构 | 吸声材料填充密实、无下沉，护面层完好无破损 |
| 声学性能 | 插入损失（消声量）应符合设计要求，查验声学检测报告 |
| 安装方向 | 阻性消声器安装方向须与气流方向标识一致，**不可反向** |

---

## 5.4 一般项目

| 条目 | 要求 |
|------|------|
| 5.3.1 | 风阀叶片间隙均匀、启闭标记清晰 |
| 5.3.2 | 电动风阀接线正确、接地可靠 |
| 5.3.3 | 风口与风管连接严密，无漏风 |
| 5.3.4 | 静压箱内壁光滑、清洁，无油污杂物 |
| 5.3.5 | 检查门开关灵活、密封良好，位置便于操作 |

---

## 5.5 部件与 CAMduct 的交接

在 CAMduct 工作流中，风管部件不在 CAMduct 内直接制造（CAMduct 主要处理风管及配件），但需在以下环节加以关注：

| CAMduct 环节 | 部件关联 |
|-------------|----------|
| **Service 标注** | 在风管上标注防火阀、调节阀的安装位置和类型代号 |
| **开孔标注** | 风口接管（Spigot / Collar）的尺寸和位置在管段上精确标注 |
| **法兰匹配** | 确保 CAMduct 输出的风管法兰与部件法兰的螺栓孔距和规格一致 |
| **消声器段** | 在 风管建模与设计 中预留消声器和静压箱的安装空间 |

> [!tip] 防火阀定位
> 根据 [第6章 风管系统安装](/knowledge/pipe-fitting-spec/第6章-风管系统安装/)|第6章 6.2.3 强条要求，防火阀距防火墙表面 ≤ 200mm。在 CAMduct 输出的管段图中应精确标注此距离。
