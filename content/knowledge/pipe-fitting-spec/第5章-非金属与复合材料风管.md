# 第5章 非金属与复合材料风管制作

> [!important] 章节定位
> 第5章规定玻璃钢（FRP）、酚醛复合板、聚氨酯复合板、玻纤复合板等非金属及复合材料风管的制作工艺。此类风管兼具**结构强度**与**保温性能**，广泛用于对保温、防腐、洁净度有特殊要求的空调系统。

---

## 一、非金属风管分类与适用条件

| 风管类型 | 材质构成 | 适用场景 | 不适用场景 |
|----------|----------|----------|------------|
| **玻璃钢风管 (FRP)** | 玻璃纤维增强不饱和聚酯树脂 | 化工防腐、地下工程、潮湿环境 | — |
| **酚醛复合风管** | 酚醛泡沫芯材 + 双面铝箔 | 一般空调、洁净空调（B1级难燃） | 高压系统（>1500Pa） |
| **聚氨酯复合风管** | 聚氨酯硬泡芯材 + 双面铝箔/彩钢 | 一般空调、保温要求高场合 | 高压系统、排烟系统 |
| **玻纤复合风管** | 离心玻璃棉板 + 复合玻纤布面层 | 消声要求高的空调系统 | 含尘/含油烟系统 |
| **无机玻璃钢风管** | 氯氧镁水泥 + 玻璃纤维网格布 | 防排烟风管 | 一般空调（自重大） |

---

## 二、玻璃钢风管 (FRP) 制作工艺

### 2.1 原材料要求

| 材料 | 技术要求 |
|------|----------|
| **树脂** | 不饱和聚酯树脂，固化后氧指数 ≥ 30%（阻燃型） |
| **玻璃纤维** | 无碱或中碱玻璃纤维布/毡，不得使用石棉制品 |
| **填料** | 氢氧化铝等阻燃填料，添加量 ≤ 树脂量 30% |
| **固化剂 + 促进剂** | 按厂家配比使用，严禁超量 |

### 2.2 制作工艺流程

```
模具清理 → 涂脱模剂 → 铺贴玻璃纤维布 → 涂刷树脂 →
逐层铺贴（交替玻纤布与树脂）→ 固化 → 脱模 → 修边 → 检验
```

### 2.3 铺贴工艺要求

| 项目 | 技术要求 |
|------|----------|
| **铺层顺序** | 表面毡 → 短切毡 → 玻纤布交替铺层 |
| **树脂含量** | 风管壁树脂含量 50%～55%（质量比） |
| **每层含胶量** | 均匀浸润，无白斑、无气泡 |
| **层间结合** | 每层铺贴后用压辊滚压排气，确保层间密实 |
| **固化时间** | 常温固化 ≥ 24h，或加温固化（60～80°C，4～6h） |

### 2.4 法兰制作

| 项目 | 技术要求 |
|------|----------|
| **法兰形式** | 与风管整体成型（模压法兰边）或后粘接 |
| **法兰宽度** | ≥ 30mm（长边≤500mm），≥ 40mm（长边>500mm） |
| **法兰厚度** | ≥ 风管壁厚 + 2mm |
| **螺栓孔** | 预埋金属衬套或固化后钻孔，孔距 ≤ 120mm |

### 2.5 质量要求

| 项目 | 合格标准 |
|------|----------|
| **外观** | 表面平整光滑，无气泡、无裂纹、无分层 |
| **巴氏硬度** | ≥ 40 HBa |
| **树脂固化度** | ≥ 90%（丙酮擦拭法，不粘手不发黏） |
| **壁厚偏差** | ≤ ±0.5mm（设计壁厚 ≤ 3mm），≤ ±0.8mm（设计壁厚 > 3mm） |

## 三、酚醛复合风管制作工艺

### 3.1 酚醛板特性

| 特性 | 参数 |
|------|------|
| **芯材** | 酚醛泡沫（Phenolic Foam） |
| **面层** | 双面压花铝箔（厚度 ≥ 0.06mm） |
| **燃烧性能** | B1 级（难燃），氧指数 ≥ 38% |
| **导热系数** | ≤ 0.025 W/(m·K) |
| **使用温度** | -50°C ~ +120°C |

### 3.2 制作工艺流程

```
板材切割（专用切割机/开槽刀）→ 开 V 形槽（45°）→
涂专用胶粘剂 → 折弯成型 → 插入法兰/插条 → 密封胶封缝 →
铝箔胶带外贴接缝 → 检验
```

### 3.3 关键工艺参数

| 工序 | 技术要求 |
|------|----------|
| **开槽** | V 形槽角度 45°，槽底留 0.5～1mm 铝箔面层（不得切断） |
| **涂胶** | 专用酚醛复合板胶粘剂，双面涂胶，均匀无漏涂 |
| **折弯加压** | 折弯后立即加压固定，保持压力至胶粘剂初凝（通常 ≥ 30min） |
| **内角密封** | 风管内四角接缝涂密封胶，气密防潮 |
| **外贴铝箔** | 所有外接缝用 50mm 宽铝箔胶带密封 |

### 3.4 法兰连接

| 项目 | 技术要求 |
|------|----------|
| **法兰类型** | 铝合金插条式法兰、镀锌钢角码法兰、PVC 法兰 |
| **法兰安装** | 插入式法兰必须插入到位，用自攻螺钉固定（间距 ≤ 200mm） |
| **法兰密封** | 法兰面贴 ≥ 5mm 厚闭孔海绵密封条 |

## 四、聚氨酯复合风管制作工艺

### 4.1 聚氨酯板特性

| 特性 | 参数 |
|------|------|
| **芯材** | 聚氨酯硬质泡沫（PUR/PIR） |
| **面层** | 双面彩钢板 / 压花铝箔 |
| **燃烧性能** | B1 级（PIR 改性聚氨酯）或 B2 级（普通 PUR） |
| **导热系数** | ≤ 0.022 W/(m·K) |
| **使用温度** | -50°C ~ +100°C |

### 4.2 制作工艺要点

| 工序 | 技术要求 |
|------|----------|
| **切割** | 用专用开槽刀或 CNC 切割机，精度 ±0.5mm |
| **开槽** | 同酚醛板，V 形槽 45°，留面层不断 |
| **涂胶** | 聚氨酯专用胶粘剂，双面涂胶 |
| **折弯成型** | 折弯后夹具固定 20～30min |
| **接缝处理** | 内角涂密封胶 + 外贴铝箔胶带 |

### 4.3 与酚醛板的主要区别

| 对比项 | 酚醛复合风管 | 聚氨酯复合风管 |
|--------|--------------|----------------|
| **阻燃性** | 更优（B1，难燃，离火自熄） | B1（PIR）或 B2（PUR） |
| **保温性** | 好（λ = 0.025） | 更好（λ = 0.022） |
| **价格** | 较贵 | 经济 |
| **烟毒性** | 低烟低毒 | 燃烧时发烟量较大 |
| **适用风压** | ≤ 1500Pa | ≤ 1500Pa（PUR建议 ≤ 800Pa） |

---

## 五、玻纤复合风管制作工艺

### 5.1 特性与适用

| 项目 | 说明 |
|------|------|
| **材质** | 高密度离心玻璃棉板（密度 ≥ 64kg/m³）+ 复合玻纤布面层 |
| **主要优势** | 兼具**风管结构** + **消声功能**，省去独立消声器 |
| **适用系统** | 低速空调送风系统（风速 ≤ 10m/s） |
| **严禁使用** | 厨房排油烟、工业含尘气体、高温烟气、消防排烟系统 |

### 5.2 制作工艺

| 工序 | 技术要求 |
|------|----------|
| **板材切割** | 专用刀具切割，切口平整无撕裂 |
| **开槽** | 内面层留连接翼（翻边），开槽深度 = 板厚 - 内面层厚度 |
| **连接** | 板材拼接用专用胶粘剂 + 骑马钉/扒钉固定 |
| **内面层密封** | 所有接缝用玻纤布条 + 树脂/胶粘剂密封 |

## 六、各类型风管允许偏差对比

| 项目 | 玻璃钢 | 酚醛/聚氨酯复合 | 玻纤复合 |
|------|:------:|:--------------:|:--------:|
| 长边尺寸偏差 | ±2mm | ±1.5mm | ±2mm |
| 对角线偏差 | ≤3mm | ≤2mm | ≤3mm |
| 壁厚偏差 | ±0.5mm | —（板材定厚） | —（板材定厚） |
| 法兰面平面度 | ≤2mm | ≤2mm | ≤2mm |
| 拼接缝间隙 | ≤1mm | ≤0.5mm | ≤1mm |
