# 第7章 风管系统安装

> [!important] 章节定位
> 第7章是 GB 50738 最关键的**安装工艺章**，涵盖支吊架制作安装、风管连接组对、穿越结构处理、法兰垫片选型、柔性短管安装等核心内容。安装质量直接决定系统的严密性、安全性和使用寿命。

---

## 一、支吊架设计与安装

### 1.1 支吊架间距

| 风管类型 | 水平风管最大间距 | 垂直风管最大间距 |
|----------|:----------------:|:----------------:|
| **非保温金属风管** | ≤ 3.0m | ≤ 4.0m |
| **保温金属风管** | ≤ 2.5m | ≤ 4.0m |
| **非金属风管（玻璃钢）** | ≤ 2.0m | ≤ 3.0m |
| **复合材料风管（酚醛/聚氨酯）** | ≤ 2.0m | ≤ 3.0m |
| **垂直风管（所有类型）** | — | **每层楼板至少 1 个固定支架** |

### 1.2 支吊架距风口/阀门的要求

| 位置 | 最大距离 |
|------|:--------:|
| **支吊架距风口** | ≤ 200mm |
| **支吊架距风阀** | ≤ 200mm |
| **支吊架距防火阀** | ≤ 200mm（保证熔断动作不被阻碍） |
| **支吊架距弯头/三通** | ≤ 300mm |
| **支吊架距柔性短管** | ≤ 300mm |

### 1.3 支吊架制作与防腐

| 项目 | 技术要求 |
|------|----------|
| **悬吊方式** | 镀锌螺杆（≥ M8）+ 膨胀螺栓/化学锚栓固定 |
| **严禁事项** | 🔴 严禁将支吊架焊接到建筑结构钢筋上 |
| **横担** | 角钢或槽钢，宽度 ≥ 风管宽度 + 100mm（两侧各50mm余量） |
| **抱箍** | 圆形风管用扁钢抱箍（≥ 25×3mm），衬橡胶垫防磨损 |
| **防腐** | 碳钢支吊架必须热浸镀锌或涂刷防锈漆 ≥ 两道 |
| **抗震支吊架** | 防排烟风管、穿越抗震缝的风管必须设置 |

### 1.4 支吊架间距计算速查

| 风管长边 (mm) | 金属非保温 (m) | 金属保温 (m) | 非金属 (m) |
|:-------------:|:---------------:|:------------:|:----------:|
| ≤ 400 | 3.0 | 2.5 | 2.0 |
| 400 ~ 630 | 3.0 | 2.5 | 2.0 |
| 630 ~ 1000 | 3.0 | 2.5 | 2.0 |
| 1000 ~ 1500 | 3.0 | 2.5 | 2.0 |
| 1500 ~ 2000 | 2.5 | 2.5 | 2.0 |
| > 2000 | 2.5 | 2.5 | — |

## 二、风管穿越结构墙体/楼板

> [!warning] 强制性要求（GB 50738 第 7.2.4 条）
> 风管穿越防火墙、楼板时必须严格遵守以下规定：

### 2.1 穿越防火墙

| 项目 | 技术要求 |
|------|----------|
| **预埋套管** | 🔴 必须设置**钢制预埋套管**，壁厚 **≥ 1.6mm** |
| **套管间隙封堵** | 套管与风管之间必须用**不燃柔性材料**封堵密实 |
| **防火阀位置** | 🔴 防火阀距防火墙表面 **≤ 200mm** |
| **阀-墙间风管** | 防火阀与防火墙之间的风管必须用 **≥ 1.6mm** 厚钢板制作 |
| **风管材料** | 穿越防火分区的风管本体、框架、固定材料均为**不燃材料** |

### 2.2 穿越楼板

| 项目 | 技术要求 |
|------|----------|
| **套管高出楼面** | 高出地面 ≥ 50mm（防水要求） |
| **套管壁厚** | ≥ 1.6mm（同防火墙要求） |
| **封堵材料** | 不燃柔性材料 + 防火密封胶 |
| **垂直风管固定** | 每层楼板处必须设固定支架（角钢框架+膨胀螺栓） |

### 2.3 穿越普通墙体

| 项目 | 技术要求 |
|------|----------|
| **套管** | 宜设套管（非强制），壁厚 ≥ 0.75mm |
| **封堵** | 柔性材料封堵，无漏风 |
| **间隙** | 风管与墙体孔洞间隙 ≥ 50mm（保温风管≥100mm，为保温层留空间） |

## 三、风管连接组对

### 3.1 法兰连接

| 工序 | 操作要求 |
|------|----------|
| **垫片安装** | 垫片整条连续、**45° 斜口搭接**（非对接），不得有断开或缺口 |
| **垫片位置** | 垫片不得凸入风管内部（凸入会增加阻力、积尘），应与法兰面齐平 |
| **螺栓紧固** | 🔑 **十字对角分 2～3 轮均匀拧紧**，不得一轮拧死 |
| **螺栓方向** | 全部螺栓方向一致（螺母朝同一侧），便于检查 |
| **螺栓间距** | 低压 ≤150mm；中压 ≤120mm；高压 ≤100mm |
| **法兰平行度** | 两法兰面贴合紧密，局部间隙 ≤ 2mm |

### 3.2 插接式连接（S 型插条 / C 型插条）

| 工序 | 操作要求 |
|------|----------|
| **插接深度** | 插条插入重叠长度 **≥ 25mm** |
| **固定方式** | 插入到位后用自攻螺钉固定，螺钉间距 **≤ 300mm** |
| **四边插条** | 四边插条均需正确插入，不得遗漏 |
| **密封** | 中压系统插接缝须涂密封胶 |

### 3.3 风管纵向缝错开

> [!important]
> 风管纵向咬口缝或焊接缝必须**错开布置**，不得在一条直线上连续贯通（防止应力集中和漏风）。

## 四、法兰垫片选型

| 输送介质 / 工况 | 垫片材质 | 厚度 |
|----------------|----------|:----:|
| **一般空气（≤ 70°C）** | 闭孔海绵橡胶 | 3～5mm |
| **一般空气（≤ 70°C）** | 橡胶板（δ=3～5mm） | 3～5mm |
| **高温烟气（> 70°C）** | 石棉橡胶板、耐高温硅橡胶 | 3～5mm |
| **含油空气** | 耐油橡胶板 | 3～5mm |
| **洁净空调** | 闭孔海绵橡胶（不得产生粉尘，垫片内移与风管内壁齐平） | 3～5mm |
| **防排烟系统** | 不燃材料（膨胀型防火密封条/陶瓷纤维垫片） | 按产品要求 |

> [!warning] 垫片安装通病
> - ❌ 垫片断开/对接 → 漏风
> - ❌ 垫片凸入风管内 → 积尘 + 增加阻力
> - ❌ 多垫重叠 → 法兰不平
> - ❌ 洁净空调垫片外露 → 积尘源

## 五、柔性短管安装（帆布接头）

### 5.1 安装位置

- 风机进出口（最主要）
- 穿越伸缩缝/沉降缝
- 与振动设备连接处

### 5.2 安装要求

| 项目 | 技术要求 |
|------|----------|
| **长度** | 🔑 **150～300mm**（过短减振无效；过长下垂增加阻力） |
| **材质** | 一般空调：帆布/人造革；防排烟：🔴 **不燃材料**（硅胶玻纤布） |
| **安装状态** | 自然状态下安装，不得拉伸或扭曲 |
| **不得替代** | 🔴 不得代替弯头/变径管使用 |
| **防结露** | 冷风系统柔性短管外应做保温，且不影响柔性变形 |
| **固定** | 两端法兰压紧，螺栓间距 ≤ 100mm |

### 5.3 柔性短管常见问题

| 问题 | 原因 | 影响 |
|------|------|------|
| 过长下垂 | 安装过长或风机振动 | 局部阻力增大 → 风量下降 |
| 扭曲安装 | 法兰错位强行连接 | 短管撕裂漏风 |
| 未保温 | 遗漏 | 结露滴水 |
| 材质不阻燃 | 错用帆布于排烟系统 | 🔴 火灾安全隐患 |
