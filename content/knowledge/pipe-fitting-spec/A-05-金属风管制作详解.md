# A.05 金属风管制作详解

> 矩形与圆形金属风管的全部接合形式——接口、接缝、超尺寸拼接缝及风管加固的详细制作规范。

---

## A.05.1 接合形式总览

| 大类         | 定义                   | 用途    |
| ---------- | -------------------- | ----- |
| **接口**     | 一节管与另一节管首尾相连的端口接管形式  | 管段间连接 |
| **接缝**     | 多块料件组合成一节管的边缝合管形式    | 单节成型  |
| **超尺寸拼接缝** | 料件限于材料板宽分割成多块的组合拼接形式 | 超大管件  |

## A.05.2 矩形风管 — 接口

### A.05.2.1 共板式法兰（共板法兰）

![](/img/knowledge/pipe-fitting-spec/共板2.webp)

| 项目 | 内容 |
|------|------|
| **特点** | 加工速度快、节省材料、降低能耗、节省运输费用 |
| **用料** | 45mm |
| **制作** | 标准直管：流水线直接压制成连体法兰 |
|  | 非标直管/弯头/三通/四通/配件：下料后单机压制成型 |
| **配件** | 角码、卡子、密封垫、螺栓、螺母、垫片 |

**变种**：

| 变种 | 区别 | 弊端 |
|------|------|------|
| **内共板法兰** | 折向管内 | 标准直管需人工切角，加强筋为凹筋（常规为凸筋） |
| **共板法兰打孔** | 对接面按要求打螺丝孔 | 更费工，需额外增加加工费 |

### A.05.2.2 角铁法兰
![](/img/knowledge/pipe-fitting-spec/角铁2.webp)

| 项目       | 内容                             |
| -------- | ------------------------------ |
| **特点**   | L 型 90°，强度大、稳固，可随意拆卸和改变布局，方便维护 |
| **翻边用料** | 7mm                            |
| **制作**   | ①下料、打孔 → ②焊接 → ③铆接/焊接到管口       |
| **配件**   | 螺栓、螺母、垫片                       |

**法兰型号规格标准**：

| 风管边长 b / 直径 D | 国标 (边宽×厚 mm) | 本厂方法兰 (材料×规格 mm) | 本厂圆法兰 (材料规格料厚 mm) |
|----------------------|---------------------|----------------------------|-------------------------------|
| d(D) ≤ 630 | 25×3 | 30×30 | 30×20×2.0 最大 D700 |
| 630 < d(D) ≤ 1500 | 30×3 | 30×40 | 30×20×2.5 最大 D1600 |
| 1500 < d(D) ≤ 2500 | 40×4 | 40×40 | 40×20×4.0(5.0) |
| 2500 < d(D) ≤ 4000 | 50×5 | 50×50 | 50×20×5.0 |

### A.05.2.3 插条边
![](/img/knowledge/pipe-fitting-spec/插条边.webp)

| 项目 | 内容 |
|------|------|
| **特点** | 密封性强、工序简单、成本低 |
| **用料** | 15mm |
| **制作** | 通过 C 型插条插入翻边 180° 的插条边，牢固连接 |
| **配件** | C 型插条 |

### A.05.2.4 直边、堵
![](/img/knowledge/pipe-fitting-spec/直边+堵2.webp)

| 类型 | 用料 | 说明 |
|------|------|------|
| **直边** | 0mm | 利口边，默认保留 50mm 直位。需加固/加强时可做扣边（插条边） |
| **堵** | 接口 30mm | 接口对接封堵板，封板四边用料 7mm（同联合咬口） |

### A.05.2.5 翻边、多段折边
![](/img/knowledge/pipe-fitting-spec/方管翻边2.webp)

**翻边**：
- 分**内折边**和**外折边**
- 常规尺寸：**30mm**
- 可选尺寸：10 / 15 / 20 / 25 / 30 / 35 / 40 / 50mm

**多段折边**（一般为内折边，多用于烟罩）：

| 段数 | 常规尺寸 |
|------|----------|
| 两段 | 50 + 20 |
| 三段 | 50 + 30 + 20 |

## A.05.3 矩形风管 — 接缝

### A.05.3.1 联合咬口（大小边）
![](/img/knowledge/pipe-fitting-spec/大小边.webp)

| 组成部分 | 别名 | 用料 | 默认位置 |
|----------|------|------|----------|
| **公口** | 小边 / 单边 | 7mm | 靠长边 |
| **母口** | 大边 / 双边 | 30mm | 靠短边 |

制作：标准直管流水线直接成型；非标件下单机压制成型。

### A.05.3.2 螺丝
![](/img/knowledge/pipe-fitting-spec/螺丝-接缝.webp)

| 项目 | 内容 |
|------|------|
| **特点** | 搭 30mm，对穿孔锁螺丝 |
| **制作** | 激光下料 → 开螺丝孔 → 单边 30mm 折弯 90° → 孔对穿 |

### A.05.3.3 搭边焊
![](/img/knowledge/pipe-fitting-spec/大边20+直焊边.webp)

| 类型 | 搭接量 | 工艺 |
|------|--------|------|
| 搭 30mm | 30mm | 碰焊机碰焊或人工焊接 |
| 搭 10mm 左右 | ~10mm | 焊接 |

制作：激光下料 → 单边 30mm 折弯 90°

### A.05.3.4 直边焊

直边焊接，开料无附加耗材。

### A.05.3.5 超板宽尺寸咬口拼接缝（平咬口）
![](/img/knowledge/pipe-fitting-spec/咬口拼接.webp)

| 项目       | 内容                           |
| -------- | ---------------------------- |
| **触发条件** | 料件展开超出激光割料板宽（1240mm）         |
| **处理**   | 拆成多块平咬口拼接                    |
| **制作**   | 单机设备压制成型，单条拼接缝 **20 + 20mm** |

## A.05.4 圆形风管 — 接口

### A.05.4.1 管口楞筋（滚压内筋/外筋）
![](/img/knowledge/pipe-fitting-spec/滚内外筋.webp)
默认所有管口 **30mm** 位置压一道**外筋**。特殊要求以生产单备注为准。

### A.05.4.2 翻边、剪口翻边
![](/img/knowledge/pipe-fitting-spec/圆管外翻.webp)

| 参数 | 值 |
|------|-----|
| **默认翻边** | 10mm |
| **可选尺寸** | 8 / 13 / 15mm |
| **不可翻边条件** | 口径 > 2000mm 或 料厚 > 2.0mm |
| **剪口翻边** | 翻边尺寸要求 > 15mm 时需做剪口翻边 |

### A.05.4.3 插接
![](/img/knowledge/pipe-fitting-spec/圆管插接.webp)

| 项目 | 内容 |
|------|------|
| **方式** | 用内接使两个管件相连 |
| **异形件处理** | 管口 **-3** 插入直管件 |
| **限制** | 管口径 > 800mm **不建议**使用插接形式 |

### A.05.4.4 圆角钢法兰
![](/img/knowledge/pipe-fitting-spec/圆法兰.webp)

| 项目 | 内容 |
|------|------|
| **特点** | 强度大、稳固，可随意拆卸和改变布局，方便维护 |
| **翻边用料** | 10mm |
| **制作** | 角铁通过特制设备卷弯，加工后保持两条边垂直 90° |
| **配件** | 螺栓、螺母、垫片 |
| **规格标准** | 同角铁法兰规格表（见 2.2 节） |

## A.05.5 圆形风管 — 接缝

### A.05.5.1 平咬口
![](/img/knowledge/pipe-fitting-spec/圆管平咬口.webp)
圆形风管接缝咬合连接口。单机压制成型，单条拼接缝 **15 + 15mm**。

### A.05.5.2 立咬口
![](/img/knowledge/pipe-fitting-spec/圆管立咬口.webp)
圆形风管**分段**咬合连接口（常用于圆弯头）。单机压制成型，单条拼接缝 **15 + 15mm**。

### A.05.5.3 碰焊
![](/img/knowledge/pipe-fitting-spec/圆管碰焊.webp)
叠 5mm 或 30mm，碰焊机碰焊。

## A.05.6 风管加固

### A.05.6.1 加固目的

管径或边长较大的风管，断面易变形，运行中因振动产生噪声。加固就是为了避免断面变形和减少噪声。

### A.05.6.2 圆形风管加固

| 规则 | 内容 |
|------|------|
| **一般** | 圆形风管刚性强，两端法兰起一定加固作用，一般**不作加固处理** |
| **需加固条件** | 直径 ≥ 800mm **且** 管段长度 > 1250mm，或总表面积 > 4m² |
| **中段加固** | 每隔 1500mm 加设一个扁钢加固圈（抱箍），用铆钉固定 |
| **接缝加固** | 直径 > 500mm，纵向咬口两端用铆钉或点焊固定（防止运输/吊装裂开） |
| **法兰要求** | 直径 ≥ 800mm 务必两端法兰连接 |

### A.05.6.3 矩形风管加固
![](/img/knowledge/pipe-fitting-spec/管件加固.webp)

| 加固方式 | 细则 |
|----------|------|
| **角铁框加固** | 用角钢框加固，强度大；角钢规格可略小于法兰规格 |
| **加强筋** | 间距 ≤ 300mm，靠近法兰处间距 ≤ 150mm；凸出部分在风管外表面；共板法兰风管均应轧制 |
| **螺杆内支撑** | 需打加强孔；两端专用垫圈应置于风管受力（压）面 |

**矩形风管需加固条件**：

| 条件 | 加固阈值 |
|------|----------|
| 边长 | > 630mm |
| 保温风管边长 | > 800mm |
| 管段长度 | > 1250mm |
| 低压平面积 | > 1.2m² |
| 中/高压平面积 | > 1.0m² |

> 满足**任一**条件即需加固。
