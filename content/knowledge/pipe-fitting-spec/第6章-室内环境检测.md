# 第6章 室内环境检测

第 6 章规定了 HVAC 系统投运后**室内环境参数**的检测方法，包括温度、相对湿度、风速、噪声（A 声级）和照度。这些参数是评估空调通风系统是否达到设计效果的**最终判定依据**。

---

## 6.1 温度检测

### 6.1.1 检测仪表

| 仪表类型 | 精度 | 适用场景 |
|----------|:---:|----------|
| **水银温度计** | ±0.1°C | 标准比对、校准用 |
| **数字温度计（热电偶/热敏电阻）** | ±0.2°C | 现场测试（推荐） |
| **红外测温仪** | ±1°C | 表面温度快速筛查 |
| **自记式温度计** | ±0.5°C | 长时间连续监测 |

### 6.1.2 测点布置

| 参数 | 要求 |
|:----:|------|
| **测点高度** | 距地面 **1.2 m**（人员活动区呼吸带高度） |
| **测点距墙** | ≥ 0.5 m |
| **测点数量** | 房间面积 ≤ 16m²：1 点；16~30m²：2 点；30~60m²：3 点；> 60m²：每增加 20m² 加 1 点 |
| **测量时间** | 系统稳定运行 ≥ 1h 后进行 |

### 6.1.3 判定标准

| 参数 | 允许偏差 | 说明 |
|:----:|:------:|------|
| 空调房间温度 | 设计值 **± 1°C** | 舒适性空调 |
| 工艺性空调（恒温） | 设计值 **± 0.5°C** | 精密空调 |
| 供暖房间温度 | 设计值 +2°C / -1°C | 冬季供暖 |

---

## 6.2 相对湿度检测

### 6.2.1 检测仪表

| 仪表类型 | 精度 | 适用场景 |
|----------|:---:|----------|
| **通风干湿球温度计** | ±2% RH | 标准方法 |
| **电子湿度计（电容/电阻式）** | ±3% RH | 现场测试（推荐） |
| **露点仪** | ±0.5°C DP | 精确测量、低温环境 |
| **自记式温湿度计** | ±3% RH | 连续监测 |

### 6.2.2 测点布置

同温度测点，温湿度可同步测量。

### 6.2.3 判定标准

| 参数 | 允许偏差 |
|:----:|:------:|
| 舒适性空调相对湿度 | 设计值 **± 5%** RH |
| 工艺性空调相对湿度 | 设计值 **± 3%** RH（精密级） |

---

## 6.3 风速检测

### 6.3.1 检测仪表

| 仪表类型 | 量程 (m/s) | 精度 | 适用场景 |
|:------:|:--------:|:---:|----------|
| **热球风速仪** | 0.05~30 | ±3% 读数 | 风口风速、室内微风速（推荐） |
| **叶轮风速仪** | 0.3~40 | ±2% 读数 | 风管风速、大风口 |
| **毕托管 + 微压计** | 1~50 | ±1% | 风管内部风速（标准方法） |

### 6.3.2 风口风速检测

| 风口类型 | 测点布置 | 读数 |
|----------|----------|------|
| **散流器/百叶风口** | 风口格栅外 50mm 处，分格法网格测量 | 各点取平均值 |
| **条缝风口** | 沿条缝长度方向均匀 3~5 点 | 各点取平均值 |
| **喷口** | 喷口出口截面中心 | 直接读数 |

### 6.3.3 判定标准

| 检测项 | 标准 |
|:----:|------|
| 风口风速 | 设计值 ± 10% |
| 工作区风速（舒适性空调） | 冬季 ≤ 0.2 m/s，夏季 ≤ 0.3 m/s |
| 工作区风速（工艺性空调） | 按设计值 |

---

## 6.4 噪声检测

### 6.4.1 声级测量

| 参数 | 要求 |
|:----:|------|
| **测量量** | **A 声级**，dB(A) |
| **仪表** | 精密声级计（1 型）或普通声级计（2 型），测量前后须校准 |
| **测点高度** | 距地面 **1.2 m** |
| **测点距墙** | ≥ 1.5 m |
| **测点距窗** | ≥ 1.5 m |
| **测量时间** | 稳态噪声测 ≥ 15s；非稳态测代表性时段 |

### 6.4.2 测点布置

| 房间面积 S (m²) | 最少测点数 |
|:--------------:|:--------:|
| S ≤ 50 | 1 |
| 50 < S ≤ 100 | 2 |
| 100 < S ≤ 200 | 3 |
| S > 200 | 每增加 100m² + 1 点 |

### 6.4.3 背景噪声修正

| 总噪声与背景噪声差值 (dB) | 修正值 (dB) |
|:---------------------:|:--------:|
| ≥ 10 | 0 |
| 6~9 | -1 |
| 4~5 | -2 |
| 3 | -3 |
| < 3 | 测量无效（须降低背景噪声） |

> 实测噪声 = 总噪声读数 - 背景噪声修正值

### 6.4.4 判定标准

| 建筑类型 | 允许噪声 dB(A) |
|----------|:------------:|
| 住宅卧室 | ≤ 30（夜间）/ ≤ 37（昼间） |
| 办公、会议室 | ≤ 40 |
| 商场、餐厅 | ≤ 50 |
| 工业厂房 | 按设计 / GBZ 2.2 |
| 洁净室 | ≤ 60 |

---

## 6.5 照度检测

### 6.5.1 检测方法

| 参数 | 要求 |
|:----:|------|
| **仪表** | 照度计（精度 ±5%） |
| **测点高度** | 距地面 **0.75 m**（桌面高度） |
| **测点布置** | 等面积网格法，网格数按房间面积确定 |

### 6.5.2 判定标准

| 房间类型 | 平均照度 (lx) |
|----------|:----------:|
| 办公室 | ≥ 300 |
| 设计室/绘图室 | ≥ 500 |
| 走廊 | ≥ 50 |
| 设备机房 | ≥ 100 |

> 照度均匀度 = 最小照度 ÷ 平均照度 ≥ 0.7

---

## 6.6 室内环境检测汇总

| 检测项目 | 仪表 | 测点高度 | 关键指标 |
|:------:|------|:------:|------|
| 温度 | 数字温度计 ±0.2°C | 1.2 m | ± 1°C |
| 相对湿度 | 电子湿度计 ±3%RH | 1.2 m | ± 5% RH |
| 风速 | 热球/叶轮风速仪 | 工作区 1.2m | 冬季 ≤ 0.2m/s |
| 噪声 | 声级计 (A 声级) | 1.2 m | 按建筑类型 |
| 照度 | 照度计 | 0.75 m | 按房间功能 |
