# 第6章 风管部件与消声器制作

> [!important] 章节定位
> 第6章规定风口、风阀、消声器、静压箱、柔性短管等风管系统部件的制作与组装要求。这些部件是风管系统的**功能节点**，直接关系到系统气流组织、风量调节、消声降噪和防火安全性能。

---

## 一、风口与百叶制作

### 1.1 风口分类

| 风口类型 | 功能 | 常见形式 |
|----------|------|----------|
| **送风口** | 向空调区域送风 | 百叶风口、散流器、喷口、旋流风口 |
| **回风口** | 从空调区域回风 | 单层/双层百叶、格栅风口 |
| **排风口** | 排除室内空气 | 百叶风口、格栅风口 |
| **新风口** | 引入室外新风 | 防雨百叶风口（带防虫网） |

### 1.2 风口制作要求

| 项目 | 技术要求 |
|------|----------|
| **材质** | 铝合金型材（主材）、镀锌钢板、不锈钢板 |
| **百叶叶片** | 叶片间距均匀、转动灵活，调节连杆联动可靠 |
| **外框** | 框架对角焊接或角码连接，外框对角线偏差 ≤ 2mm |
| **表面处理** | 铝合金阳极氧化（膜厚 ≥ 10μm）或粉末喷涂（≥ 60μm） |
| **组装精度** | 叶片与外框间隙 1～2mm，叶片之间间隙均匀 |
| **附件** | 风口带调节阀时，阀片转动灵活、关闭严密 |

### 1.3 常见问题

> [!warning] 制作通病
> - ❌ 百叶叶片变形导致调节卡滞
> - ❌ 外框焊接变形 → 安装时无法平整贴合
> - ❌ 表面划伤 → 阳极氧化层破损 → 现场腐蚀
> - ❌ 散流器扩散圈不同心 → 送风偏流

## 二、风阀制作

### 2.1 风阀分类与功能

| 风阀类型 | 功能 | 关键要求 |
|----------|------|----------|
| **手动调节阀** | 风量调节（初调节后锁定） | 阀片转动灵活、刻度指示清晰 |
| **电动调节阀** | 自控系统联动调节风量 | 执行器匹配、阀片关闭严密 |
| **防火阀 (70°C)** | 火灾时 70°C 熔断关闭 | 易熔片动作可靠、信号输出正常 |
| **排烟防火阀 (280°C)** | 排烟时 280°C 熔断关闭 | 耐高温、远程复位 |
| **止回阀** | 防气流倒灌 | 阀片启闭灵活、反向严密 |
| **定风量阀 (CAV)** | 保持恒定风量 | 机械自力式或电控式 |
| **变风量阀 (VAV)** | 按需求调节风量 | 带执行器 + 风速传感器 |

### 2.2 风阀制作工艺

| 工序 | 技术要求 |
|------|----------|
| **阀体框架** | 镀锌钢板折弯组焊，对角线偏差 ≤ 2mm |
| **阀片（阀板）** | 钢板冲压或型材拼接，平面度 ≤ 1mm/500mm |
| **轴承/轴套** | 含油铜基轴承或滚动轴承，转动灵活无卡阻 |
| **密封** | 阀片与阀体间设弹性密封条，关闭时漏风率 ≤ 2% |
| **执行器安装** | 执行器输出轴与阀轴同心，联轴器松紧适度 |
| **防火阀熔断器** | 易熔片熔点 70°C±2°C / 280°C±5°C，动作可靠 |

### 2.3 风阀质量检查

| 检查项目 | 合格标准 |
|----------|----------|
| **阀片全开/全关** | 灵活无卡阻，限位准确 |
| **密封性** | 全关状态透光检查无可见光穿透 |
| **执行器联动** | 0～100% 开度与信号对应，无空程/滞后 |
| **熔断器** | 温控水浴法验证熔断温度 ±2°C |
| **防腐** | 镀锌层完好，划伤处涂富锌漆 |

## 三、消声器制作

### 3.1 消声器类型

| 类型 | 消声原理 | 适用频率 | 常见形式 |
|------|----------|:--------:|----------|
| **阻性消声器** | 吸声材料吸收声能 | 中高频 | 片式消声器、管式消声器、折板式消声器 |
| **抗性消声器** | 声波反射/干涉 | 低频 | 膨胀腔消声器、共振腔消声器 |
| **阻抗复合消声器** | 阻性+抗性复合 | 全频段 | 阻抗复合式 |
| **微穿孔板消声器** | 微穿孔共振吸声 | 宽频 | 微穿孔板消声器（洁净环境适用） |

### 3.2 阻性消声器制作工艺（最常用）

| 工序 | 技术要求 |
|------|----------|
| **外壳** | 镀锌钢板 δ ≥ 1.0mm，咬口或焊接，严密不漏风 |
| **吸声材料** | 离心玻璃棉板（密度 ≥ 32kg/m³），或矿渣棉 |
| **护面层** | 玻璃纤维布（防止纤维飞散）+ 穿孔板（穿孔率 ≥ 20%，孔径 φ4～6mm） |
| **填充** | 吸声材料填充密实、均匀，不得有空洞或沉降 |
| **间距** | 消声片间距 ≤ 200mm（中高频消声），片厚 ≥ 100mm |

### 3.3 消声器质量控制要点

| 项目 | 合格标准 |
|------|----------|
| **外壳严密性** | 漏光法检验无透光，漏风量符合要求 |
| **吸声材料保护** | 护面玻纤布无破损，穿孔板无堵塞 |
| **消声片平直度** | 偏差 ≤ 2mm/m |
| **法兰连接** | 同金属风管法兰标准，垫片满垫无漏风 |
| **洁净空调专用** | 微穿孔板消声器，严禁使用玻璃纤维吸声材料 |

> [!warning] 消声器制作通病
> - ❌ 玻璃棉填充不密实 → 使用中沉降堵孔 → 消声效果下降 + 阻力增大
> - ❌ 护面玻纤布破损 → 玻璃纤维飞散 → **洁净室灾难**
> - ❌ 穿孔板开孔率不足 → 消声效果差
> - ❌ 外壳漏风 → 消声器旁通 → 消声失效

## 四、静压箱制作

### 4.1 功能与原理

静压箱是设置在风机出口或风管干管上的**气流稳压装置**：

| 功能 | 说明 |
|------|------|
| **气流均压** | 将风机出口动压转化为静压，减少气流脉动 |
| **降噪** | 降低气流再生噪声 5～10 dB(A) |
| **风量分配** | 为多支管提供均匀的静压条件 |

### 4.2 制作工艺要求

| 项目 | 技术要求 |
|------|----------|
| **材质** | 镀锌钢板，壁厚 ≥ 风管壁厚 + 0.2mm |
| **外形尺寸** | 长边 ≤ 2000mm 时偏差 ±3mm；>2000mm 时偏差 ±5mm |
| **内部** | 可内衬吸声材料（同消声器要求），穿孔板内衬 |
| **加固** | 大尺寸静压箱须设内部拉杆或外部角钢加固 |
| **法兰** | 同风管法兰标准，法兰面平整 |
| **检修门** | 大尺寸静压箱宜设检修门（≥ 400×400mm） |

## 五、柔性短管制作

### 5.1 用途

柔性短管（帆布接头）用于：

- **风机进出口**：隔离风机振动，防止传递给风管系统
- **穿越伸缩缝/沉降缝**：适应建筑结构变形
- **VAV 箱连接**：吸收安装偏差和热胀冷缩

### 5.2 制作要求

| 项目 | 技术要求 |
|------|----------|
| **长度** | 🔑 **150～300mm**（过短减振无效，过长增加阻力+下垂） |
| **材质** | 一般空调：帆布/人造革；防排烟：**不燃材料**（硅胶玻纤布/不锈钢软管） |
| **缝合** | 搭接宽度 ≥ 20mm，双道缝线，缝线间距 ≤ 30mm |
| **安装** | 两端法兰连接，不得作为变径管（不得扭曲、不得变径替代） |
| **防结露** | 冷风系统柔性短管外应做**保温**，且保温材料不得妨碍柔性变形 |
| **严密性** | 缝合处不得漏风，防排烟系统短管须做漏风量测试 |

> [!warning] 柔性短管制作红线
> - 🔴 **防排烟系统柔性短管必须使用不燃材料！**（GB 50243 强条 5.2.7）
> - 🔴 不得用柔性短管代替弯头/变径管
> - 🔴 不得将柔性短管扭曲或拉伸至极限长度安装
> - 🔴 长度不得 < 150mm，否则减振效果无效
