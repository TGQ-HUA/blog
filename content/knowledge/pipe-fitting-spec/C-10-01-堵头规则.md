# C.10.01 堵头规则

圆形风管堵头的制造规范（两件式堵头、配对法兰堵板）。

---

## C.10.01.1 圆堵头

### C.10.01.1.1 堵头（两件式）

| 部件 | 尺寸 |
|------|------|
| **内接** | 口径 **-3mm**，长度 **100mm** |
| **堵板** | 口径 > 200 → 板大 **12mm**；口径 ≤ 200 → 板大 **10mm** |
![](/img/knowledge/pipe-fitting-spec/圆堵头1.webp)
### C.10.01.1.2 圆堵板（配对法兰）

| 形式 | 说明 |
|------|------|
| 单个料件 | 圆平板开对应法兰孔 |
![](/img/knowledge/pipe-fitting-spec/圆堵头2.webp)

### C.10.01.1.3 圆堵板_平板式导入排版规则

导入排版时，输入直径自动补偿两项值：

| 条件 | 补偿公式 | 说明 |
|------|----------|------|
| 直径 > 200 | +2 + 料厚 | 大口径堵板需加大 2mm 配合封堵管口 |
| 直径 ≤ 200 | +0 + 料厚 | 小口径不加大 |

> **为何加料厚**：CAMduct 排版软件会自动减去一个料厚，需要把这个值加回来。

**自动化公式（WPS JS / IMS 导入）**：

```javascript
if (规格1a.Value2 > 200) {
    var 圆口数据 = 2 + 厚度.Value2;
} else {
    var 圆口数据 = 0 + 厚度.Value2;
}
```

> ⚠️ 需根据对应封堵管的实际直径做调整——如果堵板直径与管口直径不一致，以管口为准。
