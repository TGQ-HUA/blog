# M.02 风阀与防火阀

> 暖通风管系统中用于控制气流、防火隔断、排烟联动的全部风阀类型速查。

---

## M.02.1 风量调节阀

| 项目     | 内容                       |
| ------ | ------------------------ |
| **功用** | 调节支管风量、平衡系统压力            |
| **样式** | 矩形/圆形，多叶对开式，手柄（手动）或电动执行器 |
| **材质** | 钢制、铝合金                   |
| **特征** | 带风量刻度，可调叶片开度             |

## M.02.2 防火阀（70℃）

| 项目 | 内容 |
|------|------|
| **功用** | 防火分区隔断，70℃ 易熔片熔断自动关闭，防串火串烟 |
| **安装位置** | 穿防火墙、穿机房墙体、穿重要房间隔墙的风管上 |
| **样式** | 矩形为主，阀体 + 叶片 + 易熔合金片 + 复位手柄，可带电信号反馈 |
| **材质** | 钢制（防火阻燃） |
!af28cda15bb95790fec906b8f451c95f~tplv-be4g95zd3a-image.jpeg
---

## M.02.3 排烟防火阀（280℃）

| 项目 | 内容 |
|------|------|
| **功用** | 排烟系统专用，平时关闭、火灾时联动开启排烟，280℃ 自动关闭 |
| **样式** | 矩形/圆形，配排烟执行器，可远程/联动控制 |
| **材质** | 钢制（耐高温） |

## M.02.4 防烟阀 / 加压送风口

| **样式** | 矩形，多叶结构，电动/气动执行器，带状态反馈 |
| **材质** | 钢制、铝合金 |

## M.02.5 止回阀（单向阀）

| 项目 | 内容 |
|------|------|

| **材质** | 钢制、铝合金 |

## M.02.6 蝶阀（圆形风管专用）

| 项目 | 内容 |
|------|------|
| **功用** | 圆管气流通断控制与简易风量调节 |

## M.02.7 阀门法兰打孔规范

> 阀门对接角铁法兰时，螺丝孔的靠内尺寸需根据法兰角铁型号调整。

| 规则 | 内容 |
|------|------|
| **内空同步** | 角铁法兰内空 +5mm 时，阀门内空也需 +5mm，保证螺孔对齐 |
| **30# 角铁** | 法兰边打孔靠内 **17mm** |
| **40# 角铁** | 法兰边打孔靠内 **22mm** |
| **50# 角铁** | 法兰边打孔靠内 **27mm** |
