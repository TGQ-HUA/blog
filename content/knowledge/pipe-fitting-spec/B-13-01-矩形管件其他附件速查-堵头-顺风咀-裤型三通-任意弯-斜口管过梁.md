# B.13.01 矩形管件其他附件速查

> 矩形风管堵头/顺风咀/裤型三通/任意弯/斜口管/过梁弯的制造规则速查。

---

## B.13.01.1 堵头

### B.13.01.1.1 堵板 — 翻边式
[图片缺失: 风管资料统合-待更新输单方式_11.27_4.png]

| 项目     | 规格                                                 |
| ------ | -------------------------------------------------- |
| **尺寸** | 为封口（封尾）的口径 + 四边做小边（翻 7mm 或 10mm），对应的封口（封尾）要做大边（母口） |
| **配用** | 【直管-堵】与【短直管-堵】（1 块堵板）以及【箱子】（2 块堵板）                 |
| **注意** | 【直管-堵】的堵板需要单独列项，其他不需要列单独项目                         |

### B.13.01.1.2 堵板 — 加宽式
[图片缺失: 风管资料统合-待更新输单方式_11.27_5.png]

| 项目 | 规格 |
|------|------|
| **尺寸** | 为封口（封尾）的口径尺寸放大 60mm |
| **配用** | 一般对应做共板法兰口与堵板相铆接 |
| **注意** | 需要列单独项目，项目名称记做【平板】 |

### B.13.01.1.3 堵板 — 平板式
[图片缺失: 风管资料统合-待更新输单方式_11.27_6.png]

| 项目 | 规格 |
|------|------|
| **尺寸** | 为封口（封尾）的口径尺寸，正尺寸平板下料 |
| **配用** | 一般作为焊接管封堵 |
| **注意** | 【直管-堵】的堵板需要单独列项，其他不需要列单独项目 |

---

## B.13.01.2 顺风咀
[图片缺失: 风管资料统合-待更新输单方式_11.27_9.png]

| 项目 | 规格 |
|------|------|
| **尺寸** | 大口宽比小口宽大 100 |
| **配用** | 大头常规外翻 30mm，小头对应接口 |
| **注意** | 常规默认边平，高相等，其余同变径类似 |

---

## B.13.01.3 裤型三通
[图片缺失: 风管资料统合-待更新输单方式_11.27_13.png]

| 项目 | 规格 |
|------|------|
| **尺寸** | 主口 / 支咀 1 / 支咀 2 长度 `（A1B1）/（A2B2）/（A3B3）× L1` |
| **配用** | 同一方向，1 分为 2；不同于侧三通与正三通 |
| **注意** | 需要参图，确认支咀间隔和偏斜 |

---

## B.13.01.4 任意弯
[图片缺失: 风管资料统合-待更新输单方式_11.27_17.png]

| 项目 | 规格 |
|------|------|
| **尺寸** | 口径内长 / 外长 `（A1B1）× L1 / L2` |
| **配用** | 需参图，标注交叉线尺寸或者一个角的角度 |
| **注意** | 默认做成弧形 |

> CAMduct 拼板技巧见 I.02 CAMduct基本操作手册#I.02.3.5 特殊情况。

---

## B.13.01.5 斜口管
[图片缺失: 风管资料统合-待更新输单方式_11.27_18.png]

| 项目 | 规格 |
|------|------|
| **尺寸** | ①斜口直管 — 宽 × 高 × 底长；②斜口弯头 — 宽 A × 高 B × 内径 R − 顶加长 |
| **配用** | 一般用做于防雨/防鸟/防虫管件 |
| **注意** | 斜口无法做直位，只能做【直边】或【翻边】 |

---

## B.13.01.6 过梁弯
[图片缺失: 风管资料统合-待更新输单方式_11.27_19.png]

| 项目 | 规格 |
|------|------|
| **尺寸** | 口径 × 长 × 梁高 × 梁宽 |
| **配用** | 需参图，标注梁高和梁宽 |
| **注意** | 过梁弯默认做成菱形，当左右接口高差不一致时，只能做成弧形 |
