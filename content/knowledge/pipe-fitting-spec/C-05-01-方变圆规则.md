## C.05.01.1 方变圆排版（6 种类型）

### C.05.01.1.1 通用规则

| 部件 | 规则 |
|------|------|
| **共板框** | 成型长度 **60mm**，与主体碰焊或铆接 |
| **主体** | 方口长边 ≥ 250 → 咬口（拼接15）；< 250 → 碰焊5 |
| **圆咀** | 长度默认 **80mm**，接缝同圆管 |
| **角铁直伸量** | 30#角铁→40mm，40#角铁→50mm，50#角铁→60mm，需做好翻边；方口尺寸+2 |

### C.05.01.1.2 六种类型速查表

| 类型 | 部件组成 | 主体长度调整 | 圆口尺寸 | 方口/圆咀 |
|------|----------|-------------|----------|----------|
| ① 共板+咀 | 共板框 + 主体 + 圆咀 | **-80mm** | 圆咀原尺寸 | 主体圆口 +2以上 |
| ② 共板+法兰 | 共板框 + 主体 + 法兰 | **-10mm** | 主体圆口 +12以上 | — |
| ③ 共板+活法兰 | 共板框 + 主体 + 圆咀配法兰 | **-70mm** | 圆咀 +2 | 主体圆口 +4以上 |
| ④ 角铁+咀 | 主体 + 圆咀 | 主体+咀：**-60mm** | 圆咀原尺寸 | 方口+2，主体圆口+2以上 |
| ⑤ 角铁+法兰 | 主体 + 法兰 | 主体+法兰：**+10mm** | 主体圆口 +14以上 | 方口+2 |
| ⑥ 角铁+活法兰 | 主体 + 圆咀配法兰 | 主体+咀：**-50mm** | 圆咀+2 | 方口+2，主体圆口+4以上 |

## C.05.01.2 满焊成品方口角铁边规则

> ⚠️ 此规则**仅限**方变圆做满焊成品、且方口做铆接法兰或点焊角铁时适用。

| 项目 | 规则 |
|------|------|
| **法兰接口选择** | 主体方口选 **V角铁**（提醒生产需折角铁边） |
| **方口直位** | = 法兰型号长度（**不加 10mm**，做一样长） |

### 与咬口成品的对比

| 成型方式 | 方口是否做角铁边 | 方口直位 |
|----------|----------------|----------|
| **满焊成品** + 铆接法兰/点焊角铁 | ✅ 做角铁边 | 直位 = 法兰型号（不+10） |
| **正常咬口成品** | ❌ 不做角铁边 | 直位 = 法兰型号 **+10mm** |
