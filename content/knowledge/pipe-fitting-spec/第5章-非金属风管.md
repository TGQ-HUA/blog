# 第5章 非金属风管

第 5 章规定了非金属通风管道的材料要求、制作工艺和质量控制，主要涵盖三类非金属风管：无机玻璃钢(FRP)风管、硬聚氯乙烯(PVC-U)风管和复合风管（酚醛/聚氨酯等）。非金属风管广泛应用于**防排烟系统**（耐火风管）和**腐蚀性环境**排风系统。

---

## 5.1 无机玻璃钢(FRP)风管

**无机玻璃钢风管**（也称 FRP 或玻镁风管）以中碱玻璃纤维布为增强材料、氯氧镁水泥为基材制成，是防排烟系统的核心材料之一。

### 5.1.1 FRP 风管材料要求

| 材料 | 要求 |
|------|------|
| 玻璃纤维布 | 中碱或无碱，厚度 0.2~0.4mm，经纬密度 6×6 或 8×8 |
| 氯氧镁水泥 | MgO 含量 ≥ 80%，活性 ≥ 60% |
| 添加剂 | 抗水剂、缓凝剂、减水剂，按配比添加 |

### 5.1.2 FRP 风管壁厚与层数

| 风管长边或直径 (mm) | 最小壁厚 (mm) | 最少玻纤布层数 |
|:-----------------:|:----------:|:----------:|
| ≤ 300 | 2.5 | 4 |
| 300~500 | 3.0 | 5 |
| 500~1000 | 4.0 | 6 |
| 1000~1500 | 5.0 | 8 |
| 1500~2000 | 6.0 | 10 |
| > 2000 | 按设计 | 按设计 |

### 5.1.3 FRP 风管制作工艺

| 工艺环节 | 要求 |
|----------|------|
| **模具** | 表面光滑、脱模剂均匀涂覆 |
| **铺层** | 玻纤布搭接宽度 ≥ 50mm，每层错缝铺设 |
| **固化** | 常温固化 ≥ 7 天，或加温固化 ≥ 24h（60°C） |
| **接口法兰** | 可与管体一次成型（整体法兰），也可后装角钢法兰 |
| **表面处理** | 内表面光滑平整，外表面涂防水涂层 |

> [!warning] FRP 风管注意事项
> - FRP 风管**抗冲击性较差**，运输和安装须防止碰撞
> - 氯氧镁制品存在**返卤吸潮**问题，须严格控制 MgO/MgCl₂ 摩尔比（建议 ≥ 5:1）
> - 防排烟用 FRP 风管必须提供**耐火极限检测报告**（通常 ≥ 1.0h）

## 5.2 硬聚氯乙烯(PVC-U)风管

**硬聚氯乙烯风管**主要用于**腐蚀性气体**排风系统（如实验室排风、电镀车间排风）。

### 5.2.1 PVC-U 板材厚度

| 风管长边或直径 (mm) | 圆形风管壁厚 (mm) | 矩形风管壁厚 (mm) |
|:-----------------:|:--------------:|:--------------:|
| ≤ 320 | 3.0 | 4.0 |
| 320~450 | 4.0 | 4.0 |
| 450~630 | 4.0 | 5.0 |
| 630~1000 | 5.0 | 6.0 |
| 1000~1250 | 6.0 | 8.0 |
| 1250~2000 | 8.0 | 10.0 |
| > 2000 | 按设计 | 按设计 |

### 5.2.2 PVC-U 风管连接方式

| 连接方式 | 适用条件 | 工艺要点 |
|----------|----------|----------|
| **承插粘接** | 直径 ≤ 200mm 圆形风管 | 承口深度 ≥ 直径，PVC 专用胶均匀涂刷 |
| **法兰连接** | 矩形风管及大口径风管 | PVC 法兰板焊接在风管端部，螺栓连接 |
| **焊接** | 所有 PVC 风管拼接 | 热风焊，焊条材质与母材一致，焊接温度 220~250°C |

### 5.2.3 PVC-U 使用温度限制

| 条件 | 限值 |
|------|:--:|
| 最高连续使用温度 | +60°C |
| 最低使用温度 | -10°C |
| 热膨胀系数 | ~7×10⁻⁵ /°C |

> 当温度变化超过 30°C 时，须设置**伸缩节**补偿热胀冷缩。

## 5.3 复合风管

复合风管由**内外金属面板（铝箔）** + **保温芯材**复合而成，兼具气体输送和保温功能。

### 5.3.1 常见复合风管类型

| 类型 | 芯材 | 总厚度 (mm) | 适用场景 |
|------|------|:--------:|----------|
| **酚醛复合风管** | 酚醛泡沫 (PF) | 20~30 | 舒适性空调送风、回风 |
| **聚氨酯复合风管** | 聚氨酯泡沫 (PU) | 20~30 | 低温送风、洁净空调 |
| **玻璃棉复合风管** | 离心玻璃棉板 | 25~50 | 消声要求高的空调系统 |
| **挤塑聚苯乙烯风管** | XPS 板 | 25~30 | 一般空调通风 |

### 5.3.2 复合风管制作要求

| 项目 | 要求 |
|------|------|
| **板材切割** | V 形槽切割，保留一面铝箔完整作为弯曲铰链 |
| **折弯成型** | 沿 V 形槽折弯 90°，形成矩形风管四壁 |
| **接缝密封** | 铝箔胶带覆盖所有接缝，胶带宽度 ≥ 60mm |
| **法兰连接** | 采用**专用插接式铝合金法兰**或角钢法兰，法兰与风管间涂密封胶 |
| **加固** | 边长 > 1000mm 时须设内部支撑或外部加固框 |

### 5.3.3 复合风管使用限制

| 限制条件 | 说明 |
|----------|------|
| 最高工作压力 | ≤ 800Pa（酚醛）/ ≤ 1000Pa（聚氨酯） |
| 最高风速 | ≤ 15 m/s |
| 禁止使用场景 | 防排烟系统（酚醛/PU 可燃）、高温排风（> 80°C） |

---

## 5.4 非金属风管质量要求总结

| 检测项目 | FRP 风管 | PVC-U 风管 | 复合风管 |
|----------|:------:|:--------:|:------:|
| 外观 | 表面光滑、无裂纹、无分层 | 表面光滑、无气泡、无色差 | 铝箔平整、无起泡脱层 |
| 壁厚偏差 | ±10% | +15%, -0% | ±2mm |
| 法兰平面度 | ≤ 2mm | ≤ 2mm | ≤ 2mm |
| 对角线偏差 | ≤ 3mm(≤1000)/≤5mm(>1000) | 同左 | 同左 |
| 耐火性能 | 满足设计要求（通常 ≥ 1.0h） | 可燃（B2 级）| PF/PU 可燃（B2 级），玻纤 A 级 |
