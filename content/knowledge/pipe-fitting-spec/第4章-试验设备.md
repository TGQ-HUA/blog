# 第4章 试验设备

> [!important] 章节定位
> 第4章规定了通风管道耐火试验所需的所有试验设备及其技术要求。标准化的试验设备是保证试验结果**可重复性和可比性**的前提。核心设备包括耐火试验炉、热电偶系统、压力传感器、位移传感器等。

---

## 一、耐火试验炉

### 1.1 炉体结构要求

| 项目 | 技术要求 |
|------|----------|
| **炉体类型** | 水平炉（卧式炉），可容纳水平安装的管道试件 |
| **炉膛尺寸** | 应能容纳长度 ≥ **3m** 的管道试件，横截面 ≥ 试件截面的 1.5 倍 |
| **炉壁结构** | 耐火砖/耐火浇注料内衬 + 保温层，外壁温度 ≤ 环境温度+40°C |
| **燃烧器** | 布置在炉膛两侧或底部，数量保证炉温均匀性 |
| **燃料** | 天然气/液化石油气/轻柴油，配自动比例调节阀 |
| **观察窗** | 炉壁上设耐高温观察窗，用于观察试件变形/火焰穿透 |

### 1.2 炉温均匀性要求

| 指标 | 要求 |
|------|:----:|
| **炉内各点温度偏差** | 在稳定阶段，各热电偶读数与标准升温曲线偏差 ≤ ±100°C |
| **升温阶段偏差** | 试验开始 10min 内，各点温度与标准曲线偏差 ≤ ±15%（5~10min阶段放宽至 ±15%） |

> [!tip] 炉温均匀性为什么重要？
> 耐火试验炉是一个大型燃烧空间，如果炉温不均匀（例如靠近燃烧器处温度过高而远端过低），会导致试件不同部位受火强度不同，试验数据失去公正性。因此炉温均匀性是试验设备验收的重要指标。

## 二、热电偶布置

### 2.1 热电偶类型与要求

| 参数 | 要求 |
|------|------|
| **类型** | K 型（镍铬-镍硅）铠装热电偶 |
| **直径** | 测温端直径 ≤ **1.5mm**（保证快速响应） |
| **精度** | ±1°C 或 ±0.4% 读数（取大值） |
| **量程** | ≥ 1200°C |
| **数据采集频率** | ≥ 1次/分钟（推荐 ≥ 1次/30秒） |

### 2.2 炉内热电偶布置

炉内温度测量是判定耐火试验是否按标准升温曲线进行的依据，需布置 **4 个** 炉内热电偶：

| 热电偶编号 | 位置 | 说明 |
|:----------:|------|------|
| **TF1** | 管道试件上方 100mm 处，距管端 500mm | 监测试件顶部炉温 |
| **TF2** | 管道试件下方 100mm 处，距管端 500mm | 监测试件底部炉温 |
| **TF3** | 管道试件一侧 100mm 处，距管端 500mm | 监测试件侧面炉温 |
| **TF4** | 管道试件另一侧 100mm 处，距管端 500mm | 监测试件另一侧面炉温 |

```
          炉顶
    ┌───────────────────┐
    │    TF1 (上100mm)   │
    │  ┌──────────────┐  │
    │  │   管 道 试 件  │  │
    │  └──────────────┘  │
    │ TF3           TF4  │  ← 两侧各100mm
    │    TF2 (下100mm)   │
    └───────────────────┘
          炉底
```

> [!warning] 炉内热电偶安装要点
> - 热电偶测温端不得与管道试件或炉壁接触（间距 ≥ 100mm）
> - 在炉顶/炉底开孔插入，密封孔口防止漏火
> - 4 个热电偶读数取**算术平均值**作为炉温控制值

### 2.3 管道表面热电偶布置

用于判定**隔热性**——测量管道外表面（非受火面/A类试验）或管道内表面（B类试验）的温度。同样布置 **4 个**：

| 热电偶编号 | 位置 |
|:----------:|------|
| **TS1** | 管道顶面中心 |
| **TS2** | 管道底面中心 |
| **TS3** | 管道一个侧面中心 |
| **TS4** | 管道另一个侧面中心 |

> [!tip] 热电偶固定方法
> 表面热电偶须用 **耐高温陶瓷纤维毯 + 金属压板** 覆盖固定，覆盖面积约 100mm×100mm，确保测温端与管道表面紧密接触但不被火焰直接辐射。

## 三、压力传感器

### 3.1 炉内压力控制要求

| 参数 | 要求 |
|------|:----:|
| **控制压力** | **20 ± 3 Pa**（炉内相对于炉外大气压的正压） |
| **测量位置** | 管道试件中心高度平面 |
| **传感器类型** | 微差压变送器/压力传感器 |
| **精度** | ±2 Pa |
| **采样频率** | ≥ 1 次/分钟 |

### 3.2 压力控制原理

```
    炉内正压 20±3Pa
         ↓
  ┌──────────────────┐
  │   试验炉（正压）   │  → 正压确保火焰和热气流不从缝隙逸出
  │   管道试件        │  → 模拟火灾时烟气正压条件
  └──────────────────┘
         ↑
    排烟阀（调节开度）
```

> [!warning] 炉压控制的重要性
> 适度的正压（20Pa）模拟火灾时燃烧产物的烟气压力。压力过高→火焰和热气从炉体缝隙喷出→不安全；压力过低（负压）→外部冷空气吸入炉内→降低试件实际受火强度→试验结果偏"安全"（虚假合格），失去工程参考价值。

### 3.3 炉压测量点位置

| 测点 | 位置 |
|------|------|
| 压力取压口 | 管道试件中心高度对应的炉壁位置，距管道端部 1m 处 |
| 参考端 | 引至炉外大气环境，避免风力干扰 |

## 四、位移传感器

### 4.1 用途

位移传感器用于监测试验过程中管道试件的**变形量**，以判定管道是否因热变形而丧失完整性或丧失承载能力。

### 4.2 安装要求

| 项目 | 要求 |
|------|------|
| **传感器类型** | LVDT 位移计/拉绳式位移传感器 |
| **量程** | ≥ 100mm |
| **精度** | ≤ ±1mm |
| **安装位置** | 管道跨中位置的顶面和侧面（B类试验）或底面和侧面（A类试验） |
| **数量** | 至少 2 个（垂直方向 + 水平方向各1） |
| **数据记录** | 与温度数据同步记录，频率 ≥ 1次/分钟 |

## 五、其他辅助设备

| 设备 | 用途 | 技术要求 |
|------|------|----------|
| **引风机/送风机** | A类试验：向管道内通入冷空气（风速 3±0.5m/s） | 变频控制，风速可调 |
| **风速计** | 测量管内空气流速 | 精度 ±0.2m/s |
| **数据采集系统** | 同步采集炉温、管表温度、炉压、位移 | ≥ 12 通道，采样率 ≥ 2Hz |
| **计时器** | 记录耐火试验持续时间 | 精度 ±1s |
| **摄像系统** | 记录试验全程，观察火焰穿透 | 耐高温摄像头 |
