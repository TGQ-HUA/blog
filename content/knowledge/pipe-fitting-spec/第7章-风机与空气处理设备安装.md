# 第7章 风机与空气处理设备安装

第 7 章规定了通风机、空调机组、风机盘管、VAV 末端、电加热器、静电净化装置等设备的安装质量验收要求，包含**三条强制性条文**（7.2.2、7.2.10、7.2.11）。

---

## 7.1 设备安装通用要求

| 通用要求 | 具体内容 |
|----------|----------|
| **基础验收** | 设备基础混凝土强度达标，标高、水平度、预埋件位置偏差在允许范围内 |
| **减振措施** | 风机、空调机组、水泵等旋转设备须设减振器（弹簧减振器/橡胶减振垫） |
| **水平度/垂直度** | 设备安装水平度偏差 ≤ 1‰，垂直度偏差 ≤ 1‰ |
| **进出口连接** | 风机进出口与风管连接须设**柔性短管**，长度 150~300mm，防止振动传递 |
| **电气安全** | 设备金属外壳须可靠接地，电机绝缘电阻 ≥ 0.5 MΩ |

---

## 7.2 通风机安装

### 7.2.1 安装要点

| 风机类型 | 特殊安装要求 |
|----------|-------------|
| **离心风机** | 叶轮与机壳间隙均匀，无碰擦；皮带传动须设防护罩 |
| **轴流风机** | 叶尖与筒壁间隙均匀（≤ 叶轮直径的 1.5‰），叶片角度按标识安装 |
| **屋顶风机** | 底座与基础连接牢固，防雨罩完好 |
| **防爆风机** | 叶轮材质符合防爆要求（铝质/铜质），与机壳无火花摩擦 |

### 🔴 7.2.2 — 风机防护罩（强制性条文）

> [!danger] 强制性条文 7.2.2
> **通风机传动装置的外露部位以及直通大气的进、出口，必须装设防护罩（网）或采取其他安全防护措施。**
>
> - 皮带传动、联轴器等旋转部位须设**全封闭防护罩**
> - 直通大气的进/出口须设**防护网**（网孔 ≤ 25mm）

---

## 7.3 空调机组安装

| 机组类型 | 安装要点 |
|----------|----------|
| **组合式空调机组** | 各功能段按设计顺序组装，段间密封垫密实；表冷器集水管排水通畅 |
| **整体式空调机组** | 水平度 ≤ 3mm/m，机外余压满足设计 |
| **屋顶式空调机组** | 屋面基础防水完好，机组与风管柔性连接 |
| **恒温恒湿机组** | 加湿段独立安装，避免滴水影响电气元件 |

---

## 7.4 风机盘管安装

| 检验项目 | 要求 |
|----------|------|
| 水平度 | ≤ 3mm/m（卧式） |
| 垂直度 | ≤ 3mm/m（立式） |
| 凝水管坡度 | ≥ 1%（向排水方向），确保冷凝水顺畅排出 |
| 风管连接 | 风机盘管出口与风管之间设柔性短管 |
| 电气接线 | 温控器接线正确，三速开关功能正常 |

---

## 7.5 主控项目（强制性条文）

### 🔴 7.2.10 — 静电净化装置接地

> [!danger] 强制性条文 7.2.10
> **静电式空气净化装置的金属外壳必须可靠接地，接地电阻应符合设计要求。**
>
> - 静电净化器工作时产生高压静电场，外壳接地是**防触电**的关键措施
> - 接地电阻一般要求 ≤ 1Ω

### 🔴 7.2.11 — 电加热器安装

> [!danger] 强制性条文 7.2.11
> **电加热器的安装必须符合下列规定：**
> 1. 电加热器与构架间的绝热层必须使用**不燃材料**
> 2. 接线柱外露部分应设**安全防护罩**
> 3. 电加热器的**金属外壳必须接地**
> 4. 电加热器必须与**送风机联锁**（风机停则电加热器自动断电）
> 5. 连接风管的法兰垫片须采用**不燃材料**
>
> — 此条是防止电加热器引发火灾的核心安全要求。

## 7.6 一般项目

| 条目 | 内容 | 允许偏差/要求 |
|------|------|--------------|
| 7.3.1 | 风机安装水平度 | ≤ 1mm/m |
| 7.3.2 | 空调机组水平度 | ≤ 3mm/m |
| 7.3.3 | 风机盘管凝水管坡度 | ≥ 1% |
| 7.3.4 | 设备保温 | 绝热层与设备表面贴合紧密 |
| 7.3.5 | 过滤器安装 | 方向正确、密封严密 |

---

## 7.7 与 CAMduct 的关联

| CAMduct 环节 | 关联内容 |
|-------------|----------|
| **风管末端标注** | 风机进出口尺寸须与 CAMduct 输出的连接法兰匹配 |
| **柔性短管标注** | CAMduct 图纸中标明柔性短管的位置和长度要求（150~300mm） |
| **设备进出口** | 在 风管建模与设计 中，设备连接件（Connector）尺寸须与设备实际法兰一致 |
| **减振段** | 风机进出口段的风管支撑须考虑减振器压缩量 |
