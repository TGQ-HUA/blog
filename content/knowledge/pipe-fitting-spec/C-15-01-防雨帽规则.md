# C.15.01 防雨帽规则

圆形风管防雨帽的尺寸公式与安全注意事项。
![](/img/knowledge/pipe-fitting-spec/圆风帽.webp)
---

## C.15.01.1 防雨帽

### C.15.01.1.1 尺寸公式

| 参数 | 公式 |
|------|------|
| **外锥高度** | `外直径 / 4` |
| **支撑杆高度** | `雨帽小圆半径 × 0.8`（结果 < 350mm 时取雨帽小圆半径） |
| **支撑杆间隔** | **不得超过 500mm** |

> 必须焊上**裙立边**（提高风帽质量等级）。

### C.15.01.1.2 烟囱安全注意事项

```
· 法兰材料必须加厚一个等级（常规法兰太薄）
· 圆焊接管（或螺旋管）材料也需加厚一个等级
· 跟单员报价时必须提醒客户
```
