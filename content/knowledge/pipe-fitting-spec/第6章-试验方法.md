# 第6章 试验方法

> [!important] 章节定位
> 第6章是 GB 15930-2007 的**试验验证章节**，规定了如何通过标准化试验来验证第5章各项技术指标是否达标。所有型式检验和 CCC 认证均依据本章规定的方法执行。本章的试验方法适用于防火阀、排烟防火阀、排烟阀三类产品的全部性能验证。

---

## 一、试验项目总览

| 序号 | 试验项目 | 对应技术要求 | 试验性质 | 重要性 |
|:----:|----------|:------------:|:--------:|:------:|
| 1 | 外观与标志检查 | 铭牌、气流箭头、操作标识 | 目测 + 测量 | 基础 |
| 2 | **耐火试验** | 耐火 ≥ 1.5h | 型式检验 | 🔥🔥🔥 |
| 3 | **温感器动作温度试验** | 70±2°C / 280±3°C | 型式检验 + 出厂检验 | 🔥🔥🔥 |
| 4 | **漏风量试验** | ≤500 / ≤200 m³/(h·m²) | 型式检验 | 🔥🔥 |
| 5 | 启闭可靠性试验 | 50 次循环 | 型式检验 | 🔥🔥 |
| 6 | **耐盐雾腐蚀试验** | ≥ 96 h | 型式检验 | 🔥🔥 |
| 7 | 绝缘电阻试验 | ≥ 20 MΩ | 出厂检验 | 🔥 |
| 8 | 控制功能试验 | 85%~110% 额定电压 | 型式检验 + 出厂检验 | 🔥🔥 |

## 二、耐火试验 ⭐

### 2.1 试验目的

验证阀门在规定耐火时间内（≥1.5h）能否保持耐火完整性和隔热性。

### 2.2 试验装置

| 设备 | 要求 |
|------|------|
| **耐火试验炉** | 可容纳全尺寸阀门试件，炉温按 ISO 834（GB/T 9978）标准升温曲线控制 |
| **热电偶** | 炉内测温热电偶 + 背火面测温热电偶 |
| **压力传感器** | 测量炉内与阀门前后的压差 |

### 2.3 试验条件

| 参数 | 要求 |
|------|------|
| **升温曲线** | ISO 834 标准时间-温度曲线（T = T₀ + 345 × log₁₀(8t+1)） |
| **耐火持续时间** | ≥ **1.5 h** |
| **阀门安装方向** | 与实际工程安装方向一致 |

### 2.4 耐火试验升温曲线

| 时间 (min) | 炉内温度 (°C) |
|:----------:|:-------------:|
| 5 | 576 |
| 10 | 678 |
| 15 | 739 |
| 30 | 842 |
| 60 | 945 |
| **90** | **1006** |
| 120 | 1049 |

### 2.5 试验程序

| 步骤 | 操作 | 要点 |
|:----:|------|------|
| **① 安装** | 将阀门按实际工况安装在试验炉上 | 阀体与炉壁间隙用不燃材料密封 |
| **② 状态设定** | 防火阀→常开；排烟防火阀→按设计状态；排烟阀→常闭 | 模拟火灾前真实状态 |
| **③ 点火升温** | 按 ISO 834 升温曲线加热 | 连续记录炉温 |
| **④ 观察动作** | 记录温感器动作时间、阀门关闭/开启时间 | 确认在规定温度范围内动作 |
| **⑤ 持续试验** | 继续加热至 1.5 h | 观察阀门状态变化 |
| **⑥ 判定** | 检查完整性（背火面是否出现火焰穿透）和隔热性（背火面温升） | 详见判定准则 |

### 2.6 判定准则

| 判定项 | 合格标准 | 不合格条件 |
|--------|----------|-----------|
| **完整性** | 试验全程背火面无火焰穿透 | 背火面出现持续 10s 以上的火焰 |
| **完整性** | 无影响阀门功能的变形 | 阀体严重变形导致无法有效隔断 |
| **隔热性** | 背火面平均温升 ≤ 140°C | 温升超限 |
| **隔热性** | 背火面最高温升 ≤ 180°C | 任一点温升超限 |
| **动作功能** | 温感器在规定温度范围内正常动作 | 未动作或动作温度超差 |

### 2.7 不同阀门类型的耐火试验差异

| 阀门类型 | 初始状态 | 预期动作 | 最终状态 | 重点考核 |
|----------|:--------:|----------|:--------:|----------|
| **防火阀** | 常开 | 70°C 时自动关闭 | 保持关闭至 1.5h | 关闭密封后的防火完整性 |
| **排烟防火阀（常开）** | 常开 | 280°C 时自动关闭 | 保持关闭至 1.5h | 高温延迟关闭后的完整性 |
| **排烟防火阀（常闭）** | 常闭 | 电动开启→280°C 关闭 | 保持关闭至 1.5h | 先开后关的双重可靠性 |
| **排烟阀** | 常闭 | 电动开启 | 保持开启至 1.5h | 开启状态下的耐火保持 |

## 三、温感器动作温度试验 ⭐

### 3.1 试验目的

验证温感器（易熔合金元件）是否在正确的温度范围内动作。

### 3.2 试验装置

| 设备 | 要求 |
|------|------|
| **恒温箱/温控油浴** | 控温精度 ± 0.5°C，升温速率可调 |
| **温度测量仪表** | 精度不低于 0.5 级 |
| **计时器** | 分辨率 ≤ 1s |

### 3.3 试验程序

| 步骤 | 操作 |
|:----:|------|
| **①** | 将温感器（从阀门上拆下或带阀门整体）放入恒温箱中 |
| **②** | 初始温度设为比公称温度低 10~15°C，稳定 10 min |
| **③** | 以 **1°C/min** 的速率匀速升温 |
| **④** | 连续观察温感器状态，记录动作瞬间的温度 |
| **⑤** | 重复试验 3 次（3 个试件），取平均值 |

### 3.4 判定

| 阀门类型 | 公称温度 | 允差 | 合格区间 |
|----------|:--------:|:----:|:--------:|
| **防火阀** | 70°C | ± 2°C | **68°C ~ 72°C** |
| **排烟防火阀** | 280°C | ± 3°C | **277°C ~ 283°C** |

## 四、漏风量试验

### 4.1 试验目的

在规定的静压差（300Pa）下测量阀门的漏风量。

### 4.2 试验装置

| 设备 | 要求 |
|------|------|
| **漏风量测试台** | 可产生并维持 300 Pa 静压差 |
| **流量计** | 量程覆盖预期漏风量范围，精度 ≥ 2.5 级 |
| **压力计** | 精度不低于 1.0 级 |

### 4.3 试验程序

| 步骤 | 操作 |
|:----:|------|
| **①** | 将阀门安装在测试台上，密封法兰连接处 |
| **②** | 设定阀门至规定状态（常开 / 关闭） |
| **③** | 施加 **300 Pa** 静压差（阀门两侧） |
| **④** | 待压力稳定后，测量漏风量 |
| **⑤** | 按阀门有效截面积换算，单位为 m³/(h·m²) |

### 4.4 判定

| 阀门类型 | 状态 | 限值 |
|----------|:----:|:----:|
| 防火阀 / 排烟防火阀 | 常开 | ≤ 500 m³/(h·m²) |
| 防火阀 / 排烟防火阀 / 排烟阀 | 关闭 | ≤ 200 m³/(h·m²) |

---

## 五、启闭可靠性试验（50 次循环）

### 5.1 试验目的

验证阀门在规定的启闭循环次数后仍能保持全部功能。

### 5.2 试验程序

| 步骤 | 操作 |
|:----:|------|
| **①** | 阀门安装就位，确认初始状态正常 |
| **②** | 进行一次完整的开启→关闭（或关闭→开启）循环 |
| **③** | 重复 **50 次**全行程启闭 |
| **④** | 循环完成后，立即检查并试验以下项目： |

### 5.3 循环后检查项目

| 检查项 | 要求 |
|--------|------|
| **外观** | 叶片、连杆、执行机构无松动、磨损、永久变形 |
| **手动操作** | 操作灵活，无卡阻，手感一致 |
| **电控操作** | 额定电压下动作正常，到位准确 |
| **温感器** | 动作温度仍在允差范围内（单独试验） |
| **漏风量** | 仍满足标准要求 |
| **绝缘电阻** | ≥ 20 MΩ |

---

## 六、耐盐雾腐蚀试验

### 6.1 试验目的

验证阀门金属部件在盐雾环境下的耐腐蚀能力。

### 6.2 试验条件（参照 GB/T 10125）

| 参数 | 要求 |
|------|------|
| **盐溶液** | 5% NaCl（质量分数），pH 6.5~7.2 |
| **试验温度** | 35°C ± 2°C |
| **试验时间** | ≥ **96 h** |
| **盐雾沉降率** | 1~2 mL/(80cm²·h) |

### 6.3 试验后检查

| 检查项 | 合格标准 |
|--------|----------|
| **外观** | 无严重锈蚀、涂层无起泡/脱落 |
| **手动操作** | 启闭灵活，无卡死 |
| **电控操作** | 功能正常 |
| **最终判定** | 阀门各项基本功能正常，无因腐蚀导致的失效 |

---

## 七、绝缘电阻与电气强度试验

### 7.1 绝缘电阻

| 项目 | 要求 |
|------|------|
| **测试仪表** | 500V 兆欧表 |
| **测试部位** | 带电部件与壳体之间 |
| **合格值** | ≥ **20 MΩ** |

### 7.2 控制功能电压范围试验

| 项目 | 要求 |
|------|------|
| **测试电压** | 85% 额定电压 和 110% 额定电压 |
| **合格标准** | 两种电压下阀门电控功能均应正常工作 |

## 八、试验方法速查表

| 试验项目 | 试验标准引用 | 关键参数 | 判定依据 |
|----------|:------------:|----------|----------|
| 耐火试验 | ISO 834 (GB/T 9978) | 1.5h，标准升温曲线 | 完整性 + 隔热性 |
| 温感器动作温度 | 本章 | 1°C/min 升温，3 次取平均 | 70±2°C / 280±3°C |
| 漏风量 | 本章 | 300Pa 压差 | ≤500 / ≤200 m³/(h·m²) |
| 启闭可靠性 | 本章 | 50 次全行程 | 全部功能正常 |
| 盐雾腐蚀 | GB/T 10125 | 96h，5% NaCl | 功能正常，无严重腐蚀 |
| 绝缘电阻 | 本章 | 500V 兆欧表 | ≥ 20 MΩ |
| 控制电压范围 | 本章 | 85%~110% 额定电压 | 动作正常 |
