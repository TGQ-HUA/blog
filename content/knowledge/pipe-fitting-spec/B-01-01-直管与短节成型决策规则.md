# B.01.01 直管与短节成型决策规则
矩形风管直管的成型方式决策树——按先后顺序判定，前面的优先。
![](/img/knowledge/pipe-fitting-spec/矩形直管成型方式.webp)
---

## B.01.01.1 标准节成型规则

判断流程（第 1 条 → 第 7 条，**前面的优先**）：

```
┌─────────────────────────────────────────────┐
│ 第1条 · 成品检查                            │
│ 已标记为"成" → 跳过，不给出成型方式           │
├─────────────────────────────────────────────┤
│ 第2条 · 口形+开平                           │
│ 条件：半周长 < 280mm 且 最大边 < 160mm       │
│ 适用：极小的管子                             │
├─────────────────────────────────────────────┤
│ 第3条 · 口字形                              │
│ 条件：半周长 < 500mm（但不满足第2条）         │
│ 适用：中小口径，四边围成口字形                │
├─────────────────────────────────────────────┤
│ 第4条 · 8片下料                             │
│ 条件：宽 > 4000mm 且 高 > 4000mm             │
│ ⚠️ 必须在第5条前判，否则会被错误归到6片       │
├─────────────────────────────────────────────┤
│ 第5条 · 6片下料                             │
│ 条件：只有一条边 > 4000mm                    │
├─────────────────────────────────────────────┤
│ 第6条 · 4片下料                             │
│ 条件：两边都不超 4000，但半周长 > 3800mm     │
├─────────────────────────────────────────────┤
│ 第7条 · L形（默认）                          │
│ 都不满足上面 → 最普通的常规成型               │
└─────────────────────────────────────────────┘
```

### B.01.01.1.1 决策表

| 优先级 | 成型方式 | 半周长条件 | 最大边条件 | 示例 |
|--------|----------|-----------|-----------|------|
| 1 | 成品跳过 | 已标记"成" | — | — |
| 2 | 口形+开平 | < 280mm | < 160mm | 120×100 |
| 3 | 口字形 | < 500mm | 不满足第2条 | 200×200 |
| 4 | 8片下料 | — | 宽 > 4000 且 高 > 4000 | 4500×4200 |
| 5 | 6片下料 | — | 只有一边 > 4000 | 4500×2000 |
| 6 | 4片下料 | > 3800mm | 两边都 ≤ 4000 | 2000×1900 |
| 7 | L形 | 默认 | 默认 | 常规尺寸 |

## B.01.01.2 短节成型规则

### B.01.01.2.1 短直管长度规则
```
· 长度与标准节相差 ≤ 10mm → 建议做成标准直管
· 长度 > 2000mm → 建议拆分：标准直管 + 短直管
· 客户无要求 → 按短直管就料长度
```

### B.01.01.2.2 短直管接缝规则

**成品**：

| 长度 | 接缝 |
|------|------|
| ≤ 100mm | **搭边（大边20）** |
| > 100mm | **大小边** |

**半成品**：

| 长度 | 接缝 |
|------|------|
| < 100mm | **搭边（大边20）** |
| ≥ 100mm | **大小边** |

> 成品 ≤ 100mm 仍用搭边——成品班组有碰焊机可自行完成，简单产品不必跨班组增加焊接班组工作量。

### B.01.01.2.3 成型决策（第 1 条 → 第 4 条）

| 优先级 | 成型方式 | 条件 | 速记口诀 |
|--------|----------|------|----------|
| 1 | 成品跳过 | 已标记"成" | — |
| 2 | 口字形 | **条件A**：半周长 ≤ 600 且 长度 < 1155<br>**或 B**：半周长 < 550（不管长度） | 管细→直接口字；管中等→够短才口字 |
| 3 | 4片式 | **条件C**：半周长 > 3500（不管长度）<br>**或 D**：半周长 > 1200 且 长度 > 1170 | 管粗→直接4片；管中等→够长才4片 |
| 4 | L形 | 默认 | — |

## B.01.01.3 标准节 vs 短节对比

| 项目 | 标准节 | 短节 |
|------|--------|------|
| 下料方式 | 5线开平 | 激光下料 |
| 口字形门槛(条件A) | 半周长 < 500 | 半周长 ≤ 600 且 长度 < 1155 |
| 口字形门槛(条件B) | — | 半周长 < 550（无条件） |
| 4片门槛 | 半周长 > 3800 | 半周长 > 3500 或 (>1200 且 >1170) |
