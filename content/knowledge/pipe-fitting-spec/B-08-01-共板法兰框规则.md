## B.08.01.1 常规共板框
[图片缺失: gf1.png]

| 项目 | 参数 |
|------|------|
| 排版基准 | 管口径（宽×高）+ 长度默认 **50mm** |
| 接口方式 | 一头共板、一头直边 |
| 接缝 | 大边 **20mm**（全部形式） |

| 形式 | 条件 |
|------|------|
| 口字形 | 口径周长 ≤ 3000mm |
| L 形 | 口径周长 > 3000mm |

## B.08.01.2 共板接口软接的法兰框

```
· 软接长度 > 250mm → 需要排共板框（浏阳厂需配方压条）
  （算量不排，下料再排版）
· 软接两头共板 → 1节软接配 2个共板框
· 软接一头共板一头直边 → 1节软接配 1个共板框
```

**排版方式**：
- 法兰框：管口径 + 长度 **42mm**
- 方压条：管口径 - 3 + 长度 **40mm**

**接口细节**：
- 共板框：一头共板，一头**离口 15mm 位置铆接**
- 方压条：一头外翻 10，一头离口 15mm 铆接
