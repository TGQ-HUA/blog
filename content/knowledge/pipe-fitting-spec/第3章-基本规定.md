# 第3章 基本规定

第 3 章是 JGJ/T 260-2011 的**基础规定章**，规定了所有检测工作必须遵循的通用要求：检测仪器的精度等级与校准管理、检测人员的能力资质、各类检测的环境与工况条件，以及检测报告的标准化格式。本章内容是开展第 4~7 章各项检测的**前提条件**。

---

## 3.1 检测仪器精度要求（★汇总表）

> [!important] 仪器管理核心
> 检测仪器的精度直接决定检测数据的可信度。JGJ/T 260-2011 对每类检测参数都规定了最低精度要求。所有检测仪器**必须在计量检定/校准有效期内**使用，并持有有效的检定证书或校准报告。

### 3.1.1 风系统检测仪器精度

| 检测参数 | 推荐仪器 | 精度要求 | 量程要求 | 备注 |
|----------|----------|:------:|:------:|------|
| **风管漏风量（差压）** | 微压计 / 数字差压计 | ±0.5 Pa | 0~2000 Pa | 漏风量法核心仪表 |
| **孔板/喷嘴流量** | 标准孔板 + 差压变送器 | ±1.5% FS | 按测试风量选配 | 孔板孔径须标定 |
| **风口风速** | 热球风速仪 | ±3% 读数 ±0.02 m/s | 0.05~5.0 m/s | 低风速首选 |
| **风管风速** | 叶轮风速仪 | ±2% 读数 | 0.5~30 m/s | 高风速/主风管 |
| **风管风速（精密）** | 毕托管 + 微压计 | ±0.5 Pa | — | 标准方法，需直管段 |
| **风量（风口）** | 风量罩 | ±3% 读数 + 5 m³/h | 40~4000 m³/h | 自带整流和均压 |
| **风管静压** | 数字微压计 / U 形管 | ±2% FS | 0~2500 Pa | — |
| **大气压力** | 空盒气压表 | ±200 Pa (±2 hPa) | 800~1060 hPa | 漏风量计算需空气密度修正 |
| **温度（风系统）** | 玻璃水银温度计或数字温度计 | ±0.5°C | -20~100°C | 空气密度修正用 |

### 3.1.2 水系统检测仪器精度

| 检测参数 | 推荐仪器 | 精度要求 | 量程要求 | 备注 |
|----------|----------|:------:|:------:|------|
| **水流量** | 超声波流量计 | ±1.5% 读数 | DN25~DN600 | 管外夹装，无需断管 |
| **水流量（备用）** | 电磁流量计 | ±0.5% 读数 | DN50~DN600 | 需管道预设法兰 |
| **水压力** | 精密压力表 | ±1.6 级 (1.6% FS) | 0~1.6 MPa / 0~2.5 MPa | 水压试验用 |
| **水压力（精密）** | 数字压力计 | ±0.5% FS | 0~2.5 MPa | 水泵扬程测量 |
| **水温** | 玻璃水银温度计 | ±0.1°C | 0~100°C | 供回水温差测量 |
| **水温（便捷）** | 数字温度计 / 铂电阻 | ±0.2°C | -20~150°C | 便携式 |

### 3.1.3 室内环境检测仪器精度

| 检测参数 | 推荐仪器 | 精度要求 | 量程要求 | 备注 |
|----------|----------|:------:|:------:|------|
| **干球温度** | 玻璃水银温度计 / 数字温度计 | **±0.5°C** | -10~50°C | 须防辐射屏蔽 |
| **相对湿度** | 通风干湿球湿度计 | **±5% RH** | 10%~100% RH | 标准方法 |
| **相对湿度** | 电子湿度计（电容/电阻式） | ±3% RH | 0~100% RH | 快速响应 |
| **空气流速（室内）** | 热球风速仪 / 热线风速仪 | **±3% 读数 ±0.02 m/s** | 0.05~5.0 m/s | 低风速，有方向性 |
| **噪声（A 声级）** | 精密声级计 | **I 型** (±0.7 dB) 或 **II 型** (±1.0 dB) | 30~130 dB(A) | 符合 GB/T 3785.1 |
| **噪声频谱分析** | 倍频程/1/3 倍频程滤波器 | — | 31.5~8000 Hz | NR/NC 曲线评价 |
| **照度** | 照度计 | **±5% 读数** | 1~20000 lx | 一级照度计 |
| **CO₂ 浓度** | 非色散红外 (NDIR) 分析仪 | ±(50 ppm + 5% 读数) | 0~5000 ppm | 室内空气品质 |
| **黑球温度** | 黑球温度计 (φ150mm) | ±0.5°C | 0~60°C | 平均辐射温度测量 |

### 3.1.4 设备检测仪器精度

| 检测参数 | 推荐仪器 | 精度要求 | 量程要求 | 备注 |
|----------|----------|:------:|:------:|------|
| **转速** | 光电/接触式转速表 | ±1 r/min | 10~9999 r/min | 风机/水泵 |
| **电机电流** | 钳形电流表 | ±2% 读数 | 0~600 A | 三相分别测量 |
| **电机电压** | 数字万用表 | ±1% 读数 | 0~600 V | 三相平衡度 |
| **电机功率** | 功率分析仪 / 三相电能表 | ±2% 读数 | — | 计算效率用 |
| **轴承温度** | 红外测温仪 | ±2°C 或 ±2% 读数 | -20~350°C | 非接触，快速 |
| **轴承温度（接触）** | 热电偶/热电阻温度计 | ±0.5°C | 0~200°C | 设备自带或粘贴 |
| **振动速度** | 振动计 | ±5% 读数 | 0~50 mm/s | 旋转机械振动评价 |
| **皮带张力** | 皮带张力计 | ±5% 读数 | — | 皮带传动风机 |

### 3.1.5 仪器管理要求

| 管理项目 | 要求 |
|----------|------|
| **检定/校准** | 所有检测仪器须由国家认可的计量检定机构定期检定/校准，并在有效期内使用 |
| **现场校准** | 每次检测前后，对关键仪表（微压计、温度计）进行现场零点和量程校准 |
| **仪器台账** | 检测单位须建立仪器台账，记录每台仪器的型号、编号、检定日期、有效期 |
| **标识管理** | 合格仪器贴绿色"合格"标签，限用仪器贴黄色"准用"标签，不合格仪器贴红色"停用"标签并隔离 |

## 3.2 检测人员资质

### 3.2.1 人员能力要求

| 岗位角色 | 资质要求 | 职责 |
|:------:|----------|------|
| **检测负责人** | ① 暖通空调/建筑环境相关专业大专以上学历<br/>② 从事暖通检测工作 ≥ 3 年经验<br/>③ 持有相关检测上岗证 | 制定检测方案、审核检测报告、技术把关 |
| **主检测员** | ① 暖通空调/机电相关专业中专以上学历<br/>② 从事暖通检测工作 ≥ 1 年经验<br/>③ 持有相应检测项目上岗证 | 仪器操作、数据记录、现场判定 |
| **辅助检测员** | 经培训合格，熟悉检测流程和安全操作规程 | 协助布点、仪器搬运、工况确认 |
| **报告编写员** | 熟悉检测数据处理方法，经授权 | 数据整理、计算、报告编制 |

### 3.2.2 人员数量要求

| 检测项目类型 | 最少人员配置 |
|:----------:|:----------:|
| 漏风量/漏光法检测 | ≥ 2 人（1 操作 + 1 记录/巡查） |
| 风量平衡检测 | ≥ 2 人（1 调节 + 1 测量记录） |
| 室内环境检测 | ≥ 2 人（1 仪器操作 + 1 数据记录） |
| 设备性能检测 | ≥ 2 人（1 仪表 + 1 安全监护） |
| 水系统检测 | ≥ 2 人（压力试验须设专人监护） |

### 3.2.3 安全要求

| 安全事项 | 要求 |
|----------|------|
| **高空作业** | 高处检测（≥ 2m）须系安全带，设监护人员 |
| **电气安全** | 接触带电设备检测时须持电工证，使用绝缘工具 |
| **压力试验安全** | 水压试验压力 ≥ 1.0 MPa 时，试验区域设警示标志，人员站在安全距离外 |
| **转动设备安全** | 检测风机/水泵时衣物收紧，严禁在设备运行时伸手触摸转动部件 |
| **个人防护** | 施工现场须佩戴安全帽、防护鞋，必要时佩戴护目镜和防噪耳塞 |

## 3.3 检测条件

> [!important] 检测条件不满足 = 检测结果无效
> 检测条件直接影响检测结果的真实性和代表性。JGJ/T 260-2011 规定，当检测条件不满足时，应在检测报告中**如实记录实际条件**，并评估其对检测结果的影响。

### 3.3.1 风系统检测条件

| 检测项目 | 工况条件 | 环境条件 |
|----------|----------|----------|
| **风管漏风量** | 被测风管段独立封堵隔离，不接入系统 | 无特殊环境要求（但风速影响微压计零漂） |
| **漏光法** | 风管内部清洁，无遮挡光源的杂物 | 🔴 **暗环境**（夜间或遮光室内），外部照度 ≤ 5 lx |
| **风量平衡** | 🔴 **系统风阀全部处于全开状态**（初调）或设计位置（复调） | 空调系统连续稳定运行 → |
| **风机性能** | 风机在系统设计工况下运行 | 空调系统连续稳定运行 → |

### 3.3.2 空调系统运行条件（★关键）

| 条件 | 要求 | 原因 |
|:----:|------|------|
| **连续运行时间** | 🔴 **≥ 2 小时**（空调系统在检测前须连续稳定运行至少 2 小时） | 消除启停过渡过程对温湿度、风速等参数的干扰 |
| **供冷/供热** | 按设计工况正常运行（夏季供冷工况 / 冬季供热工况） | 检测数据须代表典型运行工况 |
| **自动控制系统** | 投入正常运行（DDC、传感器、执行器处于自动控制状态） | 反映系统真实运行调节能力 |
| **末端设备** | 所有末端（风机盘管、VAV Box、风口等）正常运行，过滤器清洁 | 避免末端故障导致系统压力分布异常 |
| **新风系统** | 新风阀开度按设计设定（最小新风量或全新风工况按检测方案） | 影响室内温湿度和风量平衡 |

### 3.3.3 室外环境条件

| 条件 | 要求 |
|:----:|------|
| **室外风速** | 🔴 **≤ 3 级**（≤ 5.4 m/s，树枝轻摇级以下）。大风天影响建筑自然渗透风量、风口风量和室内温湿度分布 |
| **室外温度** | 在设计室外计算温度范围内（偏离过大时须在报告中说明并分析影响） |
| **降雨/降雪** | 不宜在暴雨或大雪天气进行室外风口风量和室内环境检测（湿度漂移大） |
| **日照** | 室内温湿度检测时应记录日照情况（窗帘开闭状态），避免太阳直射温度计探头 |

### 3.3.4 水系统检测条件

| 检测项目 | 工况条件 |
|----------|----------|
| **水压试验** | 管道系统安装完毕、阀门处于开启状态、系统最高点设排气阀、最低点设泄水阀 |
| **水力平衡** | 🔴 **系统所有阀门全部开启**（初调阶段），水泵在设计转速下运行 |
| **水流量测量** | 管道满管流、直管段长度 ≥ 10D（上游）+ 5D（下游），管内无气泡 |
| **水温测量** | 系统运行稳定后（≥ 30 min）测量，供回水温差 ≥ 3°C |

### 3.3.5 室内环境检测条件

| 条件 | 要求 |
|:----:|------|
| **照明** | 正常开启（照度检测时按设计工况） |
| **门窗** | 关闭（测噪声时门窗关闭，测温湿度/风速时按正常使用状态） |
| **人员** | 检测区域内除检测人员外，不应有其他人员活动（减少热源和气流干扰） |
| **测点保护** | 温度计探头须加防辐射屏蔽罩（避免周围壁面辐射影响） |

## 3.4 检测方法选择原则

| 检测阶段 | 检测方法选择原则 |
|:------:|------|
| **施工过程检测** | 优先选用快速、无损、可重复的检测方法（如漏光法初检→漏风量法抽检） |
| **调试检测** | 采用定量检测方法，以数据驱动系统平衡调节 |
| **竣工验收检测** | 采用标准方法（规程明确规定的仪器和步骤），检测数据须具备法定效力 |
| **仲裁检测** | 严格按 JGJ/T 260 规定的**基准方法**执行，不得简略步骤 |

## 3.5 检测报告格式

### 3.5.1 检测报告基本结构

JGJ/T 260-2011 规定的检测报告应包含以下基本内容：

| 序号 | 报告组成部分 | 内容要求 |
|:----:|:----------|----------|
| 1 | **封面** | 报告编号、工程名称、委托单位、检测单位（盖章）、报告日期 |
| 2 | **检测人员页** | 检测负责人、主检测员、辅助检测员签名 + 上岗证编号 |
| 3 | **工程概况** | 工程名称、地点、建筑面积、空调系统类型、设计参数概要 |
| 4 | **检测依据** | 列出引用的标准编号和名称（JGJ/T 260-2011、GB 50243-2016 等） |
| 5 | **检测仪器清单** | 仪器名称、型号、编号、精度等级、检定有效期、校准证书编号 |
| 6 | **检测条件记录** | 室外气象条件（温度/湿度/风速）、系统运行时间、阀门状态、照明/门窗状态 |
| 7 | **检测数据** | 按检测项目分类记录原始数据、计算公式、中间计算值和最终结果 |
| 8 | **判定结论** | 逐项给出"合格"或"不合格"的判定，不合格项须说明原因 |
| 9 | **附件** | 测点布置图、现场照片、仪器检定证书复印件、设计图纸相关页 |

### 3.5.3 检测报告管理要求

| 管理项目 | 要求 |
|----------|------|
| **编号规则** | 检测报告应有唯一性编号（建议：单位代码-年份-序号） |
| **签字盖章** | 检测人员手写签名，加盖检测单位检测专用章和 CMA 章（如适用） |

## 3.6 基本规定汇总速查

| 类别 | 核心要求 |
|:----:|------|
| **仪器精度** | 各参数仪器精度须满足 3.1.1~3.1.4 汇总表要求，所有仪器在检定有效期内 |
| **人员资质** | 检测负责人 ≥ 3 年经验；每项检测 ≥ 2 人；高工作业/电气操作持相关资质证 |
| **空调运行条件** | 🔴 系统**连续稳定运行 ≥ 2 小时**后方可开始检测 |
| **阀门状态** | 风系统检测：🔴 **所有风阀全部开启**（初调阶段） |
| **室外风力** | 🔴 **≤ 3 级**（≤ 5.4 m/s） |
| **漏光环境** | 暗环境，外部照度 ≤ 5 lx |
| **水压试验** | 试验压力 1.5 倍工作压力（≥ 0.6 MPa），稳压 ≥ 10 min |
| **检测报告** | 须包含：工程信息 + 仪器清单 + 检测条件 + 原始数据 + 判定结论 + 人员签名 |
