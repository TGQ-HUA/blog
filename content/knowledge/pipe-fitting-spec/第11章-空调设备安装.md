# 第11章 空调设备安装

> [!important] 章节定位
> 第11章（内含原第9章冷热源设备、第11章空调设备）涵盖空调系统中各类冷热源设备及末端设备的安装施工工艺，包括冷水机组、冷却塔、水泵、换热器、风机盘管和 VAV 末端等。本章与 第8章 风机与空气处理设备安装|第8章（风机与空气处理设备） 互补，共同构成完整的暖通设备安装施工体系。

---

## 一、冷水机组安装

### 1.1 基础准备

| 项目 | 技术要求 |
|------|----------|
| **基础验收** | 基础尺寸、标高、预留地脚螺栓孔位置须与设备安装图一致 |
| **基础强度** | 混凝土强度 ≥ C20，养护期 ≥ 14 天 |
| **基础平面度** | ≤ 5mm/m，全长 ≤ 10mm |
| **减振措施** | 冷水机组必须设置减振垫或弹簧减振器，与基础之间隔离 |

### 1.2 就位与找正

| 项目 | 技术要求 |
|------|----------|
| **搬运** | 使用滚杠或叉车搬运就位，螺杆式/离心式机组不得倾斜 > 15° |
| **找正** | 用框式水平仪在机组底座和压缩机底座上找平，纵横向水平度 ≤ 1mm/m |
| **标高** | 偏差 ≤ ±10mm |
| **中心线** | 与基础中心线偏差 ≤ 10mm |

### 1.3 附属设备与管线连接

| 项目 | 要求 |
|------|------|
| **冷冻水/冷却水管** | 管道与机组连接处须设**柔性接头**（橡胶软接头或波纹补偿器），防止机组振动传递至管道 |
| **阀门布置** | 蒸发器和冷凝器的进出水口均应设置闸阀/蝶阀 + 压力表 + 温度计 |
| **水流开关** | 蒸发器出水侧须安装水流开关（靶式），与压缩机启动联锁 |
| **电气接线** | 机组电源线按厂家图纸接线，注意相序保护 |

## 二、冷却塔安装

### 2.1 安装位置要求

| 项目 | 要求 |
|------|------|
| **通风** | 冷却塔四周及顶部应保证足够的通风空间，进风口距障碍物 ≥ 2 倍塔高 |
| **回风短路** | 多台冷却塔并排放置时，间距 ≥ 塔体直径（避免湿热空气短路回流） |
| **基础** | 钢架基础或混凝土基础，高于屋面 ≥ 200mm（防积水） |
| **排水** | 溢流管和排污管接入屋面排水系统 |

### 2.2 安装要点

| 项目 | 技术要求 |
|------|----------|
| **塔体水平度** | ≤ 2mm/m |
| **填料安装** | 填料块平整密实排列，无大间隙，防止气流旁通 |
| **布水器** | 旋转布水器转动灵活，布水均匀 |
| **风机安装** | 风筒与叶尖间隙均匀（偏差 ≤ 2mm），叶片角度一致 |
| **补水** | 浮球阀灵敏可靠，补水水位正常 |
| **减振** | 冷却塔基座与屋面之间设橡胶减振垫 |

---

## 三、水泵安装

### 3.1 基础与减振

| 项目 | 要求 |
|------|------|
| **基础高度** | 高出地面 ≥ 100mm，周边设排水沟 |
| **减振** | 水泵基座下设橡胶减振垫或弹簧减振器；进出水口安装橡胶软接头 |
| **地脚螺栓** | 二次灌浆后强度达标方可最终紧固 |

### 3.2 找正找平

| 项目 | 允许偏差 |
|------|:--------:|
| **泵体水平度（纵向）** | ≤ 0.1mm/m |
| **泵体水平度（横向）** | ≤ 0.2mm/m |
| **联轴器同轴度** | 径向位移 ≤ 0.05mm，角位移 ≤ 0.2/1000 |
| **皮带传动** | 两皮带轮端面在同一平面内，偏移量 ≤ 1mm |

### 3.3 管路连接

| 项目 | 要求 |
|------|------|
| **吸入管** | 由水池至泵入口应保持上升坡度（防止气囊），异径管用偏心大小头（顶平） |
| **排出管** | 安装止回阀 + 闸阀，止回阀在闸阀与泵之间 |
| **压力表** | 吸入口装真空压力表，排出口装压力表 |
| **冲洗** | 管道冲洗合格后方可与水泵最终连接，防止焊渣/杂物进入泵体 |

## 四、换热器安装

### 4.1 板式换热器

| 项目 | 要求 |
|------|------|
| **安装位置** | 留足板片拆装空间（单侧 ≥ 换热器宽度 + 500mm） |
| **水平度** | ≤ 2mm/m |
| **管道连接** | 一、二次侧介质流向按铭牌标示，不得接反 |
| **试压** | 安装后按工作压力的 1.25 倍进行水压试验 |

### 4.2 壳管式换热器

| 项目 | 要求 |
|------|------|
| **就位固定** | 底座与基础预埋钢板焊接或用地脚螺栓固定 |
| **水平度** | 冷凝器应有 ≥ 0.5% 的坡向集液包的坡度（利于排液） |
| **安全阀** | 壳程须装安全阀，排放口引至安全位置 |
| **温度计/压力表** | 冷热介质进出口均需安装 |

---

## 五、风机盘管安装

### 5.1 吊装要求

| 项目 | 技术要求 |
|------|----------|
| **吊装方式** | 使用专用吊杆（φ8~φ10 全螺纹吊杆）+ 减振吊钩 |
| **水平度** | 水平安装的风机盘管水平度 ≤ 1mm/m |
| **安装高度** | 回风口下缘与吊顶面平齐，风口可拆卸清洗 |
| **检修空间** | 电机侧和配管侧各留 ≥ 400mm 检修空间 |

### 5.2 管路连接

| 项目 | 要求 |
|------|------|
| **供回水管** | 采用铜管或镀锌钢管，连接处设**活接头**，进出水方向不得接反 |
| **电动阀** | 回水管上安装电动二通阀或三通阀，接线按电气图纸 |
| **Y 型过滤器** | 供水管入口侧必须安装 Y 型过滤器，滤网规格 ≥ 60 目 |
| **排气阀** | 盘管系统最高点设手动或自动排气阀 |
| **冷凝水管** | 排水坡度 ≥ 0.5%（比一般冷凝水管要求更严），不得接入污水管 |

### 5.3 电气与调试

| 项目 | 要求 |
|------|------|
| **接线** | 按接线图正确连接风机三速（高/中/低）和电动阀线 |
| **接地** | 金属外壳可靠接地 |
| **试运转** | 通电检查风机旋转方向、各档转速正常，无异常噪声 |
| **冷凝水** | 通水后检查凝结水盘排水通畅，无积水、无渗漏 |

## 六、VAV 末端（变风量末端装置）安装

### 6.1 安装要求

| 项目 | 技术要求 |
|------|----------|
| **安装位置** | 按设计图纸定位，通常安装于吊顶内，距送风口不宜 > 3m |
| **气流方向** | 箱体上的气流方向箭头必须与系统实际送风方向一致 |
| **吊装方式** | 使用专用吊挂装置，吊杆承受全部重量，不得由连接风管支撑 |
| **检修空间** | 控制箱/执行器侧预留 ≥ 500mm 操作空间 |

### 6.2 管路与传感器安装

| 项目 | 要求 |
|------|------|
| **入口直管段** | VAV 箱入口前须有 ≥ 3 倍管径的直管段（保证风速传感器测量准确） |
| **再热盘管** | 带再热的 VAV 末端，热水管连接要求同风机盘管 |
| **风量传感器** | 十字风速传感器安装方向正确，取压管传至控制器 |
| **风阀执行器** | 执行器轴与风阀轴同心，行程范围内无卡阻 |

### 6.3 控制接线与调试

| 项目 | 要求 |
|------|------|
| **DDC 接线** | 按控制器端子图接线（电源 24VAC、风速传感器、执行器、再热阀） |
| **地址设定** | DDC 控制器地址拨码与图纸一致，避免冲突 |
| **最小/最大风量** | 在 DDC 中设定最小风量（保证新风量）和最大风量（设计风量） |
| **功能测试** | 机组运行后逐一检查 VAV 末端响应温度调节信号正常 |

## 七、设备安装通用要求

### 7.1 减振降噪措施

| 设备类型 | 减振措施 |
|----------|----------|
| **冷水机组** | 弹簧减振器/橡胶减振垫 + 管道柔性接头 |
| **冷却塔** | 橡胶减振垫 + 风机变频控制（夜间低速） |
| **水泵** | 惰性减振台座 + 弹簧减振器 + 橡胶软接头 |
| **风机盘管** | 减振吊钩 + 送回风软连接 |
| **空气处理机组** | 弹簧减振器 + 帆布软接头（风口） |

### 7.2 设备安装偏差允许值汇总

| 项目 | 允许偏差 |
|------|:--------:|
| 设备平面位置 | ≤ ±10mm |
| 设备标高 | ≤ ±10mm |
| 设备水平度 | ≤ 1mm/m（一般设备）/ ≤ 0.1mm/m（泵/压缩机等精密设备） |
| 联轴器同轴度（径向） | ≤ 0.05mm |
| 联轴器同轴度（轴向） | ≤ 0.2/1000 |
| 皮带轮端面偏差 | ≤ 1mm |
