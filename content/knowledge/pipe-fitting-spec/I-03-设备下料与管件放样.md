# I.03 设备下料与管件放样

板宽限制、折件能力、静压箱堵面选择、特殊管件审批流程。

---

## I.03.1 板宽限制

| 规则               | 值                          |
| ---------------- | -------------------------- |
| 默认板宽             | 1250mm（不锈钢约1240mm）         |
| CAD 放样限制         | 含 **1230mm** 以内（特殊 1240mm） |
| 焊接件料宽 ≈ 1240 需拼板 | 务必调整排版/尺寸/五线开平             |

---

## I.03.2 折件限制

| 限制                | 参数                    |
| ----------------- | --------------------- |
| 横向最长              | **3150mm**（极限 3500mm） |
| 长度 > 2000，纵向      | ≤ 500mm               |
| 长度 > 2000 + 折 90° | 纵向 ≤ 250mm            |
| 焊接管件接口两端折边        | 至少留 **40mm** 直位       |
| 非特殊必要             | 宽度 ≥ 10mm             |
| 所有折弯位置            | 减 V 口                 |

---

## I.03.3 静压箱堵面选择

三个尺寸（长×宽×高）中选堵面：

优先级（从高到低）：
1. 长度 ≠ 600 且 ≠ 1200（避免不就料）
2. 最小尺寸 > 1400 → 大面做堵
3. 长度 ≤ 1400
4. 三个都 < 1200 → 开口面做堵

> 客户要求长度与就料差 50 以内 → 建议调整为就料长度。

---

## I.03.4 特殊管件审批

```
画放样图 → 发生产确认 → 上传下料
数量 > 3 → 先下样品 → 确认 pass → 量产
```

CAD 放样下料：实际下料内容须与示意图完全相符；放样料件分开排版，绘制人员上传并登记平方数；示意图排版前/下单后都须确认。
