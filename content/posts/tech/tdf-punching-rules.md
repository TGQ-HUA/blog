## 正文：共板法兰打孔的两种规则，你得知道

共板法兰是现在风管最常用的接口方式——不用焊角铁，法兰和生产一次成型。但打孔这件事，规矩一直在变。

风管厂原来一直用的打孔规则叫**固定首孔**：每边两端各留110mm，中间剩余的长度再均分。这个规则的来源已经说不清了，可能是早年跟着某家空调设备厂的工艺走的。

到了去年底我们发现：固定首孔经常出问题。

### 固定首孔（旧规则）的毛病

旧规则：首孔距管端固定110mm，中间（剩下长度-220mm）再均分。

打个比方，800mm的边长，孔距≤140mm：

```
中间区域 = 800 - 220 = 580mm
段数 = Ceil(580/140) = 5段 → 6个孔（含两端首孔）
实际中间孔距 = 580/5 = 116mm
```

算下来中间孔距116mm，但端头的首孔间距110mm——**两端和中间的孔距不一致**。

这个"不一致"就是安装队老刘骂人的原因。两根管子同样的宽度，A管打了5个孔，B管打了6个孔，孔位偏移。法兰螺丝孔对不上，要么硬塞要么扩孔，现场打架。

而且这个110mm首孔距还不是国标规定的。GB50243-2016里关于法兰连接螺栓的规定是"风管边长≤250mm时螺栓间距≤100mm；边长＞250mm时螺栓间距≤150mm"，压根没提固定首孔这回事。**110mm是当初抄了某家设备厂商的标准，各家厂不一样。** 你厂110，他厂100，工地一碰就乱。

### 整边均分（新规则）怎么解决

今年初我们改了规则，改成**整边均分**：

```
段数 = Ceil(边长 / 要求孔距)
实际孔距 = 边长 / 段数
```

还拿800mm边长、孔距≤140mm举例：

```
段数 = Ceil(800/140) = 6段 → 5个孔
实际孔距 = 800 / 6 = 133.33mm ✓
```

孔全部均匀，每两孔间距一样，133.33mm。两根800mm的管子不管谁做、什么时候做，打出来都一样。安装队那边的孔位就对得上了。

边界条件也改了两条：

| 条件 | 旧规则（固定首孔） | 新规则（整边均分） |
|------|-----------------|-----------------|
| 首孔边距 | 固定110mm | 自适应 |
| 超短边不打 | 边长＜220mm不打孔 | 边长＜150mm不打孔 |
| 孔距极小时 | — | 孔距＜100mm时用固定首孔兜底 |
| 孔位一致性 | 不同边长孔数可能不同 | 同边长同结果 ✓ |

### 为什么改规则不只是技术问题

改打孔规则，技术上很简单——改几行计算代码的事。真正难的是**让车间和设备按新规则走**。

风管厂的共板法兰机原来一直按固定首孔跑的，机器的打孔程序写死了。换规则意味着：

1. **CAMduct的接口参数要改**——法兰口数据重新配
2. **设备打孔程序要更新**——共板法兰机的行程重新设
3. **车间师傅要适应**——你让一个干了8年的师傅改习惯，比改代码难三倍
4. **库存零件要核对**——老规则打的半成品和配件，改完后就不能混用了

当时为了推这个事，技术部做了一个决策：**2026年1月6日之后的新项目统一用新规则，存量未完结项目继续走旧规则。** 两套并存，给车间三个月过渡期。

还在CAMduct里加了配置项，让工人一眼能看清当前订单走哪套规则——不需要记日期，系统自己判断。

## 延伸：一个打孔规则折射出的行业问题

共板法兰打孔这件事，说大不大，说小不小。

往小了说，就是一行代码的事。往大了说，它反映出风管行业一个普遍问题：**太多工艺参数是"历史遗留"而不是"标准定义"的。**

你去问10家风管厂的打孔首孔距，可能能问出5种不同的数——110、100、90、80、75。每家都说"我们一直这么干的"。但问依据是什么？没人说得清。

GB50243给了螺栓间距的上限，但没有规定首孔边距和均分方式。JGJ141-2017也一样。这就给每家厂留下了自己"发挥"的空间。问题是这些"发挥"在工地上碰上了，就对不上。

我们这次换成整边均分，说到底不是发明了什么新技术，而是**把"约定俗成"变成了"可计算的规则"**。现场师傅不需要记首孔多少，也不用看什么对照表，给边长和孔距，算出来就是标准答案。

这也是我一直在推的事情——**把老师傅脑子里的"凭感觉"，变成系统里跑得出来的"凭规则"。**

## 实操：关于打孔规则的一张速查表

做共板法兰的，下面这几条可以直接拿去用：

### 新规则（推荐，整边均分）

| 边长范围 | 孔数(孔距≤150mm时) | 实际孔距 | 备注 |
|----------|-------------------|---------|------|
| ＜150mm | 0 | — | 超短边，不打孔 |
| 150-300mm | 1~2 | 自适应 | 小边，基本符合国标 |
| 300-600mm | 2~4 | 自适应 | 常规场景 |
| 600-1000mm | 4~7 | 自适应 | 建议确认安装队孔距要求 |
| ＞1000mm | ≥7 | 自适应 | 大边，注意设备行程能力 |

### 切换到新规则的检查清单

1. □ CAMduct接口参数更新（法兰口数据重新配）
2. □ 共板法兰机打孔程序更新（行程参数）
3. □ 通知车间师傅新规则生效日期
4. □ 存量订单标注"旧规则"，新订单走新规则
5. □ 安装队同步——告诉对方我们改规则了，孔位会均匀
6. □ 配件库存半成品做好防混用标记
