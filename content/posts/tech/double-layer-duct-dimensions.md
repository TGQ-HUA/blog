## 引子：一个真实问题

上个月车间老周跑来，一脸苦相。双层保温弯头做完了，外壁装上去才发现短了快20公分。我说那肯定是内外壁尺寸搞混了——单子上写个"900×500 R300"，谁知道是内胆尺寸还是外壁尺寸？

老周说："单子就一行字，我哪知道哪层是哪层。"

这事真不赖老周。双层保温管这玩意儿，三层结构——内胆包棉再套外壁，听着不复杂，但内外壁的尺寸换算没个死规矩，十有八九要出事。料废了、棉废了、工时也搭进去了，碰上工期紧的时候，师傅们心态直接崩。

## 正文：一个200mm的差值，管住所有件

双层保温管的核心逻辑其实一句话就能说清：**内胆负责走风，外壁负责保温和保护。所以内胆要伸出来对接，外壁往里缩。**

### 核心公式

内外壁的尺寸关系，所有管件类型都走同一套规则：

| 参数 | 内胆 → 外壁关系 |
|------|----------------|
| **长度** | 外壁长 = 实际长 - 200mm |
| **弯头R** | 外壁R = 内胆R - 夹层厚度 |
| **直位** | 外壁直位 = 内胆直位 - 100 - 勾股偏差 |
| **S弯/大小头 长** | 外壁长 = 实际长 - 200（同直管） |
| **支咀长** | 外壁咀 = 内胆咀 - 100 - 勾股偏差 |

这套公式来自风管厂排版规范手册（L.01.1.1），所有管件通用。

> **勾股偏差**到底是什么？内外壁半径差导致斜边长度差。夹层厚50mm时，这个差值约20.7mm。不是什么精密计算，但少了它内外壁就对不上。

### 直管——最基础的案例

圆形直管，常规长度1230mm：

| | 外壁 | 内胆 |
|--|------|------|
| 开料长度 | 1000mm（1米宽钢卷） | 1250mm（常规钢卷） |
| 实际长度 | 实长 - 200 | 实长 + 20 |

矩形直管也一样。共板1160接口的矩形双层管，外壁长度960mm，内胆长度按1160+做。两边各让出100mm给对接搭接，内胆额外留20mm。

### 弯头——最容易翻车的地方

弯头内外壁要**独立下料**，不能共用排版参数。内胆R更大、直位更长，内外径不一样。

拿最常见的90°矩形弯头举例（R300，夹层50）：

| 部件 | 直位 | R | 分段 |
|------|------|---|:----:|
| **内胆** | 170.7mm（外壁直位+100+20.7） | 129.3mm（外壁R+29.3） | 3段 |
| **外壁** | 50mm | 100mm | 3段 |

内胆直位比外壁直位多出120.7mm（100+勾股偏差20.7），这个数不是拍脑袋定的，是勾股定理算出来的。

### 其他管件

S弯、大小头、三通、四通——内外壁换算规则都是同一套逻辑：外壁比内胆短200mm，直位多出来的部分按勾股偏差修正。

具体参数在规范手册里都有表，这里不全部抄了。车间师傅记住一句话就行：**凡是双层保温管，先确认单子上标的是"内胆"还是"外壁"，然后再下料。**

## 延伸：为什么小厂不爱接双层保温管的单

双层保温管这个品类，大厂做得熟，小厂接了就亏。

原因不是技术多高深，而是**工序翻倍**。一根管子，要经过：做内胆 → 包保温棉 → 做外壁 → 套外壁 → 封口，五道工序顶普通风管三倍的工时。而且料耗算不准——内外壁单独下料，边角料比普通管多出一截。

行业里很多厂对双层保温管的处理方式是"先报价再说，做的时候再算"。这种态度碰到验收严格的工地，就容易出纠纷。GB50243-2016里对保温风管有明确要求（第10.1节），但很多人只看了厚度要求，没看结构要求。

## 实操：一张表搞定

**双层保温管内外壁速查表（夹层50mm通用）**

| 参数 | 规则 | 注意 |
|------|------|------|
| 外壁总长 | 实长 - 200mm | 带法兰的按法兰面算 |
| 内胆总长 | 实长 + 20mm | 对接处留10mm搭接 |
| 内胆R | 外壁R + 夹层厚 + 勾股偏差(~20.7) | R值小于100时单独校验 |
| 外壁R | 按设计要求 | 最小值满足设备能力 |
| 内外胆对接 | **独立下料，不能合并** | 排版软件里两套参数 |
| 直位差异 | 内胆=外壁+100+勾股(~20.7) | 45°和90°分开算 |

**车间贴墙上吧，省得每次来问。**
