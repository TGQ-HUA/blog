## 引子：一个"老问题"又来了

上周三技术部小刘拿着图纸来问我："唐工，这个1000×400的风管，用0.75的板行不行？"

我说："你问过是做什么系统的吗？"

他愣了。图纸上就标了个尺寸和走向，没标风压类别。

这就是风管厂最常见的"盲猜厚度"问题——一张厚度选用表能解决的事，每隔几个月就要闹一次。有时候是车间自作主张用了薄板，风管装上鼓包了；有时候是技术部自己忘了问清楚就下了料，安装现场才发现板厚不对，返工。

## 板材厚度到底看什么

先说个最简单的事：国标GB50243-2016第4.2.1条写得清清楚楚，选厚度看三个东西：

1. **风管长边尺寸**——越长越厚
2. **风压类别**——压力越高越厚
3. **材质**——镀锌板、不锈钢、防火板各有一套表

这三个缺一个，厚度就定不了。小刘那个1000×400的风管，如果是低压送风，0.75mm够了；但如果是排烟高压系统，至少要1.0mm。

### 最容易踩的坑：风压类别没搞清楚

车间里经常出现这种情况：图纸上只标了尺寸和位置，没标系统类型。师傅按惯例"用0.75"，结果这个风管是排烟系统的（高压），装上去一开机就抖。

**区分风压类别（GB50243-2016）：**

| 风压类别 | 管内正压 (Pa) | 常见用途 |
|----------|--------------|---------|
| 微压 | P ≤ 125 | 一般通风 |
| 低压 | 125 < P ≤ 500 | 空调、通风 |
| 中压 | 500 < P ≤ 1500 | 空调主风管 |
| 高压 | 1500 < P ≤ 2500 | 排烟系统 |

排烟风管基本都是高压。排烟口一启动，那个压力不是送风能比的。

### 矩形镀锌风管板材厚度速查表

这是风管厂日常最常用的表，直接按长边尺寸和风压类别查：

| 长边尺寸 b (mm) | 微压/低压 | 中压 | 高压 |
|----------------|----------|------|------|
| b ≤ 320 | 0.5 | 0.5 | 0.75 |
| 320 < b ≤ 450 | 0.5 | 0.6 | 0.75 |
| 450 < b ≤ 630 | 0.6 | 0.75 | 1.0 |
| 630 < b ≤ 1000 | 0.75 | 0.75 | 1.0 |
| 1000 < b ≤ 1500 | 1.0 | 1.0 | 1.2 |
| 1500 < b ≤ 2000 | 1.0 | 1.2 | 1.5 |
| 2000 < b ≤ 4000 | 1.2 | 1.2 | 按设计 |

**几个关键分界点要记牢：**
- 边长超过630，低压就要上0.75了
- 边长超过1000，低压最少1.0
- 高压系统，不管边长多少，最低0.75起步

### 不锈钢板的差别

不锈钢风管有另一套规矩。因为不锈钢的强度比镀锌板略高，但价格贵得多，厚度选用主要看正压还是负压/高压：

| 长边 b 或直径 D (mm) | 正压风管 | 高压/负压风管 |
|---------------------|----------|-------------|
| 100 < b ≤ 500 | ≥ 0.5 | ≥ 0.75 |
| 320 < b ≤ 1120 | ≥ 0.75 | ≥ 1.0 |
| 1120 < b ≤ 2000 | ≥ 1.0 | ≥ 1.0 |
| 2000 < b ≤ 4000 | ≥ 1.2 | ≥ 1.2 |

不锈钢没有"0.6"这个常规厚度，常用的是0.5跳0.75，下单时别按镀锌板的习惯选。

### 圆形螺旋风管

圆形风管因为结构原因，同等条件下可以用更薄的板（圆形的受力特性比矩形好）。但这不代表可以乱用——负压系统比正压系统要求更厚，因为负压下风管容易被吸扁。

## 延伸：为什么小厂"统一用0.75"反而埋雷

我跟不少同行聊过，小厂普遍的做法是"统一0.75，碰上大的再加"。理由是省事、好备料。

这个做法在以前问题不大。20年前的风管系统大多是低压送风，边长很少超过1000。但现在不一样了：
- 排烟系统越来越多，高压风管占比明显上升
- 项目规模越来越大，边长1500、2000的风管很常见
- 监理验收越来越严，拿游标卡尺一量，板厚不够当场退货

我们厂以前也吃过这个亏。一批排烟风管用了0.75的板，安装完开机鼓包了，甲方直接要求全部拆掉重做。那批货退回来，光拆就花了一周。

现在我们的做法是：来图先确认系统类型，系统类型不明确的直接问甲方或设计院。厚度选好后在加工单上用红字标出来，车间师傅拿到单子不用猜。

## 实操：一张表搞定厚度确认

技术部下单时，按这个流程走一遍：

```
收到图纸 → 确认风管尺寸 → 确认系统类型(排烟/空调/通风) 
→ 查风压类别(高压/中压/低压) → 查厚度表 → 标注在加工单
```

如果图纸上没标系统类型，**直接问，别猜**。猜错了返工的是整个车间。

## 结尾

说来也简单。一张厚度表就能解决的事，但前提是技术部得知道自己查的是什么系统。做风管这行，最难的不是记数据，是把该问的问清楚。老李现在学聪明了，来图纸先问"这是排烟还是送风"，省了多少事。
