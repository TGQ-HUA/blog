车间做 CAMduct 排料，每次都要：复制编码 → 手动改材质 → 改接口 → 选图库 → 启动 CAMduct → 点预设 → 创建新文档 → 存盘。

一套下来半分钟，一天几十次就是半小时。关键是步骤一多就容易漏，漏了就得返工。

于是写了这个自用小工具，全自动搞定。

---

## 主要功能

### 一、编码自动处理

从 CAMduct 里复制编码行 → 粘贴到剪贴板 → 工具自动读取并处理：

- **材质替换**：编码里的"对应材质"自动换成你选的角铁材质
- **工艺处理**："是否压筋"换成你预设的压筋规则
- **接口替换**：支持共板法兰接口批量互换（固定首孔 ↔ 整边均分）
- **项目命名**：自动提取客户地址，格式化项目名称和标签
- **图库前缀**：根据图库类型自动加 `[咬口]`/`[焊接]`/`[防火]` 等标记

处理完输出 `D:\BM.txt`，CAMduct 直接导入。

### 二、全自动启动 CAMduct

处理完编码后自动：
1. 启动 CAMduct（带 `/L=ChineseTraditional`）
2. 选中指定预设（如 TK1-V23.9.25）
3. 点击"创建新文档"
4. 展开文件菜单 → 保存为 `D:\BM.txt`

全程不用动鼠标，进度条走完就行。

### 三、材质/工艺/图库管理

三个独立列表，支持增删：
- **材质**：角铁、镀锌板等材质名称
- **工艺**：压筋、不压筋等处理方式
- **图库**：咬口管、焊接管、防火板、静压箱等模板

数据存在 `编码导入数据.ini`，重启不丢。

### 四、悬浮窗模式

主窗口用完点 × 不退出，缩成右下角一个圆形小图标。点一下恢复窗口，右键退出。

窗口关掉也不碍事——开着占地方，关了下次还得重新启动。悬浮窗刚好卡在中间：不占桌面，随叫随到。

### 五、开机自启

勾上"开机时自动启动"，系统开机后自动加载。配合悬浮窗模式，开机就在右下角等着。

## 界面布局

窗口分六个区：

| 区域 | 内容 |
|------|------|
| 材质处理 | 角铁材质下拉 + 增删按钮 |
| 工艺处理 | 压筋方式选择 + 增删按钮 |
| 名称设置 | 项目名称 + 标签字符 |
| 接口调整 | 启用/禁用接口替换 + 原始/目标接口选择 + 共板打孔规则 |
| 软件设置 | 图库选择 + CAMduct 路径 + 开机自启 |
| 启动设置 | 等待秒数 + 执行按钮 |

底部「读取剪切板内容」载入编码，「执行操作」一键处理。

---

## 使用流程

1. CAMduct 里复制编码行（`Ctrl+C`）
2. 切到编码处理助手，点「读取剪切板内容」
3. 确认材质、工艺、图库、启动时间
4. 点「执行操作」
5. 等进度条走完，CAMduct 自动完成保存

全程 3 秒操作 + 自动等待 CAMduct 启动即可。

---

## 注意事项

- 使用前确保 CAMduct 路径正确（软件设置 → 选择 EXE）
- 接口替换时注意"Z共板"的特殊处理——会弹窗提示替换数量
- 个人简称会作为公司地址写入编码，不要留空
- 编码必须包含 `JOBHEADER_START` 标记，否则工具不识别
