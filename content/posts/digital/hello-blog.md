## 自我介绍

我叫奇美星，风管厂技术部主管，93 年生。

从 2012 年进入社会，干过家具导购、五金设计，最后在暖通风管制造行业扎下了根。从技术员做到技术主管，带着 4 个人的小团队，负责报价、下料、排版、绘图、NC 加工的全流程技术工作。

## 为什么要写博客

**一句话：暖通风管行业太缺技术分享了。**

我在自学 Delphi 编程的过程中，深刻体会到"找不到资料"的痛苦。搜"风管展开算法"，没几篇有用的；搜"CAMduct 教程"，能找到的都是皮毛。

很多实战经验只存在于老师傅的脑子里，一旦人走了，经验就断了。

我希望通过这个博客，把这些年积累的东西写下来：

- 风管制造的技术知识
- Delphi 编程的自学经验
- WPS 宏开发的实战代码
- 中小制造企业数字化转型的思路

## 我的工具链

| 工具 | 用途 |
|---|---|
| Delphi (RAD Studio 12) | 自研管理系统和绘图工具 |
| WPS JS 宏 | 表格自动化处理 |
| MySQL | 数据存储 |
| CAMduct 2020 | NC 码输出和加工 |
| Solidworks | 三维建模 |
| AutoCAD | 二维图纸 |
| Obsidian | 知识管理 |

## 这个博客的原则

1. **只写实战验证过的东西**——没跑通的不写
2. **代码能跑**——贴出来的代码都是生产环境用过的
3. **小白友好**——我自己就是从零自学的，知道什么地方容易卡住
4. **不藏着掖着**——同行之间互相成就，不搞技术壁垒
