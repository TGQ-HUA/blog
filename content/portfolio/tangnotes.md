## 项目概述

> 一款 Windows 桌面便签工具，支持**记忆笔记**和**任务管理**两大核心功能。深色/护眼双主题，系统托盘运行，数据纯本地存储（JSON），无需联网。

基于 Python + pywebview（Edge WebView2），MIT 开源协议。

## 四大功能模块

### 🧠 记忆便签

管理碎片化笔记、备忘、知识点。

- **标签式标题**：上方横向排列，点击切换，`+` 新建
- **Markdown 编辑**：支持加粗、代码、字体大小、颜色等格式
- **回收保护**：删除移入回收站，可恢复

### ✅ 任务便签

管理待办事项，支持多任务组分类。

- **分组管理**：目标计划 / 短期任务 / 编程计划等自定义分组
- **一键完成**：勾选 → 确认 → 自动移至时间线
- **编辑/删除**：鼠标悬停显示操作按钮

### 🗑 回收站

记忆便签删除内容的临时存放区。

- **恢复**：一键还原到原位置
- **彻底删除**：不可恢复的清理
- **清空**：批量清理回收站

### 📊 时间线

已完成任务按**年月分组**展示，包含创建/完成时间、花费天数。支持取消完成（恢复到任务面板）。

## 编辑弹窗

| 功能 | 格式 |
|------|------|
| 加粗 | `**文字**` → **文字** |
| 倾斜 | `*文字*` → *文字* |
| 强调高亮 | `****文字****` → 🔆 背景标记 |
| 代码 | `` `代码` `` |
| 字号 | `[16]文字[/]`（12/16/20 三档） |
| 颜色 | `[#E67E22]文字[/]`（🟠🔴🔵） |

**上方编辑，下方实时预览**。

## 窗口操作

| 操作 | 方式 |
|------|------|
| 拖拽移动 | 标签栏左侧 `⋮⋮` 手柄 |
| 缩放窗口 | 右下角折角手柄拖拽 |
| 隐藏到托盘 | 点击 ✕ 按钮 |
| 退出程序 | 托盘图标右键 → 退出 |

## 技术栈

| 技术 | 用途 |
|------|------|
| Python | 后端逻辑，JSON 数据读写 |
| pywebview + Edge WebView2 | 桌面壳，HTML/CSS/JS 前端渲染 |
| EasyMDE | Markdown 编辑器 |
| JSON | 本地数据存储（`%APPDATA%\TangNotes\data.json`） |
