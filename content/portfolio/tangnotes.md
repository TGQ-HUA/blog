## 项目概述

> 个人自研的轻量级桌面任务便签工具。绿色系极简风格，支持任务管理、记忆存档、Markdown 编辑、时间线追踪。

## 核心功能

| 模块 | 说明 |
|------|------|
| **任务管理** | 分类管理待办事项，支持目标计划/短期任务/编程计划等分组 |
| **记忆存档** | 碎片化信息快速记录，支持 Markdown 编辑（EasyMDE） |
| **回收站** | 误删恢复，数据不丢失 |
| **时间线** | 按时间回溯操作记录 |
| **本地存储** | JSON 文件存储，无需数据库，轻量便携 |

## 技术栈

| 技术 | 用途 |
|------|------|
| Python（Tkinter/WebView） | 桌面壳 |
| HTML + CSS + JS | 前端界面 |
| EasyMDE | Markdown 编辑器 |
| JSON | 本地数据存储 |
