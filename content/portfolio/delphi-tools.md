## 项目概述

从 2018 年开始陆续开发，目前已累计 **26 款**独立绘图计算工具，覆盖暖通风管行业**管件排版规范 A~O 十五类管件**。

> 每一款工具都经过生产一线验证，算法来自国标公式和多年实战经验。

---

## 覆盖管件类型

| 分类编号 | 管件类型 | 工具数量 |
|---------|---------|---------|
| A | 直管 | 2 款 |
| B | 弯头 | 4 款 |
| C | 三通 | 3 款 |
| D | 四通 | 1 款 |
| E | 天圆地方 | 3 款 |
| F | 变径管 | 2 款 |
| G~O | 其他管件 | 11 款 |

---

## 工作原理

```
输入管件参数（规格、角度、节数等）
  → 展开算法计算（绘图公式库 6000+ 行）
  → 自动生成 CAD 图纸
  → 支持导出 DXF / 直接调 CAD API 绘制
```

---

## 技术特点

- **绘图公式库**：6000+ 行 Pascal 代码，全中文命名，每个管件有独立的计算单元
- **参数化驱动**：输入参数即出图，无需手动画图
- **国标验证**：算法对照国标公式实现，实测误差 < 1mm
- **版本管理**：每个工具保留完整历史版本（__history 目录）

---

## 技术栈

- **开发语言**：Delphi (RAD Studio 12)
- **核心库**：绘图公式库.pas（6000+ 行）
- **输出格式**：CAD 图纸 (.dwg / .dxf)
