## 项目概述

把多年生产实战中积累的排版经验，系统化整理为**标准化规范文档**。覆盖暖通风管行业 A~O 十五类管件，每类管件包含排版规则、做单规范、算量方法。

> 行业内少有的系统化排版规范——大多数工厂靠口口相传，我们靠文档。

---

## 覆盖范围

| 编号 | 管件类型 | 内容 |
|------|---------|------|
| A | 直管 | 排版规则 + 做单规范 + 算量 |
| B | 弯头 | 排版规则 + 做单规范 + 算量 |
| C | 三通 | 排版规则 + 做单规范 + 算量 |
| D | 四通 | 排版规则 + 做单规范 + 算量 |
| E | 天圆地方 | 排版规则 + 做单规范 + 算量 |
| F~O | 其他管件 | 排版规则 + 做单规范 + 算量 |

另含 **9 项国标解读**，对照国标公式说明计算逻辑。

---

## 输出格式

- **在线版**：HTML 网页手册（build_kb.py 自动构建）
- **离线版**：CHM 帮助文件（Windows 双击即可查看）
- **源文件**：Obsidian Markdown 笔记

---

## 协作验证

- 排版规则经**制造部**实际操作验证
- 短直管接缝规则经焊接组组长应金坤确认
- 持续迭代中，新规则经讨论后纳入
