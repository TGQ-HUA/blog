# 风管板材厚度——一张表的事，车间为什么总搞错？

上周三技术部小刘拿着图纸来问我："唐工，这个1000×400的风管，用0.75的板行不行？"

我说："你问过是做什么系统的吗？"

他愣了。图纸上就标了个尺寸和走向，没标风压类别。

这种事在风管厂太常见了——一张厚度选用表能解决的事，隔三差五就要闹一次。今天把这问题彻底讲明白。

---

## 一、板材厚度看三个东西

国标**GB50243-2016第4.2.1条**写得清清楚楚，选厚度就看三样：

1. **风管长边尺寸**——越长越厚
2. **风压类别**——压力越高越厚
3. **材质**——镀锌板、不锈钢、防火板各有一套表

这三个缺一个，厚度就定不了。小刘那个1000×400的风管，如果是低压送风，0.75mm够了；但如果是排烟高压系统，至少要1.0mm。

## 二、最容易踩的坑：风压类别没搞清楚

图纸上只标了尺寸和位置，没标系统类型。师傅按惯例"用0.75"，结果风管是排烟系统（高压），装上去一开机就抖。

**风压类别速查（GB50243-2016）：**

| 风压类别 | 管内正压 (Pa) | 常见用途 |
|----------|--------------|---------|
| 微压 | P ≤ 125 | 一般通风 |
| 低压 | 125 < P ≤ 500 | 空调、通风 |
| 中压 | 500 < P ≤ 1500 | 空调主风管 |
| 高压 | 1500 < P ≤ 2500 | 排烟系统 |

排烟风管基本都是高压。排烟口一启动，那个压力不是送风能比的。

## 三、矩形镀锌风管板材厚度速查表

最常用的表，直接按长边尺寸和风压类别查：

| 长边尺寸 b (mm) | 微压/低压 | 中压 | 高压 |
|----------------|----------|------|------|
| b ≤ 320 | 0.5 | 0.5 | 0.75 |
| 320 < b ≤ 450 | 0.5 | 0.6 | 0.75 |
| 450 < b ≤ 630 | 0.6 | 0.75 | 1.0 |
| 630 < b ≤ 1000 | 0.75 | 0.75 | 1.0 |
| 1000 < b ≤ 1500 | 1.0 | 1.0 | 1.2 |
| 1500 < b ≤ 2000 | 1.0 | 1.2 | 1.5 |
| 2000 < b ≤ 4000 | 1.2 | 1.2 | 按设计 |

**几个关键分界点要记牢：**
- 边长超过630，低压就要上0.75了
- 边长超过1000，低压最少1.0
- 高压系统，不管边长多少，最低0.75起步

## 四、不锈钢板和圆管的差别

**不锈钢风管：** 强度比镀锌板略高，但价格贵得多。不锈钢没有"0.6"这个常规厚度，常用的是0.5跳0.75，下单时别按镀锌板的习惯选。

**圆形螺旋风管：** 同等条件下可以用更薄的板（圆形受力特性比矩形好）。但负压系统比正压要求更厚，负压下容易被吸扁。

## 五、为什么小厂"统一用0.75"反而埋雷

跟不少同行聊过，小厂普遍做法是"统一0.75，碰上大的再加"。理由就是省事。

这个做法在以前问题不大。20年前的风管系统大多是低压送风，边长很少超过1000。但现在不一样了：
- 排烟系统越来越多，高压风管占比明显上升
- 项目规模越来越大，边长1500、2000的风管很常见
- 监理验收越来越严，游标卡尺一量，当场退货

我们厂以前也吃过这个亏。一批排烟风管用了0.75的板，安装完开机鼓包了，甲方要求全部拆掉重做，光拆就花了一周。

## 六、技术部操作流程

```
收到图纸 → 确认风管尺寸 → 确认系统类型(排烟/空调/通风) 
→ 查风压类别(高压/中压/低压) → 查厚度表 → 标注在加工单
```

如果图纸上没标系统类型，**直接问，别猜**。猜错了返工的是整个车间。

---

说来也简单。一张厚度表就能解决的事，但前提是技术部得知道自己查的是什么系统。做风管这行，最难的不是记数据，是把该问的问清楚。
