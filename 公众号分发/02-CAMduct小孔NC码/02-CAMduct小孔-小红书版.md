## 正文

用CAMduct输出NC码
大孔正常，小孔（≤5mm）全部消失❓
找了两天原因……

结果就是一个参数的事😅

---

🔍 原因：
CAMduct有个隐藏参数叫「Pierce Size」（穿孔尺寸）
意思是：小于这个值的孔→不切割
默认值 = 5mm
所以5mm以下的孔全被跳过了！

---

✅ 解决方法（6步）：
1️⃣ File → Setup → Installed Machines
2️⃣ 选你的机床
3️⃣ Tool Setup → Global
4️⃣ 找到 Pierce Size
5️⃣ 改成 0 或 1mm
6️⃣ 保存，重新出码

搞定！所有版本CAMduct通用✔️

---

⚠️ 改了还不行？检查这3个：
• Pierce Delay 设为0.1秒
• 自动嵌套的孔缝合值
• 机床有没有添加切割刀具
