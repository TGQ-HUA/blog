# 风管咬口——公口7mm母口30mm，但什么时候该换？

> 奇美星暖通风管制作 | tangguoqi.top

---

## 引子

上个月生产单下来一批不锈钢排烟管，车间老李按老规矩做了大小边咬口。结果板子送到折弯机上一压——公口7mm那头直接崩了，裂了三条缝。

老李骂了一句，跑来问我："不锈钢不是能干咬口吗？"

能是能，但有限制。咬口最厚：镀锌1.5mm，不锈钢1.2mm。不锈钢1.5mm还做大小边？不崩才怪。

问题不在老李，在我——去年整理规范时折腾CAMduct参数，忘了给车间做咬口选型培训。师傅们都觉得"风管=大小边咬口"，不知道什么时候该换。

---

## 一、三种基本咬口

风管接缝按形状分三大种：平咬口、立咬口、联合咬口。每种下面还有变种。

### 1. 联合咬口（大小边）——95%的情况下用这个

就是车间说的"大小边"或"公母口"。公口（小边）7mm，母口（大边）30mm。流水线直接成型，效率高，密封性好。

**限制：**
- 镀锌板 ≤ 1.5mm，不锈钢板 ≤ 1.2mm
- 201#材质料厚≥1.0mm建议改直焊边

不锈钢1.5mm做大小边那次，公口边崩了三条缝。后来批了螺丝接缝，再没出过事。

**反向大小边（反咬口）：** 母口7mm，公口30mm——用在特殊弯头排版时公母对调。不是搞反了，是故意反的。

### 2. 平咬口——拼接的命门

风管展开宽度超1250mm（激光标准板宽）时，必须拆成多块用平咬口拼接。

标准数据：公口20mm + 母口20mm，合计40mm拼缝耗料。

常见于大截面风管（1600×800以上）、弯头弧板超宽、非标件。

拼接缝方向没注意那次，弯头滚圆卡住了。管件排版规范A.01明确写了：弯头/S弯的拼接缝注意方向，不得妨碍滚圆。

圆形风管也有平咬口：15+15mm，单机压制成型。

### 3. 立咬口——圆弯头的灵魂

圆形风管分段连接时用的。单条拼接缝15+15mm。

最典型用法：圆弯头分段展开后用立咬口连接成型。

---

## 二、厚板怎么办？螺丝接缝与焊接

当板厚超过咬口上限，螺丝接缝是替代方案。搭30mm，对穿孔锁螺丝。

好处是没有厚度限制，厚板薄板都能做；坏处是要打孔要锁螺丝，比咬口慢。

不锈钢风管、厚板焊接件还可以走焊接路线：

| 类型 | 搭接量 | 工艺 |
|------|--------|------|
| 搭30mm | 30mm | 碰焊或人工焊接 |
| 搭10mm | ~10mm | 焊接 |
| 直焊边 | 0mm | 无附加耗材 |

## 三、一张选型表搞定

| 条件 | 推荐接缝 | 理由 |
|------|---------|------|
| 常规矩形管(镀锌≤1.2mm) | 大小边(7+30) | 最常见，效率最高 |
| 常规矩形管(镀锌1.5mm) | 大小边(7+30) | 上限，注意质控 |
| 不锈钢矩形管(≤1.2mm) | 大小边(7+30) | 可做咬口 |
| 不锈钢矩形管(>1.2mm) | 螺丝(30+0)或焊接 | 咬口会崩边 |
| 大截面矩形管(超1250mm) | 平咬口(20+20)拼接 | 板宽不够必须拆板 |
| 圆形直管 | 平咬口(15+15) | 纵缝标准做法 |
| 圆弯头 | 立咬口(15+15) | 分段连接标准做法 |
| 静压箱出风口 | 静压咀(0+15) | 配接咀专用 |
| 厚板(>1.5mm焊接件) | 搭30焊或直焊边 | 咬口压不动 |
| 封口板 | 堵边(7+40) | 封堵专用 |

## 四、延伸思考

咬口靠的是金属塑性变形。太厚的板硬压，要么压不动，要么压崩了。

焊工师傅一句话讲透了："薄的东西靠折，厚的东西靠焊。"

风管工艺的规律：≤1.2mm薄板优先咬口；1.2~1.5mm看材质，镀锌还行不锈钢换方案；≥1.5mm焊接或螺丝。这个标准不是哪本规范硬性规定的，是设备能力和材料性能共同决定的。

不锈钢崩边那次之后，我在车间墙上贴了一张咬口选型速查表。老李看了说："早贴这个，我那几块料就不糟蹋了。"

后来他再做不锈钢件，主动过来问："唐工，这个1.5的，是不是得换螺丝？"

我说是。

技术规范再全，不让车间的人看见、看懂，就白写。现在技术部每出一个新规范，先打印三张：墙上、桌上、师傅手里。
