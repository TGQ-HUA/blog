# 【公众号版】风管超尺寸怎么办？CAMduct 拼板切割全攻略

做大弯头、大变径的时候经常遇到一个问题：

**展开件超出板材宽度了，排不下去。**

这时候就要用 CAMduct 的**拼板功能**——把一个大件分割成两块或多块，下料后再拼接起来。

听着简单，但坑很多。今天一次讲透。

---

## 什么时候触发拼板？

设置完管件尺寸点「完成」后，如果展开件超出板材宽度，CAMduct 会**自动弹出拼板设置对话框**。

---

## 分割位置选择

| 选项 | 含义 |
|------|------|
| **最大板尺寸** | 分割后最大一片刚好适合板材 |
| **等分料件** | 分成2/3等份（推荐！拼板不容易搞错） |

> 💡 两块料相差不大时选「等分料件」，否则数量多容易拼错。

---

## 分割方向设置

可调整方向和大小比例：
- 左右分割 / 上下分割
- 左大右小 / 左小右大 / 上大下小 / 上小下大

---

## ⚠️ 超尺寸接缝规则（重点！）

| 管件类型 | 拼接缝 |
|---------|--------|
| 矩形管超尺寸 | **17+3** |
| 圆管超尺寸 | **15-15** |
| 焊接管 | 无 |
| 防火板 | 无 |
| 静压箱 | **15-15** |

---

## 特殊管件的拼板处理

| 场景 | 处理方法 |
|------|---------|
| 45°弯头 | 选「45°弯拼板」 |
| 其它角度弯头 | 斜边用「特殊弯拼板」（需到数据库调角度） |
| 任意弯 | 顶板用「特殊弯拼板」/ 底板用「特殊弯拼板-反咬口」 |
| 需滚圆的侧板 | 方向**必须选「垂向」** |
| 圆直管 | 方向一般选「水平」 |
| 圆支咀 | 方向一般选「垂向」 |
| 矩形90°弯头 | 默认垂向 / 上口-左往内 |
| 伞形风帽 | 必须水平拼板 |

---

## ⚠️ 操作注意事项

1. 接缝位置选定后点**「套用」**
2. **禁止用「套用全部」**——不能确定位置的选「忽略」
3. 所有圆形风管的拼板图**必须给生产确认**
4. 拼接缝在法兰孔上时，需调整参数使其错开

---

## 一句话总结

```
超尺寸 → 选分割方式 → 设方向 → 选接缝 → 套用（不要套用全部！）
```
