# 【知乎版】CAMduct 超尺寸拼板切割完整攻略（接缝规则+特殊管件处理）

## 前言

暖通风管制造中，大弯头、大变径、大三通的展开件经常超出板材宽度。这时需要在 CAMduct 中使用拼板功能，将大件分割成多片，下料后通过咬口拼接。

拼板操作看似简单，但接缝类型选错、方向选错都会导致成品报废。本文汇总所有拼板规则和特殊情况处理。

---

## 一、触发条件

设置完管件尺寸点「完成」后，若展开件超出板材宽度，CAMduct 自动弹出拼板设置对话框。

---

## 二、分割位置

| 选项 | 说明 |
|------|------|
| 最大板尺寸 | 分割后最大一片刚好适合所选板材 |
| 等分料件 | 将超尺寸件分成 2/3 等份排版 |

> 两块料相差不大选「等分料件」，否则数量多容易拼错板。

---

## 三、分割方向

可调整方向及大小：左右分割/上下分割，左大右小/左小右大/上大下小/上小下大。

---

## 四、超尺寸接缝规则

| 管件类型 | 拼接缝 |
|---------|--------|
| 矩形管超尺寸 | **17+3** |
| 圆管超尺寸 | **15-15** |
| 焊接管 | 无 |
| 防火板 | 无 |
| 静压箱 | **15-15** |

---

## 五、特殊管件处理

| 场景 | 处理方法 |
|------|---------|
| 45°弯头 | 选用「45°弯拼板」 |
| 其它角度弯头 | 斜边选「特殊弯拼板」（需到数据库调整对应角度） |
| 任意弯顶板 | 用「特殊弯拼板」 |
| 任意弯底板 | 用「特殊弯拼板-反咬口」（正角与负角对调） |
| 需滚圆侧板（弯头外弧板/S弯侧板） | 方向**必须选「垂向」**（平行于法兰口） |
| 所有圆形风管 | 拼板图需给到生产确认 |
| 圆直管 | 方向一般选「水平」 |
| 圆支咀 | 方向一般选「垂向」 |
| 矩形90°弯头 | 默认拼接缝方向：垂向 / 上口-左往内 |
| 伞形风帽 | 必须水平拼板 |

---

## 六、操作注意事项

1. 接缝位置选定后点**「套用」**
2. **禁止选用「套用全部」**——不能确定拼接缝位置的选「忽略」
3. 拼接缝在法兰孔上时需调整侧参数使其错开
4. 可设置最小料件尺寸，保证分割后每片都能方便轧咬口、轧法兰
