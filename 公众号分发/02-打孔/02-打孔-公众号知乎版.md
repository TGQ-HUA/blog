# 共板法兰打孔：一个"固定首孔"的细节，让安装队骂了我们多少次

> 风管厂的技术改造笔记 #002

---

## 一、安装队又打电话来了

"你们打的孔怎么回事？法兰对着，孔对不上！差了好几个毫米。"

上个月安装队老刘又打来电话。跑去车间一量，问题出在**孔距不是整边均分**——两根800mm边长的管子，同样要求孔距≤140mm，一根打了5个孔，另一根打了6个。端头对上去，孔位差十几毫米。

## 二、问题出在哪：固定首孔的"历史遗留病"

风管行业长期以来用的打孔规则叫**固定首孔**——每边两端各留110mm，中间再均分。但110mm这个数是怎么来的？**不是国标规定的。**

GB50243-2016 里关于法兰螺栓间距只写了：
- 风管边长≤250mm时：螺栓间距≤100mm
- 风管边长>250mm时：螺栓间距≤150mm

压根没提首孔边距。110mm是各家厂早年抄设备厂商标准的遗留，**没有统一依据。**

固定首孔的问题在于：
- 两端110mm和中间孔距不一致
- 同宽度管子在不同订单里孔数可能不同
- 不同厂的管子到工地一对就乱

## 三、我们的解决方案：整边均分

今年初风管厂把规则改成**整边均分**——整条边全部等分，孔距一致。

**计算公式：**
```
段数 = Ceil(边长 / 要求孔距)
实际孔距 = 边长 / 段数
```

拿800mm边长、孔距≤140mm举例：
```
段数 = Ceil(800/140) = 6段 → 5个孔
实际孔距 = 800/6 = 133.33mm ✓
```

边界条件也调整了：
- 超短边（<150mm）不打孔
- 孔距<100mm时用固定首孔兜底

## 四、真正的难点：让整个系统跟上

改几行计算代码简单，真正难的是**让车间和设备跟上**。

我们一共动了四样东西：
1. **CAMduct接口参数**重新配
2. **共板法兰机打孔程序**更新
3. **车间师傅习惯**要改（干了8年的师傅改习惯，比改代码难三倍）
4. **库存半成品**防混用标记

决策方案：**2026年1月6日后新项目用新规则，存量项目继续走旧规则**。两套并存，三个月过渡期。CAMduct里加配置项，工人一眼能看清当前走哪套。

## 五、给同行的一张速查表

**新规则孔数速查（孔距≤150mm时）：**

| 边长范围 | 孔数 | 备注 |
|----------|------|------|
| ＜150mm | 0 | 超短边，不打孔 |
| 150-300mm | 1~2 | 小边 |
| 300-600mm | 2~4 | 常规场景 |
| 600-1000mm | 4~7 | 建议确认安装队要求 |
| ＞1000mm | ≥7 | 大边，注意设备行程 |

**切换到新规则的6步清单：**
1. □ CAMduct接口参数更新
2. □ 共板法兰机程序更新
3. □ 通知车间师傅生效日期
4. □ 存量订单标注旧规则
5. □ 同步安装队
6. □ 半成品防混用标记
