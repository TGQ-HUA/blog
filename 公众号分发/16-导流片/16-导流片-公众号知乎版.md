# 风管导流片：两条硬条件，一片都不能多、一片也不能少

**奇美星暖通风管制作 | tangguoqi.top**

---

## 一、一个问题

车间师傅问：500×300的弯头，要加导流片吗？

答案是：不加。边长虽达500，但内弧半径太大，不满足第二条条件。

多数人只知道"大弯头要加导流片"，却不清楚具体判定标准，更不了解部分厂家为省成本直接省略导流片。

## 二、加不加，两条必须同时满足

风管厂手册规定，导流片的设置须同时满足两个条件：

1. 弯头平面边长 ≥ 500mm
2. 内弧半径 ÷ 弯头端口边长 ≤ 0.25

**两条缺一不可。**

第一条：小弯头气流可自然导向，无需导流。

第二条是关键：内弧半径过大的弯头（大R弯），气流转向角度已足够平缓，加导流片反而增加阻力。

实例验证：800×400弯头，内弧半径300mm。300/800 = 0.375 > 0.25 → 不加。若内弧半径改为150mm，150/800 = 0.1875 ≤ 0.25 → 触发导流条件。

实操中最常见的加导流片场景是"小R急弯"——空间受限，弯头半径极小，气流转向剧烈，不加导流片局部阻力将大幅上升。

## 三、加几片、怎么放

风管厂默认采用等分规则：

| 平面边长 b | 片数 | 位置 |
|:----------:|:----:|------|
| 500 ≤ b < 1000 | 1 | b/2 |
| 1000 ≤ b < 1600 | 2 | b/3, 2b/3 |
| 1600 ≤ b < 2000 | 3 | b/4, 2b/4, 3b/4 |
| b ≥ 2000 | 4 | b/5, 2b/5, 3b/5, 4b/5 |

三个要点：

- **片数上限为4**。边长超过2000mm亦不超过四片。
- **间距不小于200mm**。过密无意义。
- **固定须牢固**。螺栓或铆钉，不应使用自攻螺丝随意固定。

导流片默认单片式，安装在弯头弧形段，沿弯头弧度铺设，位置按平面边长等分计算。

## 四、小厂为什么普遍不加

一个项目弯头数量多时，导流片的材料和人工不可忽略。

单片用料有限，但涉及制作、定位、固定，每个弯头增加数分钟工时。数十个弯头累计半天人工。部分小厂因此一律不加——监理未必检查，问责时可推说"客户未要求"。

核心矛盾：导流片降低的是系统阻力，节省的是运行能耗。这些效益在项目交付后业主使用阶段才显现，与风管厂结算无关。加导流片是工艺自觉，不加无人当场扣款。

与空调设计工程师交流得知："弯头不加导流片，局部阻力系数翻倍。风机选型已将导流片纳入计算，省略后系统实际风量将达不到设计值。"

此后风管厂将导流片纳入标准配置：该加的不商量，不该加的不浪费。

## 五、现场快速判断

需要加导流片的情形：
- 客户图纸标注 → 直接执行
- "急弯"（空间窄、R角小）→ 系统设计时通常已考虑
- 中高压系统 → 压力越高，气流越快，导流作用越显著

通常不需加的情形：
- 内弧半径足够大（R > 0.25倍边长）
- 低压系统小尺寸弯头
- 客户明确表示不需要

## 六、速查表

| 判断条件 | 结果 |
|----------|------|
| 边长 < 500mm | ❌ 不加 |
| 内弧半径/边长 > 0.25 | ❌ 不加 |
| 边长 ≥ 500 且 R≤0.25 | ✅ 加，片数=Min(4, Floor(b/500)) |
| 客户图纸标注 | ✅ 按图纸执行 |
| 客户明确不要 | ❌ 按客户要求 |
