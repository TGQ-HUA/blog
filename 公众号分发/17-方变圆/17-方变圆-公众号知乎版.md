# 天圆地方（方变圆）制作——六种连接方式，选错一种就得多焊半小时

## 一、一个真实的翻车

上周一，做单的小姑娘急得把图纸拍到桌上："这个方变圆客户催得急，共板+法兰，你帮我看看。"

我扫了一眼——方口1000×400，变Φ500，长度标了600。

按规范，过渡长度最少要 (1000-500)×1.5=750mm。600mm短了150，还没算法兰占位。好在图纸没进车间，不然师傅下好料才发现方口框怼不上圆法兰，那才叫尴尬。

**这个坑，很多做单的都踩过。**

## 二、方变圆到底"难"在哪？

矩形变圆形，中间有4个三角形平面和4个曲面需要展开。JGJ141-2017第6.6条讲得很清楚：**天圆地方是矩形风管与圆形风管之间的过渡管件**。

不是随便找个钣金工都能展开的。很多小厂硬焊，结果方口不平圆口不圆，到现场装不上，返工一次够做个新的。

## 三、六种连接方式，选错就是半小时

根据风管厂排版规范手册，方变圆按两端接口方式分了6种类型，每种的主体长度调整都不一样：

| 类型 | 部件组成 | 主体长度调整 | 适用场景 |
|------|----------|-------------|---------|
| ① 共板+咀 | 共板框+主体+圆咀 | -80mm | 最常用，现场配活法兰 |
| ② 共板+法兰 | 共板框+主体+法兰 | -10mm | 圆口锁定法兰，不可拆 |
| ③ 共板+活法兰 | 共板框+主体+圆咀配法兰 | -70mm | 圆口可拆卸检修 |
| ④ 角铁+咀 | 主体+圆咀 | -60mm | 角铁法兰场景 |
| ⑤ 角铁+法兰 | 主体+法兰 | +10mm | 圆口直接焊法兰 |
| ⑥ 角铁+活法兰 | 主体+圆咀配法兰 | -50mm | 角铁法兰+可拆卸 |

选①和选②，主体长度差了70mm。一张板宽1.25米，70mm的误差意味着要么废料，要么加焊条补救。

## 四、变径形态不止"对中"

很多人以为方变圆就是方口对中圆口，实际上有三种：

- **对中变**：最常见，轴线重合
- **平边变**：一边对齐（左平/右平/顶平/底平）
- **偏差变**：圆口偏移，高差大的要加长过渡

**过渡长度公式**：L ≥ (方口长边 - 圆直径) × 1.5

这个公式是JGJ141-2017给的。比这个短，过渡太陡，风阻大，风机白费电。

## 五、做单别忘了这些细节

**方口接缝分水岭——250mm**
- 长边≥250：咬口（拼接15mm）
- 长边<250：碰焊5mm

**圆咀默认长度**：80mm（含翻边）

**角铁直伸量**：30#→40mm，40#→50mm，50#→60mm

**报价面积公式**：
```
(宽×2 + 高×2 + 3.14×圆直径 + 接缝×4) / 2 × (长 + 方接口)
```

## 六、一张速查表

| 需要确认 | 默认值 | 说明 |
|---------|-------|------|
| 方口尺寸 | 按图 | 宽×高 |
| 圆口直径 | 按图 | Φ后数字 |
| 过渡长度 | (长边-直径)×1.5 | JGJ141-2017 |
| 圆口连接 | 共板+咀(①) | 不确定就选这个 |
| 方口连接 | 共板框 | 例外：角铁法兰 |
| 变径形态 | 对中 | 偏移要标清 |
| 方口接缝 | ≥250咬口 | <250碰焊 |
| 角铁直伸 | 按角铁型号 | 方口+2mm |

## 七、结尾

方变圆技术难度不大，就是细节绕。六种连接、三种形态、两种接缝，排列组合下来图纸上一个字写错，车间就得改半天。

**做单的多打一个问号，车间少焊半小时。**
