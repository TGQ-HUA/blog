# 【知乎版】CAMduct 选型排料完整步骤教程（新手入门，所有版本通用）

## 前言

刚接触 CAMduct 的同行经常问：软件打开了，然后呢？

今天分享一套**从零开始的完整排料流程**——从打开软件到输出 NC 码，6个步骤走完就能独立操作。适用于所有版本的 CAMduct。

---

## 一、创建新项目排版文件

1. 打开 CAMduct 软件
2. 左侧选择对应图库
3. 右侧选择「创建新文档」

【配图①：CAMduct 启动界面】

---

## 二、选择风管异形件并输入参数

### 方式：样板图库选择

1. 进入操作界面后，点菜单栏「样板图库」调出风管样式选择窗口
2. 左侧点不同文件夹切换风管分类，右侧选择具体样式
3. **双击**目标图形，弹出设置画面

设置画面分为两部分：
- **左侧**：4 个视图（俯视/侧视/正视/立体图）
- **右侧**：5 个选项卡

| 选项卡 | 说明 |
|--------|------|
| **尺寸** | 输入长/宽/直径/角度等参数（不同样式参数不同）。参数有 A/B/C/D 标识，对照左侧图形。「重绘」刷新预览；「展开」查看下料展开形状 |
| **选项** | 当前零件的附加参数，一般无需设置 |
| **项目** | 项目编号（自动编号）、数量、材料（材质+板厚） |
| **接口** | 接口类型(C)、接缝类型(S) |
| **说明** | 备注信息 |

全部设完后，点右下角**「完成」**确认。继续选下一个管件重复上述操作。

> **提示**：点工具栏「工作内容」（或菜单 工具→工作内容），可查看已设好的管件编号、样式、数量及板材。双击可重新修改。

【配图②③④：样板图库 / 参数设置界面 / 工作内容面板】

---

## 三、风管零件排版

1. 点工具栏**「自动排版」**（或菜单 工具→自动排版）
2. 弹出窗口选项一般不改，直接「确定」
3. 进度条完成后点确认，切换到排料界面

> 在「工作内容」→ 第2选项卡「排料」可查看板材排料情况。一个图标对应一张板材，双击可进入手工调整。

【配图⑤：自动排版结果】

---

## 四、写出 NC 码

1. 点**「写出 NC 码」**（或菜单 工具→写出 NC 码）→「确定」
2. 系统弹出保存窗口，选文件夹、输入文件名保存
3. 不需要保存项目文件可点「取消」——不影响 NC 码输出

> ⚠️ 排版报价或文件上传**不需要**写 NC 码。

---

## 五、完整流程总结

```
打开软件 → 创建新文档 → 样板图库选管件 → 填参数 → 自动排版 → 写NC码
```
