# 双层保温管内外壁尺寸——一个公式搞错，整根管子白做

> 双层保温管，内胆+保温棉+外壁三层结构，看着不复杂，但尺寸换算的坑，我们车间踩了不止一回。

---

## 一、从车间一个事故说起

上个月车间老周跑来，一脸苦相——双层保温弯头做完了，外壁装上去才发现短了快20公分。单子上就写了个"900×500 R300"，没标注是内胆尺寸还是外壁尺寸。

这不是第一次了。内外壁尺寸没有死规矩，十有八九要出事：料废了、棉废了、工时搭进去了，工期紧的时候师傅们心态直接崩。

## 二、核心就一个公式：外壁长 = 实长 - 200mm

双层保温管的逻辑很直白：**内胆走风，外壁保温保护，所以内胆要伸出来对接，外壁往内缩。**

| 参数 | 内外壁关系 |
|------|-----------|
| **长度** | 外壁 = 实长 - 200mm |
| **弯头R** | 外壁R = 内胆R - 夹层厚 |
| **直位** | 外壁直位 = 内胆直位 - 100 - 勾股偏差(~20.7mm) |
| **S弯/大小头** | 外壁长 = 实长 - 200 |
| **支咀长** | 外壁咀 = 内胆咀 - 100 - 勾股偏差(~20.7mm) |

**勾股偏差是什么？** 内外壁半径差导致的斜边长度差。夹层50mm时约20.7mm，不是精密计算但少了它就装不上。

## 三、弯头最容易翻车

弯头内外壁要**独立下料**，不能共用排版参数。内胆R更大、直位更长，内外径不一样。

**90°矩形弯头示例（R300，夹层50）：**

| 部件 | 直位 | R | 分段 |
|------|------|---|:----:|
| 内胆 | 170.7mm | 129.3mm | 3段 |
| 外壁 | 50mm | 100mm | 3段 |

内胆直位比外壁多出120.7mm（100+勾股偏差20.7），不是拍脑袋定的，是勾股定理算出来的。

## 四、速查表（贴车间墙上）

| 参数 | 规则 | 注意 |
|------|------|------|
| 外壁总长 | 实长 - 200mm | 带法兰的按法兰面算 |
| 内胆总长 | 实长 + 20mm | 对接处留10mm搭接 |
| 内胆R | 外壁R + 夹层厚 + 勾股偏差 | R<100时单独校验 |
| 外壁R | 按设计要求 | 最小值满足设备能力 |
| 内外胆对接 | **独立下料，不能合并** | 软件里设两套参数 |
| 直位差异 | 内胆 = 外壁 + 100 + 勾股 | 45°和90°分开算 |

## 五、行业现状与思考

双层保温管，大厂做得熟，小厂接了就亏。工序翻倍——做内胆→包棉→做外壁→套外壁→封口，五道工序顶普通管三倍工时。料耗也算不准，因为内外壁独立下料，边角料多出一截。

很多小厂的处理方式是"先报价再说，做的时候再算"。碰到验收严格的工地，就容易出纠纷。GB50243-2016第10.1节对保温风管有明确要求，但不少同行只看了厚度要求，没看结构要求。

## 一句话总结

**凡是双层保温管的单，先确认标的是"内胆"还是"外壁"，没标的直接退给设计。**
