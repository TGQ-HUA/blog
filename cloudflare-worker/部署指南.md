# 文件仓库部署指南

## 前置条件

1. 你已经有一个 Cloudflare 账号（tangguoqi.top 域名在其上）
2. 本地安装了 Node.js（你已有 v24.16.0）

## 第一步：安装 Wrangler CLI

```bash
npm install -g wrangler
```

## 第二步：登录 Cloudflare

```bash
wrangler login
```

浏览器会弹出 Cloudflare 授权页面，点"允许"即可。

## 第三步：创建 R2 存储桶

```bash
wrangler r2 bucket create blog-files
```

## 第四步：设置上传密码

```bash
cd G:\blog\cloudflare-worker
wrangler secret put UPLOAD_PASSWORD
```

输入你想设的密码（比如 `qimeixing123`），这个密码用于上传和删除文件。

## 第五步：部署 Worker

```bash
wrangler deploy
```

部署成功后会输出一个 Worker 地址，如：
```
https://blog-file-repo.你的账户.workers.dev
```

## 第六步：更新前端配置

编辑 `G:\blog\content\tools\file-repo\app.html` 第 75 行：
```javascript
const API_BASE = 'https://你的worker地址.workers.dev';
```

## 第七步：绑定自定义域名（可选）

在 Cloudflare Dashboard → Workers & Pages → blog-file-repo → Triggers → Custom Domains，
添加 `files.tangguoqi.top`，然后把 API_BASE 改为 `https://files.tangguoqi.top`。

## 使用方式

1. 访问 `https://tangguoqi.top/tools/file-repo/`
2. 输入密码 → "登录"
3. 可以上传文件、下载文件、删除文件
4. **下载不需要密码**，任何人都可以直接下载
5. 上传和删除需要密码

## 费用
- Cloudflare Workers：免费（10万次请求/天）
- R2 存储：免费（10GB 存储 + A类操作100万次/月）
- **总花销：0 元/月**
