## 二、各部位详细说明

| 编号 | 名称 | 位置 | 是什么 | 对应文件 |
|:---:|------|------|--------|---------|
| ① | **左侧栏** | 页面最左列 | 整个左边的竖条区域 | hugo.yaml + sidebar模板 |
| ② | **头像区** | 左侧栏顶部 | 头像图片+站名+副标题 | `static/img/avatar.png` |
| ③ | **导航菜单** | 左侧栏中部 | 5个按钮：首页/成果/归档/关于/搜索 | hugo.yaml → menu.main |
| ④ | **主内容区** | 页面中间（最宽） | 文章列表 或 文章正文 或 作品集 | `content/` 目录下的 .md |
| ⑤ | **右侧栏** | 页面最右列 | 搜索+归档+分类+标签 | hugo.yaml → widgets |
| ⑥ | **页面标题** | 主内容区顶部 | 文章/页面的大标题 | .md 文件的 title 字段 |
| ⑦ | **页面正文** | 主内容区主体 | Markdown 渲染出来的内容 | .md 文件正文 |
| ⑧ | **搜索框** | 右侧栏第1个 | 输入关键词搜索文章 | 自动生成 |
| ⑨ | **归档组件** | 右侧栏第2个 | 按年份统计文章数量 | 自动生成 |
| ⑩ | **分类组件** | 右侧栏第3个 | 各分类名称+文章数 | 自动生成 |
| ⑪ | **标签云** | 右侧栏第4个 | 所有标签的按钮排列 | 自动生成 |
| ⑫ | **社交链接** | 左侧栏，头像下方 | GitHub图标 + 邮箱图标 | hugo.yaml → menu.social |
| ⑬ | **暗色切换** | 左侧栏最底部 | 切换深色/浅色模式的按钮 | 自动生成 |
| ⑭ | **页脚** | 页面最底部正中 | © 2026 版权信息 | 自动生成 |
| ⑮ | **汉堡按钮** | 左上角（≡三条杠） | 手机/窄屏时展开菜单用 | 自动生成 |

## 五、首页特有元素

```
首页 = 左侧栏 + 文章卡片列表 + 右侧栏

文章卡片结构：
┌─────────────────────────────────┐
│  [分类标签]  如"暖通风管"（蓝色）    │  ← 分类标签
│                                  │
│  文章标题（大字）                   │  ← 卡片标题
│  文章摘要（小字灰色）               │  ← 卡片描述
│                                  │
│  📅 2026-06-12    🕐 5分钟阅读     │  ← 卡片元信息
└─────────────────────────────────┘
```

## 六、文章页特有元素

```
文章页 = 左侧栏 + 文章正文 + 右侧栏(目录)

文章正文结构：
┌─────────────────────────────────┐
│  文章大标题（h1）                   │
│  分类标签 · 日期 · 阅读时长          │
│  标签列表（橙色小按钮）              │
│  ─────────────────────────       │
│  ## 章节标题（h2，左边有蓝色竖线）    │
│  正文段落...                       │
│  > 引用块（浅橙色底+橙色左边框）     │
│  `行内代码`（橙色底）               │
│  代码块（深色背景+语法高亮）         │
│  ─────────────────────────       │
│  相关文章推荐                       │
│  评论区（后续接入）                  │
└─────────────────────────────────┘
```

## 七、指挥示例

跟我说的时候这样说就行：

- ❌ "那个东西不对" ← 我猜不到
- ✅ "③导航菜单的'成果'按钮，图标显示空白" ← 精确
- ✅ "②头像区的头像太丑了，重新生成" ← 明确
- ✅ "⑩分类组件里'暖通风管'没有图标" ← 可以定位
- ✅ "④主内容区的文章卡片标题字太小" ← 能改
- ✅ "⑤右侧栏的⑨归档组件不需要，去掉" ← 清楚
